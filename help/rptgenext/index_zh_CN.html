Simulink 报告生成器

从 Simulink 模型和 Stateflow 图标设计并自动生成报告
  • Release Notes
  • PDF Documentation
  • Release Notes
  • PDF Documentation

Simulink® 报告生成器™ 提供的函数和 API 能够在报告中包含框图、Stateflow® 图、MATLAB® 函数模块、 真值表、数据字典和其他模型元素。可以设计和生成 PDF, Microsoft® Word, Microsoft PowerPoint®和 HTML 格式的报告。可以生成标准报告（例如系统设计描述）以及包含设计工件（例如生成的代码、需求可跟踪性、文档和测试结果）的自定义报告。还可以包括 DO-178、ISO 26262、IEC 61508 和相关行业标准的工件。

Simulink 报告生成器能够创建 Web 视图，无需 Simulink 许可证即可从 Web 浏览器查看、导航和共享 Simulink 模型。可以将模型 Web 视图嵌入 HTML 代码生成、需求、覆盖率和其他类型的报告中。
