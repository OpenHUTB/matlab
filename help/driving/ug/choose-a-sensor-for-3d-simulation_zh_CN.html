为虚幻引擎仿真选择一个传感器

在Automated Driving Toolbox™中，您可以从虚拟环境中获取高保真度的传感器数据。该环境是使用Epic Games®的Unreal Engine®渲染的。

该表总结了您可以在这个环境中模拟的传感器模块。

Sensor Block Description Visualization Example

Simulation 3D Camera

  • 基于理想针孔相机的具有镜头的相机。请参阅What Is Camera Calibration?

  • 包括图像大小、焦距、畸变和倾斜等参数

  • 包括输出深度估计和语义分割的地面真值的选项

使用 Video Viewer 或 To Video Displa 显示相机图像。示例可视化：

Design Lane Marker Detector Using Unreal Engine Simulation Environment

使用 Video Viewer 或 To Video Display 模块显示深度图。示例可视化：

Depth and Semantic Segmentation Visualization Using Unreal Engine Simulation

使用 Video Viewer 或 To Video Display 模块显示语义分割图。示例可视化：

Depth and Semantic Segmentation Visualization Using Unreal Engine Simulation

Simulation 3D Fisheye Camera

  • 鱼眼相机可以使用Scaramuzza相机模型来描述。请参考 Fisheye Calibration Basics

  • 包括畸变中心、图像尺寸和映射系数等参数

使用 Video Viewer 或 To Video Display 模块显示相机图像。示例可视化：

 Simulate Simple Driving Scenario and Sensor in Unreal Engine Environment

Simulation 3D Lidar

  • 扫描式激光雷达传感器模型。

  • 包括检测范围、分辨率和视场等参数。

使用MATLAB函数块内的 pcplayer 显示点云数据。示例可视化：

 Design Lidar SLAM Algorithm Using Unreal Engine Simulation Environment

使用 Bird's-Eye Scope 显示激光雷达覆盖区域和检测结果。示例可视化：

 Visualize Sensor Data from Unreal Engine Simulation Environment

Simulation 3D Probabilistic Radar

  • 概率雷达模型，返回一个检测列表

  • 包括雷达准确性、雷达偏差、检测概率和检测报告等参数

使用 Bird's-Eye Scope 显示雷达覆盖区域和检测结果。示例可视化：

Simulate Vision and Radar Sensors in Unreal Engine Environment

Visualize Sensor Data from Unreal Engine Simulation Environment

Simulation 3D Vision Detection Generator

  • 返回一个对象和车道边界检测列表的相机模型

  • 包括检测准确性、测量噪声和相机内参等参数

使用 Bird's-Eye Scope 显示视觉覆盖区域和检测结果。示例可视化：

Simulate Vision and Radar Sensors in Unreal Engine Environment
