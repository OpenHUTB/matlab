以编程方式创建参与者和车辆轨迹

Open Script

这个例子展示了如何使用自动驾驶工具箱™功能以编程方式为驾驶场景创建参与者和车辆轨迹。要交互式地创建参与者和车辆轨迹，请使用Driving Scenario Designer应用程序。

参与者和车辆

驾驶场景中的参与者被定义为具有特定长度、宽度和高度的长方体对象。actor也有雷达横截面(在dBsm中指定)，您可以通过定义角坐标(方位角和仰角)来细化。长方体驾驶场景将参与者的位置定义为其底面的中心。驾驶场景使用这个点作为参与者与地面的接触点。这个点也是actor的旋转中心。

车辆是一种特殊的参与者。车辆有三个额外的属性来控制前轴和后轴的位置。

  • 轴距是指前轴和后轴之间的距离。

  • 前悬架是车辆前轴与车头之间的距离。

  • 后悬架是后轴和车辆后部之间的距离。

与参与者不同，车辆的位置是在后轴中心的地面上。这个位置对应于车辆的自然旋转中心。

下表显示了一个典型的actor列表及其相应的维度:

该代码绘制了一个参与者的位置，具有典型人类的尺寸，以及驾驶场景中的车辆。参与者和车辆分别位于位置(0,2)和(0，-2)。

scenario = drivingScenario;
a = actor(scenario,'ClassID',1,'Length',0.24,'Width',0.45,'Height',1.7);
passingCar = vehicle(scenario,'ClassID',1);
a.Position = [0 2 0]
passingCar.Position = [0 -2 0]
plot(scenario)
ylim([-4 4])

默认情况下，场景图显示了参与者的俯视视图。要更改此视图，您可以通过选择图的“视图”菜单中可用的“相机工具栏”交互式地操作场景图。或者，您可以通过使用xlim、ylim、zlim和view等函数以编程方式操作绘图。这些函数使您能够比较参与者的相对高度。

zlim([0 4])

定义轨迹

通过使用smoothTrajectory函数，您可以指定角色和车辆以给定的速度沿着一组路点遵循路径。当您指定路径点时，平滑轨迹函数将分段的曲面曲线拟合到路径点之间的每个段，从而保留点之间的曲率。clocloid曲线的曲率随行驶距离线性变化，这为驾驶员在匀速行驶时导航创造了一个非常简单的轨迹。

默认情况下，参与者轨迹在端点处没有曲率。要完成循环，请指定相同的第一个和最后一个路径点。

若要以恒定速度沿着整个轨迹运动，请将速度指定为标量值。

车辆通过其旋转中心的路径点之间的曲线。因此，为了在模拟过程中适应车辆在后轴前后的长度，您可以偏移起始和结束路径点。偏移这些路径点使车辆完全符合道路的端点。

如果车辆需要快速转弯以避开障碍物，在预定的行驶方向上放置两个靠近的点。这个例子显示了一辆车在两个地方快速转弯，但其他地方转向正常。

scenario = drivingScenario;
road(scenario, [0 0; 10 0; 53 -20],'lanes',lanespec(2));
plot(scenario,'Waypoints','on')
idleCar = vehicle(scenario,'ClassID',1,'Position',[25 -5.5 0],'Yaw',-22);

passingCar = vehicle(scenario,'ClassID',1)

waypoints = [1 -1.5; 16.36 -2.5; 17.35 -2.765; 23.83 -2.01; 24.9 -2.4; 50.5 -16.7];
speed = 15;
smoothTrajectory(passingCar,waypoints,speed)

或者，您可以通过显式设置车辆在每个航路点的偏航方向角，在这样的转弯中使用更少的航路点。在逆时针方向上偏航是正的，它的单位是度。在上一个例子的这种变化中，轨迹受到约束，使得车辆在进入左车道后处于-15度角。通过将航路点设置为NaN, smoothTrajectory函数默认将clooid曲线拟合到指向该航路点的段上。在这种情况下，这段是轨迹的最后一段。

waypoints = [1 -1.5; 16.6 -2.1; 23.7 -0.9; 52.2 -17.6];
yaw =  [0; 0; -15; NaN];
smoothTrajectory(passingCar,waypoints,speed,'Yaw',yaw)

在十字路口转弯和刹车

对于急转弯，要么在转弯的开始和结束处定义靠近的路点，要么明确地设置车辆在每个路点的偏航。这个设置忠实地呈现了转向的突然变化。

在这个例子中，车辆使用明确设置的偏航值在十字路口向左急转弯。在第一个航路点和转弯前的航路点，车辆的偏航度为0度。在转弯后的航路点和最后航路点，车辆有90度的偏航，这是车辆完成转弯后的方向。通过约束轨迹，使车辆达到这些偏航方向，车辆转弯比使用默认偏航方向时要尖锐得多。

平滑轨迹函数生成一个平滑的、限制抖动的轨迹，在路径点之间没有加速度不连续。当改变车辆速度时，例如在转弯时减速，路点之间的距离必须足够大，以便车辆在保持平稳加速的同时达到所需的速度。或者，在较短的距离上，速度的变化必须相对较小。在这个例子中，车辆在进入转弯时从6米/秒的速度减速到5米/秒。转弯完成后，车辆加速回到原来的速度。

scenario = drivingScenario;
road(scenario,[0 -25; 0 25],'lanes',lanespec([1 1]));
road(scenario,[-25 0; 25 0],'lanes',lanespec([1 1]));

turningCar = vehicle(scenario,'ClassID',1);

waypoints = [-20 -2; -5 -2; 2 5; 2 20];
speed = [6 5 5 6];
yaw = [0 0 90 90];
smoothTrajectory(turningCar,waypoints,speed,'Yaw',yaw)

plot(scenario,'Waypoints','on')

移动车辆

在定义了所有的道路、参与者和参与者轨迹之后，您可以在一个循环的驾驶场景中使用advance函数来增加每个参与者的位置。

while advance(scenario)
    pause(0.01)
end

倒车

要指定反向驾驶动作，请指定具有负速度的轨迹。当在前进和反向运动之间切换时，您必须在这些运动之间指定一个速度为0的航路点。在此航路点，车辆减速直至完全停止，然后改变行驶方向。

这个示例扩展了前面的示例。这一次，在完成左转弯后，车辆在十字路口倒车。然后，车辆再次改变方向，向前行驶，直到它停在相反的车道从它开始。由于车辆行驶速度较慢，为了加快模拟速度，请在模拟更新之间指定较短的暂停时间。

scenario = drivingScenario;
road(scenario,[0 -25; 0 25],'lanes',lanespec([1 1]));
road(scenario,[-25 0; 25 0],'lanes',lanespec([1 1]));

turningCar = vehicle(scenario,'ClassID',1);

waypoints = [-20 -2; -5 -2; 2 5; 2 20; 2 5; 5 2; -20 2];
speed = [6 5 5 0 -2 0 5];
yaw =  [0 0 90 90 -90 0 -180];
smoothTrajectory(turningCar,waypoints,speed,'Yaw',yaw)

plot(scenario,'Waypoints','on')

while advance(scenario)
    pause(0.001)
end

Next Steps

这个例子展示了如何使用drivingScenario对象为驾驶场景创建演员和车辆轨迹。要在交互式环境中模拟、可视化或修改此驾驶场景，请尝试使用以下命令将drivingScenario对象导入到driving scenario Designer应用程序中:

drivingScenarioDesigner(scenario)
