使用虚幻引擎模拟环境设计激光雷达SLAM算法

此示例使用：

  • Automated Driving Toolbox
  • Navigation Toolbox
  • Simulink
  • Computer Vision Toolbox
Open Script

这个示例展示了如何从一个3D模拟环境中记录合成激光雷达传感器数据，并使用记录的数据开发同时定位和地图构建 (SLAM) 算法。该模拟环境使用 Epic Games® 的 Unreal Engine®。

介绍

Automated Driving Toolbox™在Simulink®中集成了虚幻引擎模拟环境。与该模拟环境相关的Simulink模块可以在 drivingsim3d 库中找到。这些模块提供以下功能：

  • 在3D仿真环境中选择不同的场景

  • 在场景中放置和移动车辆

  • 在车辆上附加和配置传感器

  • 基于车辆周围的环境模拟传感器数据

这个强大的仿真工具可以在开发、测试和验证自动驾驶算法时，用于补充真实数据，从而实现对在现实世界中难以复现的场景进行测试。

在这个示例中，您将使用从仿真环境生成的合成激光雷达数据来评估激光雷达感知算法。该示例将引导您完成以下步骤：

  • 从仿真环境中记录和可视化合成激光雷达传感器数据。

  • 使用MATLAB®中的SLAM开发一个感知算法来构建地图。

在仿真环境中设置场景

首先，在仿真环境中设置一个场景，用于测试感知算法。使用一个描绘了典型城市街区的场景，其中包含一辆作为测试车辆的单独车辆。您可以使用这个场景来测试算法在城市道路环境中的性能。

接下来，在场景中选择车辆要遵循的轨迹。Select Waypoints for Unreal Engine Simulation 示例描述了如何交互式地从场景中选择一系列路径点并生成车辆轨迹。该示例使用了使用 helperSelectSceneWaypoints 函数获取的记录驾驶段，如路径点选择示例中所述。

% Load reference path for recorded drive segment
xData   = load('refPosesX.mat');
yData   = load('refPosesY.mat');
yawData = load('refPosesT.mat');

% Set up workspace variables used by model
refPosesX = xData.refPosesX;
refPosesY = yData.refPosesY;
refPosesT = yawData.refPosesT;

% Display path on scene image
sceneName = 'USCityBlock';
hScene = figure;
helperShowSceneImage(sceneName);
hold on
scatter(refPosesX(:,2),refPosesY(:,2),7,'filled')

% Adjust axes limits
xlim([-150 100])
ylim([-125 75])

LidarSLAMIn3DSimulation Simulink模型使用 Simulation 3D Scene Configuration 块配置了 US City Block 场景。该模型使用 Simulation 3D Vehicle with Ground Following 块将车辆放置在场景中。使用 Simulation 3D Lidar 块将激光雷达传感器连接到车辆上。在块对话框中，使用 Mounting 选项卡来调整传感器的放置位置。使用 Parameters 选项卡来配置传感器的属性，以模拟不同的激光雷达传感器。在这个示例中，激光雷达安装在车顶的中央位置。激光雷达传感器被配置为模拟典型的Velodyne® HDL-32E传感器。

close(hScene)

if ~ispc
    error(['3D Simulation is only supported on Microsoft',char(174),' Windows',char(174),'.']);
end

% Open the model
modelName = 'LidarSLAMIn3DSimulation';
open_system(modelName);
snapnow;

该模型记录并可视化合成的激光雷达数据。记录的数据可以通过仿真输出获得，并可用于在MATLAB中原型化您的算法。此外，该模型使用一个 From Workspace (Simulink) 块从惯性导航传感器（INS）加载模拟测量数据。INS数据是通过使用 insSensor 对象获得的，并保存在MAT文件中。

接下来的示例按照以下步骤进行：

  1. 运行模型以记录传感器生成的合成激光雷达数据，并将其保存到工作区。

  2. 使用保存在工作区的传感器数据在MATLAB中开发感知算法。感知算法使用SLAM构建周围环境的地图。

  3. 可视化构建地图的结果。

记录和可视化合成激光雷达传感器数据

Record and Visualize子系统使用 To Workspace (Simulink) 块将合成的激光雷达数据记录到工作区。Visualize Point Cloud MATLAB Function 块使用 pcplayer 对象可视化流式点云数据。Visualize INS Path MATLAB Function 块可视化流式INS数据。

运行模型。流式点云显示窗口显示合成的激光雷达传感器数据。场景显示窗口显示合成的INS传感器数据。一旦模型完成仿真，simOut 变量将保存一个包含写入工作区的变量的结构。helperGetPointCloud 函数将传感器数据提取为一个 pointCloud 对象数组。pointCloud 对象是用于保存激光雷达数据并在MATLAB中执行点云处理的基本数据结构。此外，INS数据从MAT文件中加载，稍后将用于开发感知算法。INS数据是使用 insSensor 对象获取的。INS数据已经经过处理，包含世界坐标系中的 [x, y, theta] 姿态信息。

% Update simulation stop time to end when reference path is completed
simStopTime = refPosesX(end,1);
set_param(gcs,'StopTime',num2str(simStopTime));

% Load INS data from MAT file
data = load('insMeasurement.mat');
insData = data.insMeasurement.signals.values;

% Run the simulation
simOut = sim(modelName);

% Create a pointCloud array from the recorded data
ptCloudArr = helperGetPointCloud(simOut);

使用记录的数据开发感知算法

合成的激光雷达传感器数据可以在不同场景下用于开发、实验和验证感知算法。本示例使用算法从流式激光雷达数据中构建环境的三维地图。这样的算法是定位等应用的基本构建块。它还可以用于创建用于在线定位的地理区域的高清（HD）地图。地图构建算法封装在 helperLidarMapBuilder 类中。该类利用MATLAB中的点云和激光雷达处理功能。有关更多详细信息，请参阅 Point Cloud Processing。

helperLidarMapBuilder 类接收来自激光雷达传感器的点云，并逐步构建地图，具体步骤如下：

  1. Preprocess point cloud: 对每个传入的点云进行预处理，去除地面平面和自车。

  2. Register point clouds: 使用正态分布变换 (NDT) 注册算法将传入的点云与上一个点云进行注册。pcregisterndt 函数执行注册操作。为了提高注册的准确性和效率，在注册之前使用 pcdownsample 对点云进行下采样。初始变换估计可以大大提高注册性能。在本示例中，使用INS测量来实现这一点。

  3. Register point clouds: 使用从注册中获得的估计变换将传入的点云转换到地图的参考框架中。

  4. Update view set: 将传入的点云和估计的绝对姿态作为一个视图添加到 pcviewset 对象中。使用它们之间的相对变换在当前视图和上一个视图之间添加连接。

helperLidarMapBuilder 类的 updateMap 方法完成了这些步骤。helperEstimateRelativeTransformationFromINS 函数根据模拟的INS传感器读数计算了注册的初始估计。

这样的算法在累积长序列的地图时容易出现漂移。为了减少漂移，通常会检测闭环，并使用图优化SLAM来纠正漂移。有关详细处理，请参阅 Build a Map from Lidar Data Using SLAM 的示例。helperLidarMapBuilder 类的 configureLoopDetector 方法用于配置闭环检测。一旦配置完成，每次调用 updateMap 时都会进行闭环检测，使用以下函数和类：

  • pcviewset: 管理与点云里程计相关的数据，如点云、姿态和连接。

  • scanContextDescriptor: 从每个传入的点云中提取扫描上下文描述符。扫描上下文是一个用于闭环检测的二维全局特征描述符。

  • scanContextLoopDetector: 管理扫描上下文描述符并检测闭环。它使用 scanContextDistance 计算扫描上下文描述符之间的距离，并选择最接近的特征匹配。

然后，该示例使用点云配准来接受或拒绝闭环候选，并找到闭环的变换。

% Set the random seed for example reproducibility
rng(0);

% Create a lidar map builder
mapBuilder = helperLidarMapBuilder('DownsamplePercent',0.25, ...
    'RegistrationGridStep',3.5,'Verbose',true);

% Configure the map builder to detect loop closures
configureLoopDetector(mapBuilder,'LoopConfirmationRMSE',1.5, ...
    'SearchRadius',0.15,'DistanceThreshold',0.2);

% Loop through the point cloud array and progressively build a map
skipFrames = 5;
numFrames  = numel(ptCloudArr);
exitLoop   = false;

prevInsMeas = insData(1,:);
for n = 1:skipFrames:numFrames

    insMeas = insData(n,:);

    % Estimate initial transformation using INS
    initTform = helperEstimateRelativeTransformationFromINS(insMeas,prevInsMeas);

    % Update map with new lidar frame
    updateMap(mapBuilder,ptCloudArr(n),initTform);

    % Update top-view display
    isDisplayOpen = updateDisplay(mapBuilder,exitLoop);

    % Check and exit if needed
    exitLoop = ~isDisplayOpen;

    prevInsMeas = insMeas;
end

snapnow;

% Close display
closeDisplay = true;
updateDisplay(mapBuilder,closeDisplay);

随着时间的推移，累积漂移逐渐增加，导致地图无法使用。

一旦检测到足够的闭环，可以使用姿态图优化来校正累积的漂移。helperLidarMapBuilder 类的 optimizeMapPoses 方法实现了这一功能，它使用 createPoseGraph 创建姿态图，并使用optimizePoseGraph（导航工具箱）来优化姿态图。

在优化姿态图之后，使用更新后的姿态重新构建地图。helperLidarMapBuilder 的 rebuildMap 方法使用 pcalign 来实现这一点。

使用 optimizeMapPoses 和 rebuildMap 来校正漂移并重新构建地图。在姿态图优化之前和之后可视化视图集。

% Visualize viewset before pose graph optimization
hFigViewset = figure;
hG = plot(mapBuilder.ViewSet);
view(hG.Parent,2);
title('Viewset Display')

% Optimize pose graph and rebuild map
optimizeMapPoses(mapBuilder);
rebuildMap(mapBuilder);

% Overlay viewset after pose graph optimization
hold(hG.Parent,'on');
plot(mapBuilder.ViewSet);
hold(hG.Parent,'off');

legend(hG.Parent,'before','after')

辅助函数

helperGetPointCloud Extract an array of pointCloud objects.

function ptCloudArr = helperGetPointCloud(simOut)

% Extract signal
ptCloudData = simOut.ptCloudData.signals.values;

% Create a pointCloud array
ptCloudArr = pointCloud(ptCloudData(:,:,:,1));

for n = 2 : size(ptCloudData,4)
    ptCloudArr(end+1) = pointCloud(ptCloudData(:,:,:,n));  %#ok<AGROW>
end
end

helperMakeFigurePublishFriendly 调整图形，使其在发布时捕获的截图正确。

function helperMakeFigurePublishFriendly(hFig)

if ~isempty(hFig) && isvalid(hFig)
    hFig.HandleVisibility = 'callback';
end
end

以下是示例中使用的其他支持函数或类。

helperLidarMapBuilder 逐步构建激光雷达地图，使用点云扫描。每个点云都经过处理，去除地面平面和自车，并与前一个点云进行注册。然后通过对齐和合并点云逐步构建点云地图。

helperEstimateRelativeTransformationFromINS 从惯性导航系统（INS）数据中估计相对变换。

helperShowSceneImage 显示虚幻场景的俯视图图像。

helperUpdatePolyline 更新与helperShowSceneImage结合使用的折线位置。
