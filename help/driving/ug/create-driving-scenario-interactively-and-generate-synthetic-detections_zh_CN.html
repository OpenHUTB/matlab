创建交互式驾驶场景并生成合成传感器数据

这个例子展示了如何使用 Driving Scenario Designer 应用程序创建驾驶场景，并从场景中生成视觉和雷达传感器的检测数据。您可以使用这些合成数据来测试您的控制器或传感器融合算法。

这个例子展示了创建场景和生成合成传感器数据的整个工作流程。或者，您可以从预先构建的场景中生成传感器数据。请参阅Prebuilt Driving Scenarios in Driving Scenario Designer。

创建驾驶场景

要打开应用程序，请在 MATLAB® 命令提示符下输入 drivingScenarioDesigner 。

添加道路

将曲线道路添加到方案画布。在应用程序工具行程中，单击 Add Road。然后单击画布的一角，将道路延伸至对角，然后双击画布以创建道路。

要制作道路曲线，请添加要围绕其弯曲的道路中心 它。右键单击道路中间，然后选择Add Road Center。然后将添加的道路中心拖动到其中一个空角 帆布。

要进一步调整道路，您可以单击并拖动任何道路中心。要创建更复杂的曲线，添加更多的道路中心。

添加车道

默认情况下，道路是单车道且没有车道标线。为了使场景更加逼真，将道路转换为双车道高速公路。在左侧窗格中，点击 Roads 选项卡，在 Lanes 部分展开。将 Number of lanes 设置为[1 1]。应用程序会将 Lane Width 参数设置为 3.6 米，这是一个典型的高速公路车道宽度。

道路两侧的白色实心车道标记表示路肩。中间的黄色双实心车道标记表示道路是双向的路。要检查或修改这些车道，请从 Lane Marking 列表中， 选择其中一个车道并修改车道参数。

添加障碍

要沿弯曲道路边缘添加障碍，请使用应用程序工具行程或道路 上下文菜单。在应用程序工具行程中，单击添加 Add Actor > Jersey Barrier。将光标移动到道路的右边缘，然后单击以添加障碍 沿着它。这也会打开左侧窗格中的 Barriers 选项卡。自 在屏障段之间添加 1m 的间隙更改段的值障碍选项卡中的Segment Gap (m) 属性为 1。

要使用道路上下文菜单在道路左边缘添加护栏障碍， 右键单击道路，然后选择添加 Add Guardrail > Left edge。指定护栏的屏障段之间的 1m 间隙，使用 Barriers选项卡中的 Segment Gap (m) 属性。

添加车辆

默认情况下，您添加到场景中的第一辆车是自我 车辆，这是驾驶场景中的主要车辆。自我载体包含检测车道标记、行人或其他车辆的传感器场景。添加自我车辆，然后为自我车辆添加第二辆车检测。

Add Ego Vehicle

要添加自我车辆，请右键单击道路的一端，然后选择 Add Car。要指定汽车的轨迹，请右键单击汽车，选择 Add Waypoints，然后沿路添加航点汽车通过。添加沿路的最后一个航点后，按 Enter 键。汽车沿第一个方向自动旋转 航点。为了在轨迹上获得更精确的精度，您可以调整航点。你也可以右键单击路径以添加新航点。

三角形表示车辆的姿态，原点位于车辆后轴的中心。

调整车辆的速度。在左侧窗格中，点击 Actors 选项卡，将 Constant Speed 设置为 15 m/s。为了更好地控制车辆的速度，可以在Waypoints, Speeds, Wait Times, 和 Yaw表的 v (m/s) 列中设置航点之间的速度。

Add Second Car

添加一个车辆，供自车的传感器检测。在应用程序的工具栏上，点击 Add Actor ，然后选择 Car。添加第二辆车，并设置航点，使其在与自车相反的车道上行驶，并位于道路的另一端。保持车辆的速度和其他设置不变。

Add a Pedestrian

将一个行人添加到场景中，行人正在穿越马路。在道路中间放大，右键单击道路的一侧，然后单击Add Pedestrian。然后，为行人设置路径，在道路的另一侧添加一个路径点。

默认情况下，行人的颜色与车道标线的颜色非常接近。为了使行人更加显眼，从 Actors 选项卡中，单击行人对应的颜色补丁来修改其颜色。

为了测试车辆和行人的速度，运行模拟。根据需要调整参与者的速度或其他属性，通过从 Actors 选项卡的左窗格中选择参与者来进行调整。

例如，如果车辆与行人发生碰撞，在 Waypoints, Speeds, Wait Times, and Yaw表格的 v (m/s) 列中，调整车辆或行人的速度。或者，在表格的 wait (s) 列中，在行人穿过街道之前，为车辆设置等待时间。

默认情况下，模拟在第一个参与者完成轨迹时结束。为了在所有参与者完成轨迹后才结束模拟，在应用程序工具栏上，首先单击 Settings。然后，将  Stop Condition 设置为 Last actor stops。

Add Sensors

Add camera, radar, and lidar sensors to the ego vehicle. 使用这些传感器生成方案中的检测和点云数据。

Add Camera

在应用程序工具行程中，单击添加 Add Camera。传感器画布显示放置传感器的标准位置。点击最前面的预定义传感器位置，用于将相机传感器添加到自我车辆的的前保险杠车辆。

要更精确地放置传感器,可以禁用捕捉选项，在传感器画布的左下角，单击配置传感器画布按钮  .

默认情况下，摄像头只能检测参与者而不是车道。要启用车道检测，在左窗格的 Sensors选项卡上，展开 Detection Parameters 部分，并将 Detection Type 设置为Objects & Lanes。然后展开 Lane Settings 部分，并根据需要更新设置。

Add Radars

将雷达传感器捕捉到左前轮。右键单击预定义的传感器位置，然后选择 Add Radar。默认情况下， 添加到车轮上的传感器是短程传感器。

将雷达传感器向汽车前部倾斜。将光标移到覆盖区域，然后单击并拖动角度标记。

在右前轮添加相同的雷达传感器。右键单击传感器 ，然后单击Copy。然后右键单击右前轮的预定义传感器位置，然后单击 Paste。复制的传感器的方向反映了传感器在对面轮子上的方向。

Add Lidar

将激光雷达传感器捕捉到车顶的中心。右键单击屋顶中心的预定义传感器位置，然后选择 Add Lidar。

激光雷达传感器显示为黑色。车辆周围的灰色是传感器的覆盖区域。缩小以查看覆盖区域的完整视图不同的传感器。

生成合成传感器数据

若要从传感器生成数据，请单击 Run。作为场景运行， Bird’s-Eye Plot 显示检测和点云数据。

Ego-Centric View 从自我车辆的角度显示场景。

因为您指定了激光雷达传感器，所以Ego-Centric View 和 Bird's-Eye Plot 显示的是参与者的网格表示，而不是长方体表示。激光雷达传感器使用这些更详细的参与者表示来生成点云数据。Scenario Canvas 仍然只显示长方体表示。雷达和视觉传感器基于长方体表示进行检测。

要关闭参与者网格、某些类型的检测或显示的其他方面，请使用应用程序工具栏下的 Display 属性。

默认情况下，场景在第一个参与者停止移动时结束。要按设定的时间运行场景，请在应用程序工具栏上单击 Settings 并更改停止条件。

接下来，导出传感器检测：

  • 若要将传感器数据导出到 MATLAB 工作区，请在应用工具行程中，选择 Export > Export Sensor Data。命名工作空间变量，然后单击 OK。该应用程序 将传感器数据保存为包含传感器数据的结构，例如参与者姿势，对象检测，以及每个时间步的车道检测。

  • 要导出生成场景及其传感器数据的 MATLAB 函数，请选择Export > Export MATLAB Function。此函数将传感器数据作为结构返回，将方案作为 drivingScenario 对象返回，并将传感器返回 模型作为系统对象。通过修改此函数，您可以创建 原始场景。有关此过程的示例，请参阅Create Driving Scenario Variations Programmatically。

保存方案

生成检测后，单击 Save 以保存方案文件。您还可以将传感器型号另存为单独的文件并保存道路和参与者模型一起作为单独的方案文件。

您可以从应用重新打开此方案文件。或者在 MATLAB 命令提示符下，您可以使用此语法。

drivingScenarioDesigner(scenarioFileName)
您还可以使用导出的 drivingScenario 对象重新打开场景。在MATLAB命令提示符下，使用以下语法，其中 scenario 是导出对象的名称。
drivingScenarioDesigner(scenario)
要重新打开传感器，请使用以下语法，其中 sensors 是一个传感器对象或这些对象的单元数组。
drivingScenarioDesigner(scenario,sensors)
如果您正在Simulink®中开发驾驶算法，您可以使用 Scenario Reader 模块将道路和参与者从场景文件或 drivingScenario 对象读入到您的模型中。该模块不直接读取传感器数据。要将您在应用程序中创建的传感器添加到Simulink模型中，请选择 Export > Export Simulink Model，生成包含您的场景和传感器的模型。在模型中，生成的 Scenario Reader 模块读取场景，生成的传感器模块定义传感器。
