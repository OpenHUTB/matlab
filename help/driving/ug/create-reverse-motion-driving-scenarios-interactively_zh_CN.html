交互式创建反向行驶场景

这个示例演示了如何使用 Driving Scenario Designer 应用程序创建一个反向行驶的驾驶场景。在这个示例中，您可以指定一个车辆完成一个三点转弯。

三点转弯场景

三点转弯是一种基本的驾驶操作，用于在道路上改变行驶方向。三点转弯包括以下几个步骤：

  1. 向前行驶并向道路的对侧转向。

  2. 在倒车时向原来的一侧转向。

  3. 向前行驶，朝着对侧完成方向的改变。

您可以使用反向行驶来设计更复杂的场景，以测试自动驾驶算法。

Add Road

打开 Driving Scenario Designer 应用程序。

drivingScenarioDesigner

在场景中添加一条直路。右键单击 Scenario Canvas 窗格，然后选择 Add Road。将道路延伸到画布的顶部，使其长度约为50米。双击以将道路添加到画布中。

扩展道路的宽度，以留出足够的空间让车辆完成三点转弯。在左侧窗格的 Roads 选项卡上，将 Width (m) 从 6 增加到 8。

Add Vehicle

在道路上添加一辆车辆。右键单击道路的右下角，然后选择 Add Car。放大车辆和道路的前半部分，这是车辆执行三点转弯的区域。

Add Trajectory

指定车辆完成三点转弯的轨迹。

  1. 右键单击车辆，然后选择 Add Forward Waypoints。指针将显示在画布上的 (x,y) 位置和汽车行驶的运动方向，您可以在指定路标点时查看。

  2. 指定三点转弯的第一段。单击以添加向左转向道路左侧的路标点。

  3. 指定三点转弯的第二段。按下 Ctrl+R 切换到指定反向路标点。然后，单击以添加向右转向道路右侧的路标点。

  4. 指定三点转弯的第三段。按下 Ctrl+F 切换回指定前向路标点。然后，单击以添加向左转向道路左侧的路标点，与第一个指定的路标点相邻。

  5. 按 Enter 键将路标点提交到画布上。

运行仿真

运行仿真。要查看汽车行驶的方向，在应用程序工具栏上选择 Display > Show actor pose indicator during simulation。

在仿真运行时，车辆在三点转弯的每个点之间会短暂停止。在轨迹中在前进和倒车之间切换时，应用程序会自动将切换发生的路标点上的 v (m/s) 值设置为 0。

使用指定的偏航角调整轨迹

为了对轨迹进行微调，可以为车辆在特定路标点达到特定偏航角度。例如，当车辆开始倒车时，您希望车辆与起始位置形成一个正好90度的角度。

首先，通过将指针移动到车辆开始倒车的路标点上，确定该路标点的ID。然后，在左侧窗格的 Waypoints, Speeds, Wait Times, and Yaw 表格中，将相应行的 yaw (deg) 值设置为 90。例如，如果车辆在路标点5开始倒车，则更新 yaw (deg) 列的第五行。

在模拟过程中，车辆现在与起始位置形成一个正好90度的角度。要清除先前设置的偏航角值，右键单击一个路标点，然后选择 Restore Default Yaw。您还可以通过右键单击车辆并选择 Restore Default Yaws 来清除所有设置的偏航角值。

要使用预先构建的具有指定偏航角值的倒车和转弯的场景，请参阅 Euro NCAP Driving Scenarios in Driving Scenario Designer 中的预先构建的自动紧急制动 (AEB) 场景。
