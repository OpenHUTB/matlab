将OpenStreetMap数据导入到驾驶场景中

OpenStreetMap® 是一个免费的开源网络地图服务，可以让您访问众包地图数据。使用 Driving Scenario Designer 应用程序，您可以从OpenStreetMap导入地图数据，并将其用于生成驾驶场景中的道路。

本示例重点介绍在应用程序中导入地图数据的方法。或者，要将OpenStreetMap道路导入到 drivingScenario 对象中，请使用 roadNetwork 函数。

选择OpenStreetMap文件

要导入道路网络，首先必须选择包含该网络道路几何信息的OpenStreetMap文件。要从 openstreetmap.org 导出这些文件，请指定一个地图位置，手动调整该位置周围的区域，并将该区域的道路几何信息导出为扩展名为 .osm 的OpenStreetMap文件。只有完全位于指定区域内的道路才会被导出。在本示例中，您将选择之前从该网站导出的OpenStreetMap文件。

  1. 打开 Driving Scenario Designer 应用程序。

    drivingScenarioDesigner

  2. 在应用程序的工具栏上，选择 Import ，然后选择 OpenStreetMap。

  3. 在OpenStreetMap导入对话框中，浏览此文件，其中 matlabroot 是您的MATLAB®文件夹的根目录：

    matlabroot/examples/driving/data/applehill.osm

    该文件是从 https://www.openstreetmap.org 下载的，该网站提供全球范围内的众包地图数据。这些数据受到Open Data Commons Open Database License (ODbL)的许可，详情请参阅https://opendatacommons.org/licenses/odbl/。

对话框的 Select Roads 部分显示了MathWorks® Apple Hill校园的道路网络。灰色框表示从 openstreetmap.org 选择的地图区域。灰色框的中心点是地理参考点。点击该点可以显示或隐藏坐标数据。当将道路导入到该应用程序时，该点将成为驾驶场景的原点。

选择要导入的道路

在“选择道路”对话框的 Select Roads 部分，选择要导入到驾驶场景中的道路。可选择的道路以黑色显示。

注意

您选择的道路数量直接影响应用程序的性能。请选择您创建驾驶场景所需的最少道路。

由于这个道路网络很小，点击 Select All 以选择所有道路。所选道路将以蓝色显示。

以下是您可以选择道路导入的操作。

Goal Action
从该区域选择单个道路。 单击单个道路以选择它们。
从该区域选择所有道路。 单击 Select All。
从该区域选择所有道路，但排除其中的几条。 单击 Select All，然后单击单个道路以取消选择。
从该区域选择道路的子集，例如选择该区域上半部分的所有道路。

在地图显示的左上角，单击“选择道路”按钮。然后，绘制一个矩形框选中道路。

  • 要从此选择中取消选择一部分道路，请单击“取消选择道路”按钮。然后，绘制一个矩形框取消选择道路。

  • 要取消选择所有道路并重新开始，请单击 Deselect All。

导入道路

在选择网络中的所有道路后，点击 Import。应用程序会暂停导入并将其中一条道路标记为红色。应用程序无法正确渲染该道路的几何形状，因此无法导入该道路。

点击 Continue 以继续导入所有其他选定的道路。应用程序将导入道路并生成道路网络。

为了保持与地理地图显示的相同对齐，场景画布的X轴在底部，Y轴在左侧。在不从地图导入的驾驶场景中，X轴在左侧，Y轴在底部。这种对齐与 Automated Driving Toolbox™ 的世界坐标系一致。场景的原点对应于地理参考点。

默认情况下，禁用道路交互。禁用道路交互可以防止意外修改网络，并通过隐藏道路中心来减少视觉杂乱。如果您想修改道路，在 Scenario Canvas 的左下角，点击配置场景画布按钮。然后，选择 Enable road interactions。

比较导入的道路与地图数据

应用程序中生成的道路网络与OpenStreetMap道路网络不同。例如，检查道路网络的这一部分的差异。

OpenStreetMap Road Network Imported Driving Scenario

导入场景中道路之间的过渡更加突然，因为应用程序不支持随着车道数量的变化逐渐变窄的车道。此外，由于应用程序不从OpenStreetMap导入车道级别的信息，生成的道路网络中的车道数量仅基于道路网络中指定的行驶方向，其中：

  • 所有单行道被导入为单车道道路。

  • 所有双行道被导入为双车道道路。

这些车道都具有相同的宽度，这可能会导致突然的过渡，如表中所示的示例。

有关导入OpenStreetMap数据的限制的更多详细信息，请参阅 Driving Scenario Designer 应用程序参考页面的 Limitations 部分。

保存场景

保存场景文件。保存场景后，您无法将其他OpenStreetMap道路导入其中。相反，您必须创建一个新的场景并导入新的道路网络。

现在，您可以向场景中添加参与者和传感器，为测试驾驶算法生成合成车道和物体检测，或将场景导入到Simulink®中。
