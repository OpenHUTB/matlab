使用Driving Scenario Designer创建具有多车道规范的道路

这个示例展示了如何使用 Driving Scenario Designer 应用程序交互式地创建具有多个车道规格的道路。您可以通过为该道路定义多个车道规格来添加或删除车道。

这个示例展示了创建具有特殊超车车道的道路的工作流程。超车车道使较慢的车辆可以进入右车道，以便较快的车辆可以在左车道上无障碍地超车。

您还可以以编程方式定义多个车道规格。有关更多信息，请参阅compositeLaneSpec。

Open Driving Scenario Designer

要打开应用程序，请在 MATLAB® 命令提示符处输入 drivingScenarioDesigner。

Add Road

在场景画布上添加一条直线道路。在应用程序工具栏上，选择 Add Road。然后，在画布的底部中心点击一次，将道路延伸到顶部中心，并双击画布以创建道路。要为道路中心指定精确的坐标，请在左侧窗格上的 Roads 选项卡上展开 Road Centers 部分，并输入此表中显示的x轴和y轴位置的值。

Road Center x (m) y (m)
1 0 0
2 100 0

Define Multiple Lane Specifications

默认情况下，道路是一个没有车道标线的单段、单车道的道路。要定义多个车道规格，将道路分割为所需数量的道路段，并为每个道路段指定不同的车道规格。对道路车道和段的编号顺序取决于道路的方向。有关详细信息，请参阅Draw Direction of Road and Numbering of Lanes and Composite Lane Specification。

Divide Road into Segments

在左窗格的 Roads 选项卡上，将 Number of Road Segments 设置为 3。默认情况下，道路的长度均匀分为指定数量的道路段。您可以使用 Segment Range 参数编辑道路段的长度，该参数表示每个道路段的归一化范围。将 Segment Range 参数设置为 [0.4 0.3 0.3]。

为每个道路段指定车道信息

要为每个道路段定义车道规格，请从 Road Segment 列表中选择所需的段。然后，展开 Lanes 部分，并按照以下表格设置 Number of lanes 和 Lane Width (m) 参数。

Road Segment Number of Lanes Lane Width (m)
Segment 1 (range = 40% ) [1 1] 4
Segment 2 (range = 30% ) [1 2] 4
Segment 3 (range = 30% ) [1 1] 4

此图显示了指定车道信息后的道路。

您可以检查或修改所选道路段中车道的 Lane Types 和 Lane Markings参数。此示例使用这些参数的默认值。

定义道路段之间的连接

默认情况下，当您在道路上添加或删除车道时，道路段之间没有锥度。为了使场景更加逼真，可以指定车道在道路段之间线性锥度的信息。

展开 Segment Taper 部分，从列表中选择 Taper 1 ，并设置以下参数：

  • Shape — Linear

  • Length (m) — 15

从列表中选择 Taper 2 ，并为其相应的参数指定相同的值。

注意

对于双向道路段，该应用程序根据指定的 Number of Lanes 参数确定添加或减少车道的位置。当您为单向道路创建多个车道规格时，Segment Taper 还包含 Position 参数。使用该参数，您可以指定添加或减少车道的道路段边缘。

此图像显示了在指定 segment taper信息后的道路。

下一步

此示例展示了如何使用 Driving Scenario Designer 应用程序创建具有多个车道规格的道路。您可以向场景中添加角色和轨迹。您还可以添加传感器并生成合成检测。有关更多信息，请参阅 Create Driving Scenario Interactively and Generate Synthetic Sensor Data。

要将道路网络导出为ASAM OpenDRIVE®文件，请参阅 Export Driving Scenario to ASAM OpenDRIVE File。
