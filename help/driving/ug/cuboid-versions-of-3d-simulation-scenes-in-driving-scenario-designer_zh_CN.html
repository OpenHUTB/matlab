驾驶场景设计器中 3D 模拟场景的长方体版本

Driving Scenario Designer 应用提供从 3D 重新创建场景的预生成方案 长方体模拟环境中的模拟环境。在这些长方体版本中 场景中，您可以添加使用简单框形表示的车辆并指定其 轨迹。然后，您可以使用场景的更高保真度 3D 模拟版本，在 Simulink® 模型中模拟这些车辆和轨迹。这 3D环境使用Epic Games® 的Unreal Engine® 渲染这些场景。有关环境的更多详细信息，请参阅用于 Unreal Engine Simulation for Automated Driving。

Choose 3D Simulation Scenario

Open the Driving Scenario Designer app. At the MATLAB® command prompt, enter drivingScenarioDesigner.

该应用程序将 3D 模拟场景保存为名为scenario files的MAT文件。要打开场景文件，请首先在应用程序工具栏上选择 Open > Prebuilt Scenario 。打开的 PrebuiltScenarios 文件夹包含所有可在应用程序中使用的预建场景的子文件夹。

双击 Simulation3D 文件夹，然后选择表中描述的其中一个场景。

注意

Driving Scenario Designer 应用程序不包括这些场景的立方体版本：

  • Large Parking Lot

  • Open Surface

  • Parking Lot

  • Virtual Mcity

要为这些不受支持的场景或自定义场景生成车辆轨迹，请使用 Select Waypoints for Unreal Engine Simulation 示例中描述的过程。

修改场景

加载场景后，您可以向场景中添加车辆，设置它们的轨迹，并将其中一个车辆指定为自车。有关如何执行这些操作的示例，请参阅 Create Driving Scenario Interactively and Generate Synthetic Sensor Data。

如果您计划在相应的 3D 场景中模拟这些车辆，请避免修改道路网络或现有的道路对象，如障碍物和交通锥。否则，可能会破坏两个版本之间的一致性，导致模拟结果不同。为了防止意外更改现有道路网络，道路交互默认处于禁用状态。如果您想修改道路网络，请在 Scenario Canvas 窗格的左下角单击“配置Scenario Canvas”按钮。然后，选择 Enable road interactions。

您可以向自车添加传感器，但无法在 3D 环境中重新创建这些传感器。环境具有自己的传感器，以 Simulink 块的形式存在。有关更多详细信息，请参阅 Choose a Sensor for Unreal Engine Simulation。

保存场景

由于这些场景是只读的，要保存您的场景文件，您必须将其另存为一个新文件夹中的副本。在应用程序工具栏上，选择 Save > Scenario File As。

您可以从应用程序中重新打开保存的场景文件。或者，您可以在MATLAB命令提示符处输入以下命令，其中 scenarioFileName 表示要打开的场景文件。

drivingScenarioDesigner(scenarioFileName)

在 Simulink 中为 3D 环境重新创建场景

在保存包含您添加的车辆和其他角色的场景文件后，您可以在Simulink中重新创建这些车辆的轨迹。在高层次上，按照以下工作流程进行操作：

  1. Configure 3D scene — 在新模型中，添加一个 Simulation 3D Scene Configuration 块，并指定与您的场景文件对应的 3D 场景。

  2. Read actor poses from scenario file — 添加一个 Scenario Reader 块，并从您的场景文件中读取每个时间步的角色姿势。这些姿势构成了角色的轨迹。

  3. Transform actor poses — 从Scenario Reader块输出角色，包括自车。使用 Vehicle To World 和 Cuboid To 3D Simulation 块将角色姿势转换为3D环境中使用的坐标系。

  4. Specify actor poses to vehicles — 添加与模型中的车辆相对应的 Simulation 3D Vehicle with Ground Following 块。将转换后的角色姿势指定为车辆块的输入。

  5. Add sensors and simulate — 添加传感器，模拟3D环境，并使用 Bird's-Eye Scope 查看传感器数据。

有关按照此工作流程进行操作的示例，请参阅 Visualize Sensor Data from Unreal Engine Simulation Environment。
