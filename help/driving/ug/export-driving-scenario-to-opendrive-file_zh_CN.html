将驾驶场景导出到 ASAM OpenDRIVE 文件

ASAM OpenDRIVE®是一种开放的文件格式，可以用于指定大型和复杂的道路网络。使用 Driving Scenario Designer 应用程序，您可以将驾驶场景中的道路、车道、交叉口和角色导出到 ASAM OpenDRIVE 文件中。原始场景和导出场景之间可能存在差异。有关详细信息，请参阅 Limitations。

该应用程序支持将驾驶场景导出为OpenDRIVE®文件版本V1.4、V1.5和ASAM OpenDRIVE文件版本V1.6。

若要以编程方式将 drivingScenario 对象中的道路、车道、交叉路口和参与者导出到 OpenDRIVE 文件，请使用 export 对象 drivingScenario 对象的函数。

加载场景文件

要打开 Driving Scenario Designer 应用程序，请在 MATLAB® 命令提示符下输入以下命令：

drivingScenarioDesigner

要加载场景文件，请在应用程序工具栏上，单击 Open > Scenario File。您选择的文件必须是具有 .mat 文件扩展名的有效驾驶场景会话文件。

从 MATLAB 根文件夹中导航到并打开此文件：

matlabroot/examples/driving/data/LeftTurnScenario.mat

Scenario Canvas 选项卡显示场景。

注意

您还可以使用 Driving Scenario Designer 应用程序创建场景，然后将场景导出为 ASAM OpenDRIVE 文件。有关如何创建场景的信息，请参阅 Create Driving Scenario Interactively and Generate Synthetic Sensor Data。

导出到ASAM OpenDRIVE

要将场景中的道路、车道、交叉口和参与者导出到 ASAM OpenDRIVE 文件，请在应用程序工具栏上单击 Export > ASAM OpenDRIVE File。

应用程序将打开OpenDRIVE导出对话框。使用以下选项指定文件信息：

  • File Path — 使用.xodr 或 .xml 文件扩展名为输出的 ASAM OpenDRIVE 文件指定名称。默认情况下，应用程序将文件保存在当前工作目录中。要指定不同的文件位置，请单击 Browse。如果已存在具有指定名称的文件，则应用程序将覆盖现有文件。

  • File Version — 将输出文件的文件格式版本指定为Version 1.4 (default), Version 1.5 或 Version 1.6。

  • Export Actors — 选择此参数将参与者导出到 ASAM OpenDRIVE 文件中。默认情况下，此参数已选中。

  • Show errors and warnings — 选择此参数以显示导出过程中发生的错误和警告。默认情况下，此参数已选中。

一旦您指定了文件信息，请单击 Export。如果您选择了 Show errors and warnings 参数，则“导出ASAM OpenDRIVE结果”对话框将显示导出过程中发生的任何警告和错误。您可以将这些警告和错误复制到文本文件中。然后，关闭对话框。

检查导出的场景

要使用 Driving Scenario Designer 应用程序检查导出的场景，请在应用程序工具栏上，选择 Import > ASAM OpenDRIVE File。选择导出的 ASAM OpenDRIVE 文件，然后单击 Open。该应用程序不支持导入ASAM OpenDRIVE文件中指定的参与者。有关更多详细信息，请参阅 Import ASAM OpenDRIVE Roads into Driving Scenario 。

Actual Scenario Exported Scenario

注意，导出的道路网络在道路交叉口处显示出一些细微的变化。有关这些变化的更多信息，请参阅 Limitations。

限制

Roads

  • 在场景中，立方多项式和参数化立方多项式几何类型被导出为螺旋几何类型。这会导致导出的道路几何形状在道路为曲线时出现一些变化。例如，在下图中，注意到输入道路中的尖角在导出为ASAM OpenDRIVE格式时变得相对平滑。

    Input Road Exported ASAM OpenDRIVE Road

  • 当相邻道路的段重叠时，应用程序不会导出道路的重叠段。

    Input Roads Exported ASAM OpenDRIVE Roads

Lanes

  • 当具有多个车道规格的道路在两个道路段之间包含锥度时，应用程序会导出没有锥度的道路。

    Input Road Exported ASAM OpenDRIVE Road

  • 当由多个段组成的道路连接到一个交叉口时，应用程序不会导出该道路。

Junctions

  • 道路网络的交叉口在处理时没有车道连接信息，因此导出的交叉口形状可能不准确。

    Input Roads Exported ASAM OpenDRIVE Roads

  • 当一个交叉口没有连接到任何道路时，应用程序不会导出该交叉口。

    Input Roads Exported ASAM OpenDRIVE Roads

    要将一个独立的交叉口导出到ASAM OpenDRIVE文件中，您可以手动拖动进入或离开的道路，并将它们连接到交叉口。

Actors

  • 该应用程序不会导出任何存在于交叉口或具有多个道路段的道路上的参与者。

  • 在导出用户定义的参与者时，应用程序将对象类型设置为 'none'。

ASAM OpenDRIVE 导入

  • 当您导出包含导入的ASAM OpenDRIVE场景的驾驶场景对象时，ASAM OpenDRIVE导入的限制也适用于ASAM OpenDRIVE导出。有关ASAM OpenDRIVE导入的限制信息，请参阅 Import ASAM OpenDRIVE Roads into Driving Scenario。
