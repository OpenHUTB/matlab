安装定制场景支持包

Step 1 of 4 in Customize Unreal Engine Scenes for Automated Driving

要在虚幻编辑器的安装中自定义场景并在Simulink®中模拟这些场景，您必须首先安装虚幻引擎4项目的自动驾驶工具箱界面。

注意

这些安装说明适用于R2022b。如果您使用的是以前的版本，请参阅其他版本的文档。

验证软件和硬件要求

在安装支持包之前，请确保您的环境满足虚幻引擎仿真环境要求和限制中描述的最低软件和硬件要求。

安装支持包

要安装虚幻引擎®4项目支持包的自动驾驶工具箱™界面，请遵循以下步骤:

  1. 在MATLAB®Home选项卡的环境部分，选择Add-Ons > Get Add-Ons。

  2. 在Add-On Explorer窗口中，搜索虚幻引擎4项目支持包的自动驾驶工具箱界面。点击Install。

    注意

    您必须具有安装文件夹的写权限。

使用支持包设置场景定制

虚幻引擎4项目支持包的自动驾驶工具箱界面包括这些组件:

  • AutoVrtlEnv文件夹 - 一个包含AutoVrtlEnv.Uproject文件及相应的支持文件的虚幻引擎项目文件夹。该项目包含预构建场景的可编辑版本，您可以从模拟3D场景配置块的场景名称参数中选择。

  • MathWorkSimulation - 一个在Simulink和虚幻编辑器之间建立连接的插件。这是联合仿真所必需的。

  • MathWorksAutomotiveContent - 一个插件，包含汽车对象的元素，可以使用虚幻编辑器修改。这是联合仿真所必需的。

  • RoadRunnerScenes 文件夹- 包含虚幻引擎项目和使用RoadRunner场景编辑软件创建的场景的相应可执行文件的文件夹。此文件夹包含以下子文件夹:

    • RRScene -  包含RRScene的虚幻引擎项目文件夹。Uproject文件及相应的支持文件。这个项目包含一个可编辑的场景版本，在公路车道跟随RoadRunner场景示例中使用。

    • WindowsPackage -  包含可执行的RRScene.exe和支持文件的文件夹。使用这个可执行文件来共同模拟在公路车道跟随RoadRunner场景示例中解释的Simulink模型。

    • RoadRunnerMaterials - 一个插件，它包含了roadrunner场景中对象的特性。

要设置场景定制，你必须将AutoVrtlEnv项目和MathWorksSimulation插件文件夹复制到本地机器上。要自定义RoadRunner场景中使用的公路车道跟随RoadRunner场景示例，你还必须复制RRScene项目到你的本地机器上，下载RoadRunner materials插件并复制到你的本地项目中。

安装和设置支持包之后，就可以开始定制场景了。如果您想使用使用虚幻引擎4项目支持包的先前版本的自动驾驶工具箱界面开发的项目，您必须迁移该项目，使其与当前支持的虚幻编辑器版本兼容。请参阅使用先前支持包开发的迁移项目。否则，请参见使用Simulink和虚幻编辑器自定义场景。

复制AutoVrtlEnv项目到本地文件夹，MathWorksSimulation和MathWorksAutomotiveContent插件到虚幻编辑器

要将所有支持包组件复制到本地机器上的一个文件夹中，并配置您的环境，以便您可以自定义场景，请使用copyExampleSim3dProject函数。例如，以下代码将文件复制到C:\project。

sim3d.utils.copyExampleSim3dProject("C:\project");

(可选)拷贝“RRScene工程”到本地文件夹

要在RRScene项目文件夹中自定义场景，请将项目复制到本地机器上。

  1. 指定包含项目的支持包文件夹的路径。还要指定复制项目的本地文件夹目标。这段代码使用了前几节中的支持包路径和本地文件夹路径。

    rrProjectSupportPackageFolder = fullfile( ...
    matlabshared.supportpkg.getSupportPackageRoot, ...
    "toolbox","shared","sim3dprojects","driving", ...
    "RoadRunnerScenes","RRScene");
rrProjectLocalFolder = fullfile(localFolder,"RRScene");

  2. 将RRScene项目从支持包文件夹复制到本地目标文件夹。

    if ~exist(rrProjectLocalFolder,"dir")
    copyfile(rrProjectSupportPackageFolder,rrProjectLocalFolder);
end

RRScene.uproject文件及其所有支持文件现在位于指定的本地文件夹中名为RRScene的文件夹中。例如:C:\Local\RRScene。

(可选)复制RoadRunnerMaterials插件到虚幻编辑器

当在RRScene项目文件夹中定制场景时，你还必须将RoadRunnerMaterials插件复制到你的插件项目文件夹中。

  1. 下载包含最新RoadRunner插件的ZIP文件。参见下载插件(RoadRunner)。将ZIP文件的内容解压缩到本地机器。解压后的文件夹名为“RoadRunner Plugins x.x.x”，其中x.x.x为插件的版本号。

  2. 指定RoadRunnerMaterials插件的路径。这个插件位于解压文件夹的虚幻/插件文件夹中。更新此代码以匹配下载插件的位置和插件版本号。

    rrMaterialsPluginFolder = fullfile("C:","Local","RoadRunner Plugins 1.0.3", ...
    "Unreal","Plugins","RoadRunnerMaterials");

  3. 在您的本地RRScene项目中，创建一个Plugins文件夹，用于复制插件。这段代码使用上一节中指定的本地RRScene项目的路径。

    rrProjectPluginFolder = fullfile(rrProjectLocalFolder,"Plugins","RoadRunnerMaterials");

  4. 将RoadRunnerMaterials插件复制到本地项目的Plugins文件夹中。

    copyStatus = copyfile(rrMaterialsPluginFolder,rrProjectPluginFolder);
if copyStatus
    disp("Successfully copied RoadRunnerMaterials plugin to RRScene project plugins folder.")
else
    disp("Unable to copy RoadRunnerMaterials plugin to RRScene project plugins folder.")
end

安装和设置支持包之后，就可以开始定制场景了。如果您想使用使用虚幻引擎4项目支持包的先前版本的自动驾驶工具箱界面开发的项目，您必须迁移该项目，使其与当前支持的虚幻编辑器版本兼容。请参阅使用先前支持包开发的迁移项目。否则，请参见使用Simulink和虚幻编辑器自定义场景。
