读取和可视化HERE HD Live地图数据

HERE HD Live Map 1 (HERE HDLM)是由HERE Technologies开发的基于云的网络服务，可让您访问高度精确、持续更新的地图数据。这些数据由瓦片地图图层组成，包含道路和车道的拓扑和几何信息、道路级属性和车道级属性，以及沿途的障碍物、标志和杆子等信息。这些数据适用于各种高级驾驶辅助系统（ADAS）应用，包括定位、场景生成、导航和路径规划。

使用Automated Driving Toolbox™的函数和对象，您可以配置和创建HERE HDLM读取器，从HERE HDLM网络服务中读取地图数据，然后可视化某些图层的数据。

输入凭据

在使用HERE HDLM网络服务之前，您必须输入从与HERE Technologies的协议中获得的凭据。要设置您的凭据，请使用 hereHDLMCredentials 函数。

hereHDLMCredentials setup

请输入有效的 Access Key ID 和 Access Key Secret，并点击 OK。这些凭据将在您的计算机上的MATLAB®会话的其余部分中保存。要在您的计算机上保存凭据以供将来的MATLAB会话使用，在对话框中选择 Save my credentials between MATLAB sessions。这些凭据将保留保存，直到您删除它们。

配置读取器以搜索特定目录

在HERE HDLM网络服务中，地图数据存储在一组称为目录的数据库中。每个目录大致对应于不同的地理区域，如北美或西欧。通过创建一个 hereHDLMConfiguration 对象，您可以配置HERE HDLM读取器仅搜索特定目录的地图数据。您还可以选择指定要搜索的目录版本。这些配置可以加快读取器的性能，因为读取器不会搜索不必要的目录以获取地图数据。

例如，创建一个配置，用于大致对应于北美地区的目录。

config = hereHDLMConfiguration('hrn:here:data::olp-here-had:here-hdlm-protobuf-na-2');

使用此配置创建的读取器仅从指定的目录中搜索地图数据。

配置HERE HDLM读取器是可选的。如果您不指定配置，读取器将默认搜索所有目录中的地图瓦片。读取器将返回在这些瓦片中找到的最新版本的目录中的地图数据。

创建特定地图瓦片的读取器

hereHDLMReader对象从一组地图瓦片中读取HERE HDLM数据。默认情况下，这些地图瓦片的缩放级别设置为14，对应于大约5-10平方公里的矩形区域。

创建 hereHDLMReader 对象时，您可以选择从中读取数据的地图瓦片。您可以直接指定地图瓦片的ID。或者，您可以指定驾驶路线的坐标，并从该路线的地图瓦片中读取数据。

加载北美驾驶路线的纬度-经度坐标。为了参考，显示该路线在地理坐标轴上的位置。

route = load('geoSequenceNatickMA.mat');
lat = route.latitude;
lon = route.longitude;

geoplot(lat,lon,'bo-')
geobasemap('streets')
title('Driving Route')

使用指定的驾驶路线和配置创建一个 hereHDLMReader 对象。 

reader = hereHDLMReader(lat,lon,'Configuration',config);
该读取器使您能够读取此驾驶路线所在的瓦片的地图数据。地图数据存储在一组包含有关地图各个方面的详细信息的图层中。该读取器支持从道路中心线模型、HD车道模型和HD定位模型的地图图层中读取数据。

有关这些模型中图层的更多详细信息，请参阅 HERE HD Live Map Layers。

读取地图图层数据

read 函数读取所选地图瓦片的数据。地图数据以一系列图层对象的形式返回。从包含道路拓扑几何的图层中读取数据。

topology = read(reader,'TopologyGeometry')
topology = 

  2×1 TopologyGeometry array with properties:

   Data:
    HereTileId
    IntersectingLinkRefs
    LinksStartingInTile
    NodesInTile
    TileCenterHere2dCoordinate

   Metadata:
    Catalog
    CatalogVersion

每个地图图层对象对应于使用 hereHDLMReader 对象选择的地图瓦片。这些地图瓦片的ID存储在读取器的 TileIds 属性中。检查第一个地图瓦片的地图图层对象的属性。您的目录版本和地图数据可能与此处显示的不同。

topology(1)
ans = 

  TopologyGeometry with properties:

   Data:
                    HereTileId: 321884279
          IntersectingLinkRefs: [42×1 struct]
           LinksStartingInTile: [905×1 struct]
                   NodesInTile: [635×1 struct]
    TileCenterHere2dCoordinate: [42.3083 -71.3782]

   Metadata:
                       Catalog: 'hrn:here:data::olp-here-had:here-hdlm-protobuf-na-2'
                CatalogVersion: 3321

TopologyGeometry 图层对象的属性对应于该图层的有效HERE HDLM字段。在这些图层对象中，图层字段的名称已修改以适应MATLAB对象属性的命名约定。有关图层对象的更多详细信息，请参阅 read 函数参考页面上的 layerData 输出参数描述。

可视化地图图层数据

要可视化地图图层的数据，请使用 plot 函数。绘制返回的地图图层的拓扑几何。绘图显示了地图瓦片内的边界、节点（交叉口和死胡同）和链接（街道）。如果链接超出了指定地图瓦片的边界，图层数据将包括该链接。

plot(topology)

地图图层绘图返回的是地理坐标轴上的绘图。要自定义地图显示，可以使用地理坐标轴的属性。有关更多详细信息，请参阅GeographicAxes Properties。在绘图上叠加驾驶路线。

hold on
geoplot(lat,lon,'bo-','DisplayName','Route')
hold off
