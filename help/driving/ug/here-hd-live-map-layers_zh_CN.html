HERE HD Live Map图层

HERE HD Live Map 1 (HERE HDLM)，是由HERE Technologies开发的基于云的Web服务，可以让您访问高精度、持续更新的地图数据。这些数据由瓦片地图图层组成，包含道路和车道的拓扑和几何信息、道路级和车道级属性，以及沿途发现的障碍物、标志和杆子等信息。这些数据存储在一系列与地理区域对应的地图目录中。

要访问一组地图瓦片的图层数据，请使用 hereHDLMReader 对象。有关 hereHDLMReader 工作流程的信息，请参阅 Read and Visualize HERE HD Live Map Data。

这些图层分为以下模型：

  • Road Centerline Model — 提供道路拓扑、形状几何和其他道路级属性

  • HD Lane Model — 包含车道拓扑、高精度几何和车道级属性

  • HD Localization Model — 包括多个特征，如道路标志，以支持定位策略

可用的图层因地理区域而异，因此并非每个地图瓦片都有所有图层可用。当您在 hereHDLMReader 对象上调用 read 函数并指定地图图层名称时，该函数将以对象形式返回图层数据。有关这些图层对象的更多信息，请参阅 read 函数的参考页面。

道路中心线模型

道路中心线模型表示道路网络的拓扑结构。它由对应于街道的链接和对应于交叉口和死胡同的节点组成。对于每个地图瓦片，该模型中的图层包含有关这些链接和节点的信息，例如道路网络的二维线几何、速度属性和路由属性。

该图显示了 TopologyGeometry 图层的绘图，可视化了地图瓦片中节点和链接的二维线几何。

以下是 hereHDLMReader 对象可以读取的Road Centerline Model的地图图层。

Road Centerline Model Layers Description
TopologyGeometry

道路的拓扑和二维线几何。该图层还包含地图瓦片中链接（街道）和节点（交叉口和死胡同）的定义。
RoutingAttributes

与导航和条件相关的道路属性。这些属性以参数化方式映射到拓扑图层中的二维折线几何。

RoutingLaneAttributes

核心导航车道属性和条件，例如道路上的车道数。这些值以参数化方式映射到道路链接上的二维折线。

SpeedAttributes

与速度相关的道路属性，例如速度限制。这些属性被映射到拓扑图层的二维折线几何。
AdasAttributes

精确几何测量，例如道路的坡度、高程和曲率。使用这些数据开发高级驾驶辅助系统 (ADAS)。
ExternalReferenceAttributes

对其他HERE地图的外部链接、节点和拓扑的引用。
LaneRoadReferences (also part of HD Lane Model)

道路和车道组的引用和范围信息。使用这些数据在Road Centerline Model和HD Lane Model之间进行位置转换。

HD Lane模型

HD Lane模型表示车道组（即链接（街道）内的车道）的拓扑和几何结构。在该模型中，车道的形状通过2D和3D位置建模，并支持厘米级的准确性。该模型提供了几个车道属性，包括车道类型、行驶方向以及车道边界的颜色和样式。

该图显示了 LaneTopology 图层对象的绘图，可视化了地图瓦片中车道组及其连接器的二维线几何。

以下是 hereHDLMReader 对象可以读取的高清车道模型的地图图层。

HD Lane Model Layers Description
LaneTopology

HD Lane模型的拓扑，包括车道组、车道组连接器、车道和车道连接器的拓扑。该图层还包含车道模型的简化二维边界几何，用于确定地图瓦片的关联性和溢出。

LaneGeometryPolyline

由一组连接成折线的三维点组成的三维车道几何。

LaneAttributes

车道级别的属性，例如行驶方向和车道类型。
LaneRoadReferences (also part of Road Centerline Model)

道路和车道组的引用和范围信息。用于在Road Centerline模型和HD Lane模型之间进行位置转换。

HD定位模型

HD定位模型包含了交通标志、障碍物和杆等数据，帮助自动驾驶车辆准确地确定其在道路网络中的位置。

这个表格显示了 hereHDLMReader 对象可以读取的HD Localization模型的地图图层。该阅读器不支持对该图层数据的可视化。

HD Localization Model Layers Description
LocalizationBarrier

道路上发现的护栏（如护栏和Jersey护栏）的位置、尺寸和属性
LocalizationPole

沿着道路或悬挂在道路上的交通信号灯杆和其他杆的位置、尺寸和属性
LocalizationSign

沿着道路上发现的交通标志面的位置、尺寸和属性
