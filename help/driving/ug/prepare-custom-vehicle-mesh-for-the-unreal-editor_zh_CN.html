为虚幻编辑器准备自定义车辆网格

此示例展示如何创建与 Automated Driving Toolbox™ Interface for Unreal Engine® 4 Projects 支持包中的项目兼容的车辆网格。可以在 Simulation 3D Vehicle with Ground Following 模块中指定网格，以便在运行仿真时在虚幻编辑器中可视化车辆。

开始之前，请安装适用于虚幻引擎 4 项目的 Automated Driving Toolbox Interface for Unreal Engine 4 Projects 支持包。请参阅 Install Support Package for Customizing Scenes。

要创建兼容的自定义车辆网格，请按照以下工作流程步骤操作。

步骤

描述

第 1 步：设置骨骼层次结构

在三维创建环境中，设置车辆网格骨骼层次结构并指定零件名称。
第 2 步：分配材料

（可选）将材料分配给车辆零件。
第 3 步：导出网格和骨架

将车辆网格和骨架导出为 .fbx 文件格式。
第 4 步：将网格导入虚幻编辑器

将车辆网格导入到虚幻编辑器中
第 5 步：设置模块参数

使用 Simulation 3D Vehicle with Ground Following block 模块参数

Note

为了创建网格，本示例使用三维创建软件 Blender® 2.80 版本。

第 1 步：设置骨骼层次结构

  1. 将车辆网格导入三维建模工具，例如 Blender，使用 Umodel 将 AutoVrtlEnv\Content\Environment\Vehicles\Muscle\Meshes\SK_MuscleCar.uasset 解包，导出成 .gltf 格式。

  2. 为了确保该网格与 Automated Driving Toolbox Interface for Unreal Engine 4 Projects 支持包界面中的动画组件兼容，请对网格中的车辆部件使用此命名约定。

    车辆零件 名称

    底盘

    VehicleBody

    左前轮

    Wheel_FL

    右前轮

    Wheel_FR

    左后轮

    Wheel_RL

    右后轮

    Wheel_RR

    方向盘

    Wheel_Steering

    左前灯

    Lights_Headlight_Left

    右前灯

    Lights_Headlight_Right

    左侧指示灯

    Indicator_L

    右侧指示灯

    Indicator_R

    车牌号

    Vehicle_Plate

    刹车灯

    Lights_Brake

    倒车灯

    Lights_Reverse

    左前制动钳

    BrakeCaliper_FL

    右前制动钳

    BrakeCaliper_FR

    左后制动钳

    BrakeCaliper_RL

    右后制动钳

    BrakeCaliper_RR

  3. 设置车身对象， VehicleBody 作为车轮对象和其他车辆对象的父对象。

第 2 步：分配材料

（可选）将材料槽分配给车辆零件。在此示例中，网格使用一种材料用于底盘，另一种材料用于四个轮子。

  1. 创建材料槽并将其分配给车辆底盘。确认第一个车辆槽位对应车身。

  2. 创建材料槽并将其分配给轮子。

第 3 步：导出网格和骨架

将网格和骨架导出到 .fbx 文件格式。例如，在 Blender 中：

  1. 在 Object Types 窗格中，选择 Armature 和 Mesh。

  2. 在 Transform 窗格中，设置：

    • 缩放（Scale）至 1.00

    • 缩放应用（Apply Scalings）到 All Local

    • Forward to X Forward

    • Up to Z Up

    选择 Apply Unit。

  3. 在几何图形 Geometry 窗格上：

    • 将平滑（Smoothing）设置为 Face

    • 选择 Apply Modifiers

  4. 在 Armature 窗格上，设置：

    • Primary Bone Axis 设置为 X Axis

    • Secondary Bone Axis 设置为 Z Axis

    选择 Export FBX。

第 4 步：将网格导入虚幻编辑器

  1. 在虚幻编辑器中打开虚幻引擎项目 AutoVrtlEnv.uproject。

  2. 在编辑器中，将 FBX® 文件导入为骨架网格物体。将骨架（Skeleton）分配给 SK_PassengenerVehicle_Skeleton 资产。

第 5 步：设置模块参数

在 Simulink® 模型中，设置这些 Simulation 3D Vehicle with Ground Following 模块参数：

  • Type 设置为 Custom.

  • Path 设置为包含导入网格的虚幻引擎工程路径。
