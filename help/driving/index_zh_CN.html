自动驾驶工具箱

设计、仿真和测试高级驾驶辅助系统和自动驾驶系统
  • Release Notes
  • PDF Documentation
  • Release Notes
  • PDF Documentation

自动驾驶工具箱™ 提供用于设计、仿真和测试高级驾驶辅助系统和自动驾驶系统的算法和工具。可以设计和测试视觉和激光雷达感知系统，以及传感器融合、路径规划和车辆控制器。可视化工具包括鸟瞰图和传感器覆盖范围、检测和跟踪，以及视频、激光雷达和地图的显示。该工具箱可让导入并使用 HERE 高清实时地图数据和 ASAM OpenDRIVE® 路网。

使用 Ground Truth Labeler 应用程序，可以自动标记真值，以训练和评估感知算法。对于感知、传感器融合、路径规划和控制逻辑的硬件在环 (Hardware-In-the-Loop, HIL) 测试和桌面仿真，可以生成和模拟驾驶情景。可以在逼真的三维环境中模拟摄像头、雷达和激光雷达传感器输出，并在 2.5D 模拟环境中模拟物体和车道边界的传感器检测。

自动驾驶工具箱提供常见高级驾驶辅助系统和自动驾驶功能的参考应用示例，包括前向碰撞警告、自动紧急制动、自适应巡航控制、车道保持辅助和泊车服务。该工具箱支持用于快速原型设计和硬件在环测试的 C/C++ 代码生成，并支持传感器融合、跟踪、路径规划和车辆控制器算法。
