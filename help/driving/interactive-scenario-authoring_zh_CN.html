交互式场景编写

通过使用 Driving Scenario Designer 应用程序交互式地创建驾驶场景

Driving Scenario Designer 应用程序使您能够使用交互界面设计合成驾驶场景。您可以在场景中包括道路、参与者和传感器，并从场景中生成合成传感器检测。该应用程序还提供了各种预先构建的场景，用于在不同条件下测试您的驾驶算法。

Apps

Driving Scenario Designer 设计驾驶场景，配置传感器，并生成合成数据

Topics

  • Create Driving Scenario Interactively and Generate Synthetic Sensor Data

    使用 Driving Scenario Designer 应用程序创建驾驶场景，并从场景中生成传感器检测和点云数据。

  • Create Roads with Multiple Lane Specifications Using Driving Scenario Designer

    使用 Driving Scenario Designer 应用程序创建具有多个车道规格的道路。

  • Generate INS Sensor Measurements from Interactive Driving Scenario

    使用 Driving Scenario Designer 应用程序创建的驾驶场景生成融合的惯性导航和GPS测量数据，并将数据导出到 MATLAB® 中。
