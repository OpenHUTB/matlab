场景导入和导出

从诸如HERE HD Live Map之类的地图服务导入道路，并使用ASAM OpenX文件格式导入和导出场景

使用 drivingScenario 对象或 Driving Scenario Designer 应用程序，从 ASAM OpenDRIVE®文件和Web地图服务（如OpenStreetMap®、HERE HD Live Map和Zenrin Japan Map API 3.0（Itsumo NAVI API 3.0））导入道路网络。您可以编辑导入的道路网络，添加参与者，并构建场景。然后，您可以将驾驶场景导出为ASAM OpenDRIVE和ASAM OpenSCENARIO® 格式。

Functions

drivingScenario 创建驾驶场景
roadNetwork 向驾驶场景添加道路网络
export 将驾驶场景导出为ASAM OpenDRIVE或ASAM OpenSCENARIO文件
hereHDLMCredentials 设置或删除HERE HD Live Map凭证
zenrinCredentials 设置或删除Zenrin Japan Map API 3.0（Itsumo NAVI API 3.0）凭证

Apps

Driving Scenario Designer 设计驾驶场景，配置传感器，并生成合成数据

Topics

ASAM OpenX File Formats

  • Import ASAM OpenDRIVE Roads into Driving Scenario
    将ASAM OpenDRIVE道路网络中的道路和车道导入到 Driving Scenario Designer 应用程序或 drivingScenario 对象中。
  • Export Driving Scenario to ASAM OpenDRIVE File
    将驾驶场景中的道路、车道、交叉口和参与者导出到ASAM OpenDRIVE文件。
  • Export Driving Scenario to ASAM OpenSCENARIO File
    将驾驶场景中的道路网络、参与者和轨迹导出到ASAM OpenSCENARIO文件。

Web Map Services

  • Import HERE HD Live Map Roads into Driving Scenario
    将HERE HD Live Map Web服务中的道路数据导入到 Driving Scenario Designer 应用程序或 drivingScenario 对象中。
  • Import OpenStreetMap Data into Driving Scenario
    将 OpenStreetMap 文件中的地图数据导入到 Driving Scenario Designer 应用程序或 drivingScenario 对象中。
  • Import Zenrin Japan Map API 3.0 (Itsumo NAVI API 3.0) into Driving Scenario
    将 Zenrin Japan Map API 3.0 (Itsumo NAVI API 3.0) Web服务中的道路数据导入到 Driving Scenario Designer 应用程序或 drivingScenario 对象中。
