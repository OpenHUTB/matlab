plot

绘制驾驶场景

collapse all in page

语法

plot(scenario)
plot(scenario,Name,Value)

描述

example

plot(scenario) 创建一个三维图，使用正交透视，从驾驶场景 scenario 正上方的视角观察。

example

plot(scenario,Name,Value) 使用一个或多个名称-值对来指定选项。例如，您可以使用这些选项在图中显示道路中心线和参与者的路径点。

例子

创建和显示道路边界

Open Live Script

创建一个驾驶场景，其中包含在场景的世界坐标中指定的八字形道路。将场景的世界坐标转换为自我车辆的坐标系统。

创建一个空的驾驶场景。

scenario = drivingScenario;

向场景中添加一个八字形道路，并显示场景。

roadCenters = [0  0  1
             20 -20  1
             20  20  1
            -20 -20  1
            -20  20  1
              0   0  1];

roadWidth = 3;
bankAngle = [0 15 15 -15 -15 0];
road(scenario,roadCenters,roadWidth,bankAngle);
plot(scenario)

向场景中添加一个自车。将车辆定位在世界坐标 (20, -20) 处，并使其朝向以 -15 度的偏航角。

ego = actor(scenario,'ClassID',1,'Position',[20 -20 0],'Yaw',-15);

使用 roadBoundaries 函数获取自车坐标系下的道路边界。将自车指定为输入参数。

rbEgo1 = roadBoundaries(ego);

在鸟瞰图上显示结果。

bep = birdsEyePlot;
lbp = laneBoundaryPlotter(bep,'DisplayName','Road');
plotLaneBoundary(lbp,rbEgo1)

使用 roadBoundaries 函数获取世界坐标系下的道路边界。将场景指定为输入参数。

rbScenario = roadBoundaries(scenario);

使用 driving.scenario.roadBoundariesToEgo 函数将道路边界转换为自车坐标系下的坐标。

rbEgo2 = driving.scenario.roadBoundariesToEgo(rbScenario,ego);

在鸟瞰图上显示道路边界。

bep = birdsEyePlot;
lbp = laneBoundaryPlotter(bep,'DisplayName','Road boundaries');
plotLaneBoundary(lbp,{rbEgo2})

创建包含多个参与者和道路的驾驶场景

Open Live Script

创建一个驾驶场景，包含一个弯道、两条直道和两个参与者：一辆汽车和一辆自行车。两个参与者都在道路上行驶60秒。

创建驾驶场景对象。

scenario = drivingScenario('SampleTime',0.1','StopTime',60);

使用道路中心点创建弯道，该弯道沿着半径为800米的圆弧。圆弧起始角度为0°，结束角度为90°，采样间隔为5°。

angs = [0:5:90]';
R = 800;
roadcenters = R*[cosd(angs) sind(angs) zeros(size(angs))];
roadwidth = 10;
cr = road(scenario,roadcenters,roadwidth);

添加两条默认宽度的直道，使用每个端点的道路中心点。在第一条直道上，两侧道路边缘添加障碍物。

roadcenters = [700 0 0; 100 0 0];
sr1 = road(scenario,roadcenters);
barrier(scenario,sr1)
barrier(scenario,sr1,'RoadEdge','left')
roadcenters = [400 400 0; 0 0 0];
road(scenario,roadcenters);

获取道路边界。

rbdry = roadBoundaries(scenario);

在场景中添加一辆汽车和一辆自行车。将汽车放置在第一条直道的起始位置。

car = vehicle(scenario,'ClassID',1,'Position',[700 0 0], ...
    'Length',3,'Width',2,'Height',1.6);

将自行车放置在道路上更远的位置。

bicycle = actor(scenario,'ClassID',3,'Position',[706 376 0]', ...
    'Length',2,'Width',0.45,'Height',1.5);

绘制场景。

plot(scenario,'Centerline','on','RoadCenters','on');
title('Scenario');

显示参与者的姿势和配置文件。 

allActorPoses = actorPoses(scenario)
allActorPoses=242×1 struct array with fields:
    ActorID
    Position
    Velocity
    Roll
    Pitch
    Yaw
    AngularVelocity


allActorProfiles = actorProfiles(scenario)
allActorProfiles=242×1 struct array with fields:
    ActorID
    ClassID
    Length
    Width
    Height
    OriginOffset
    MeshVertices
    MeshFaces
    RCSPattern
    RCSAzimuthAngles
    RCSElevationAngles

由于这个场景中有障碍物，每个障碍物段都被视为一个参与者，actorPoses 和 actorProfiles 函数将返回所有静止和非静止参与者的姿势。为了仅获取车辆和自行车等非静止参与者的姿势和配置文件，首先使用 scenario.Actors.ActorID 属性获取它们对应的参与者ID。

movableActorIDs = [scenario.Actors.ActorID];

然后，使用这些ID来筛选仅包含非静止参与者姿势和配置文件的结果。

movableActorPoseIndices = ismember([allActorPoses.ActorID],movableActorIDs);

movableActorPoses = allActorPoses(movableActorPoseIndices)
movableActorPoses=2×1 struct array with fields:
    ActorID
    Position
    Velocity
    Roll
    Pitch
    Yaw
    AngularVelocity


movableActorProfiles = allActorProfiles(movableActorPoseIndices)
movableActorProfiles=2×1 struct array with fields:
    ActorID
    ClassID
    Length
    Width
    Height
    OriginOffset
    MeshVertices
    MeshFaces
    RCSPattern
    RCSAzimuthAngles
    RCSElevationAngles

输入参数

collapse all

scenario — 驾驶场景
drivingScenario object

驾驶场景，以 drivingScenario 对象的形式进行指定。

名称-值参数

将可选的参数对指定为 Name1=Value1,...,NameN=ValueN 的形式，其中 Name 是参数名称，Value 是相应的值。名称-值参数必须出现在其他参数之后，但是参数对的顺序无关紧要。

在R2021a之前，使用逗号将每个名称和值分隔开，并用引号括住 Name。

Example: plot(sc,'Centerline','on','RoadCenters','on') 显示每个道路段的中心线和道路中心点。

Parent — 绘制图形的坐标轴
Axes object

用于绘制图形的坐标轴，指定为由 'Parent' 和 Axes 对象组成的逗号分隔对。如果不指定 Parent，将创建一个新的图形。

Centerline — 显示道路的中心线
'off' (default) | 'on'

显示道路的中心线，指定为由 'Centerline' 和 'off' 或 'on' 组成的逗号分隔对。中心线沿着每个道路段的中间走向。中心线在交叉口或道路分叉等区域是不连续的。

RoadCenters — 显示道路中心
'off' (default) | 'on'

显示道路中心，指定为由'RoadCenters'和 'off' 或 'on'组成的逗号分隔对。道路中心定义了绘图中显示的道路。

Waypoints — 显示参与者的路径点
'off' (default) | 'on'

显示参与者的路径点，指定为由 'Waypoints' 和 'off' 或 'on'组成的逗号分隔对。路径点定义了参与者的轨迹。

Meshes — 显示参与者的网格模型
'off' (default) | 'on'

显示参与者的网格模型而不是长方体，指定为由 'Meshes' 和 'off' 或 'on'组成的逗号分隔对。

ActorIndicators — 绘制指示器的参与者周围
[] (default) | vector of ActorID integers

绘制指示器的参与者周围，指定为由 'ActorIndicators' 和一个包含 ActorID 整数的向量组成的逗号分隔对。驾驶场景绘图会在具有指定 ActorID 值的参与者周围绘制圆圈。每个圆圈的颜色与其所环绕的参与者相同。这些圆圈不表示传感器覆盖区域。

使用这个名称-值对，在包含多辆车辆的驾驶场景中突出显示自车。

ParkingLotEdges — 显示停车场边缘编号
'off' (default) | 'on'

显示停车场边缘编号，指定为逗号分隔的一对，包括'ParkingLotEdges'和'off'或'on'。边缘编号的顺序基于每个停车场中定义的顶点的顺序。

建议

  • 要旋转任何绘图，在图形窗口中选择 View > Camera Toolbar。
