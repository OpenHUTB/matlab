record

运行驾驶场景并记录参与者状态

collapse all in page

语法

rec = record(scenario)

描述

example

rec = record(scenario) 返回一个记录了驾驶场景 scenario 仿真中参与者状态的记录 rec。要记录一个场景，您必须定义至少一个参与者的轨迹。

例子

collapse all

从驾驶场景中记录参与者的位置姿态

Open Live Script

创建一个驾驶场景，其中一辆汽车在一条双车道的道路上超越一辆静止的汽车。

scenario = drivingScenario;
road(scenario,[0 0; 10 0; 53 -20],'lanes',lanespec(2));
plot(scenario,'Waypoints','on');
stationaryCar = vehicle(scenario,'ClassID',1,'Position',[25 -5.5 0],'Yaw',-22);

passingCar = vehicle(scenario,'ClassID',1);
waypoints = [1 -1.5; 16.36 -2.5; 17.35 -2.765; ...
            23.83 -2.01; 24.9 -2.4; 50.5 -16.7];
speed = 15; % m/s
smoothTrajectory(passingCar,waypoints,speed);

记录驾驶场景仿真。

rec = record(scenario);

比较仿真开始和结束时超车汽车的记录姿态。

输入参数

collapse all

scenario — Driving scenario
drivingScenario object

驾驶场景，指定为 drivingScenario 对象。

输出参数

collapse all

rec — 在仿真过程中记录参与者状态
M-by-1 vector of structures

在仿真过程中记录的参与者状态，以 M×1 的结构向量形式返回。M是模拟中的时间步数。每个结构对应一个模拟时间步。

rec 结构具有以下字段：

Field Description Type
SimulationTime 每个时间步的仿真时间 实数标量
ActorPoses 场景坐标系中的参与者姿态 ActorPoses 结构的 N×1 向量，其中 N 是参与者的数量，包括车辆。

每个 ActorPoses 结构具有以下字段。

Field Description
ActorID

场景定义的参与者标识符，指定为正整数。
Position

参与者的位置，指定为实值向量，形式为 [x y z]。单位为米。
Velocity

参与者在 x、y 和 z 方向上的速度(v)，指定为实值向量，形式为 [vx vy vz]。单位为米每秒。
Roll

参与者的滚转角度，指定为实值标量。单位为度。
Pitch

参与者的俯仰角度，指定为实值标量。单位为度。
Yaw

参与者的偏航角度，指定为实值标量。单位为度。
AngularVelocity

参与者在 x、y 和 z 方向上的角速度(ω)，指定为实值向量，形式为[ωxωy ωz]。单位为度每秒。

要查看这些结构字段的完整定义，请参阅 actor 和 vehicle 函数。

Data Types: struct
