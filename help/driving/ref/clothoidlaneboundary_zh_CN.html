clothoidLaneBoundary

布洛伊德形状的车道边界模型

expand all in page

描述

一个 clothoidLaneBoundary 对象包含有关布洛伊德形状车道边界模型的信息。布洛伊德曲线是一种曲线类型，其曲率的变化率与距离成线性关系。

创建

语法

bdry = clothoidLaneBoundary
bdry = clothoidLaneBoundary(Name,Value)

描述

bdry = clothoidLaneBoundary 创建一个带有默认属性值的布洛伊德车道边界模型，命名为 bdry。

example

bdry = clothoidLaneBoundary(Name,Value) 使用一个或多个名称-值对设置属性。例如，clothoidLaneBoundary('BoundaryType','Solid') 创建一个具有实心车道边界的布洛伊德车道边界模型。请将每个属性名称用引号括起来。

属性

expand all

Curvature — Lane boundary curvature
0 (default) | real scalar

车道边界曲率，指定为实数标量。该属性表示车道边界方向相对于距离的变化率。单位为度每米。

Example: -1.0

Data Types: single | double

CurvatureDerivative — Derivative of lane boundary curvature
0 (default) | real scalar

车道边界曲率的导数，指定为实数标量。该属性表示车道曲率相对于距离的变化率。单位为度每平方米。

Example: -0.01

Data Types: single | double

CurveLength — Length of lane boundary along road
0 (default) | nonnegative real scalar

车道边界沿道路的长度，指定为非负实数标量。单位为米。

Example: 25

Data Types: single | double

HeadingAngle — Initial lane boundary heading
0 (default) | real scalar

初始车道边界航向，指定为实数标量。车道边界的航向角是相对于自车的航向角而言的。单位为度。

Example: 10

Data Types: single | double

LateralOffset — Lateral offset of ego vehicle position from lane boundary
0 (default) | real scalar

自车位置相对于车道边界的横向偏移量，指定为实数标量。自车左侧车道边界的偏移量为正值，自车右侧车道边界的偏移量为负值。单位为米。在这张图像中，自车距离左车道边界的偏移量为1.5米，距离右车道边界的偏移量为2.1米。

Example: -1.2

Data Types: single | double

BoundaryType — Type of lane boundary marking
'Unmarked' (default) | 'Solid' | 'Dashed' | 'DoubleSolid' | 'DoubleDashed' | 'SolidDashed' | 'DashedSolid'

车道边界标记的类型，指定为以下值之一。

'Unmarked' 'Solid' 'Dashed' 'DoubleSolid' 'DoubleDashed' 'SolidDashed' 'DashedSolid'

No lane marking

Solid line

Dashed line

Two solid lines

Two dashed lines

Solid line on left, dashed line on right

Dashed line on left, solid line on right

Strength — Visibility of lane boundary marking
1 (default) | real scalar in the range [0, 1]

车道标线的可见性，指定为范围在[0, 1]内的实数标量。值为 0 表示标线不可见。值为 1 表示标线完全可见。对于双车道标线，两条线使用相同的可见性强度。

Example: 0.9

Data Types: single | double

XExtent — Extent of lane boundary marking along X-axis
[0 Inf] (default) | real-valued vector of the form [Xmin Xmax]

车道标线沿X轴的范围，指定为形式为 [Xmin Xmax] 的实值向量。单位为米。X轴垂直向上，并且在自车的前进方向上为正方向。

Example: [0 100]

Data Types: single | double

Width — Width of lane boundary marking
0 (default) | nonnegative real scalar

车道标线的宽度，指定为非负实数标量。对于双车道标线，该值适用于每条车道标线的宽度以及这些标线之间的距离。单位为米。

Example: 0.15

Data Types: single | double

对象函数

创建布洛伊德车道边界

Open Live Script

创建用于表示左侧和右侧车道边界的布洛伊德曲线，并绘制这些曲线。

创建左侧车道边界。

lb = clothoidLaneBoundary('BoundaryType','Solid', ...
'Strength',1,'Width',0.2,'CurveLength',40, ...
'Curvature',-0.8,'LateralOffset',2,'HeadingAngle',10);

创建几乎相同属性的右侧车道边界。

rb = lb;
rb.LateralOffset = -2;

创建一个鸟瞰图绘图。然后，创建车道边界绘图器并绘制边界。

bep = birdsEyePlot('XLimits',[0 50],'YLimits',[-10 10]);
lbPlotter = laneBoundaryPlotter(bep,'DisplayName','Left-lane boundary','Color','r');
rbPlotter = laneBoundaryPlotter(bep,'DisplayName','Right-lane boundary','Color','g');
plotLaneBoundary(lbPlotter,lb)
plotLaneBoundary(rbPlotter,rb);
grid
hold on

绘制沿着边界的选定点的坐标。

x = 0:5:50;
yl = computeBoundaryModel(lb,x);
yr = computeBoundaryModel(rb,x);
plot(x,yl,'ro')
plot(x,yr,'go')
hold off
