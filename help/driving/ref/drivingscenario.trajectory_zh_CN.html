trajectory

在驾驶场景中创建参与者或车辆的轨迹

collapse all in page

语法

trajectory(ac,waypoints)
trajectory(ac,waypoints,speed)
trajectory(ac,waypoints,speed,waittime)
trajectory(___,'Yaw',yaw)

描述

trajectory(ac,waypoints) 从一组路径点为参与者或车辆(ac)创建轨迹。

注意

轨迹函数生成的轨迹在路径点之间的加速度不连续性，导致大量的抖动。要生成一个平滑的、限制抖动的轨迹，请使用 smoothTrajectory 函数。

example

trajectory(ac,waypoints,speed) 还指定参与者或车辆沿着轨迹前进或反向运动的速度。

example

trajectory(ac,waypoints,speed,waittime) 在前面语法的基础上，该语法还指定了参与者或车辆的等待时间。使用这种语法可以在特定的路径点上暂停参与者或车辆，从而生成停车和起步驾驶场景。

example

trajectory(___,'Yaw',yaw) 除了前面语法中的任何输入参数组合之外，指定参与者或车辆在每个航路点的偏航方向角。

例子

collapse all

仿真具有不同速度的车辆轨迹

Open Script

创建一个驾驶场景，并在其中添加一个弯曲的双车道道路。

scenario = drivingScenario('SampleTime',0.05);
roadcenters = [5 0; 30 10; 35 25];
lspec = lanespec(2);
road(scenario,roadcenters,'Lanes',lspec);

向场景中添加一个车辆。设置一个轨迹，使车辆以不同的速度绕弯驶过。

v = vehicle(scenario,'ClassID',1);
waypoints = [6 2; 18 4; 25 7; 28 10; 31 15; 33 22];
speeds = [30 10 5 5 10 30];
trajectory(v,waypoints,speeds)

绘制场景并运行仿真。观察车辆在沿着弯道行驶时如何减速。

plot(scenario,'Waypoints','on','RoadCenters','on')
while advance(scenario)
    pause(0.1)
end

生成停车和起步驾驶场景

Open Live Script

创建一个由两条双车道道路组成的驾驶场景，这两条道路以直角相交。 

scenario = drivingScenario('StopTime',2.75);
roadCenters = [50 1 0; 2 0.9 0];
laneSpecification = lanespec(2,'Width',4);
road(scenario,roadCenters,'Lanes',laneSpecification);
roadCenters = [27 24 0; 27 -21 0];
road(scenario,roadCenters,'Lanes',laneSpecification);

向场景中添加一个自车。指定车辆在每个路径点的路径和速度值。在第二个路径点上设置车辆的等待时间。生成一个轨迹，使自车以指定的速度通过指定的路径点行驶。

egoVehicle = vehicle(scenario,'ClassID',1,'Position',[5 -1 0]);
waypoints = [5 -1 0; 16 -1 0; 40 -1 0];
speed = [30; 0; 30];
waittime = [0; 0.3; 0];
trajectory(egoVehicle,waypoints,speed,waittime);

向场景中添加一辆汽车。指定汽车在每个路径点的路径和速度值。在第二个路径点上设置汽车的等待时间。生成一个轨迹，使汽车以指定的速度通过指定的路径点行驶。

car = vehicle(scenario,'ClassID',1,'Position',[48 4 0],'PlotColor',[0.494 0.184 0.556], 'Name','Car');
waypoints = [47 3 0; 38 3 0; 10 3 0];
speed = [30; 0; 30];
waittime = [0; 0.3; 0];
trajectory(car,waypoints,speed,waittime);

向场景中添加一辆救护车。生成一个轨迹，使救护车以恒定的速度通过指定的路径点行驶。 

ambulance = vehicle(scenario,'ClassID',6,'Position',[25 22 0],'PlotColor',[0.466 0.674 0.188],'Name','Ambulance');
waypoints = [25 22 0; 25 13 0; 25 6 0; 26 2 0; 33 -1 0; 45 -1 0];
speed = 25;
trajectory(ambulance,waypoints,speed);

创建一个自定义的图形窗口来绘制场景。

fig = figure;
set(fig,'Position',[0,0,800,600]);
movegui(fig,'center');
hViewPnl = uipanel(fig,'Position',[0 0 1 1],'Title','Stop-and-Go Scenario');
hPlt = axes(hViewPnl);

绘制场景并运行仿真。自车和汽车会根据它们指定的等待时间停下来，以避免与救护车相撞。

plot(scenario,'Waypoints','on','RoadCenters','on','Parent',hPlt)
while advance(scenario)
    pause(0.1)
end

仿真汽车倒车进入停车位

Open Script

仿真一个驾驶场景，其中一辆汽车倒车进入停车位。

创建一个包含停车场的驾驶场景。

scenario = drivingScenario;
vertices = [0 9; 18 9; 18 -9; 0 -9];
parkingLot(scenario,vertices,ParkingSpace=parkingSpace);

创建一辆汽车并定义其轨迹。汽车向前行驶，停下来，然后倒车进入停车位。当汽车进入停车位时，它的偏航角度与起始位置相比逆时针旋转90度。

car = vehicle(scenario,ClassID=1);
waypoints = [9 -5; 9 5; 6 -1.3; 2 -1.3];
speed = [3; 0; -2; 0];
yaw = [90 90 180 180];
smoothTrajectory(car,waypoints,speed,Yaw=yaw)

绘制驾驶场景并显示轨迹的路径点。

plot(scenario,Waypoints="on")
while advance(scenario)
    pause(0.001)
end

定义行人的轨迹

Open Script

定义一个行人的轨迹，该行人在一个交叉路口处进行急转弯向右转。

创建一个驾驶场景。添加定义交叉路口的道路段。

scenario = drivingScenario;
roadCenters = [0 10; 0 -10];
road(scenario,roadCenters);
road(scenario,flip(roadCenters,2));

向场景中添加一个行人参与者。

pedestrian = actor(scenario, ...
    'ClassID',4, ...
    'Length',0.24, ...
    'Width',0.45, ...
    'Height',1.7, ...
    'Position',[-9 0 0], ...
    'RCSPattern',[-8 -8; -8 -8], ...
    'Mesh', driving.scenario.pedestrianMesh, ...
    'Name','Pedestrian');

定义行人的轨迹。行人接近交叉路口，稍作停顿，然后在交叉路口处进行急转弯向右转。为了定义急转弯向右转，指定在交叉路口附近的两个路径点。对于这些路径点，将第二个路径点的偏航角度设置为与第一个路径点呈90度角。

waypoints = [-9 0; -0.25 0; 0 -0.25; 0 -9];
speed = [1.5; 0; 0.5; 1.5];
yaw =  [0; 0; -90; -90];
waittime = [0; 0.2; 0; 0];
trajectory(pedestrian,waypoints,speed,waittime,'Yaw', yaw);

绘制驾驶场景并显示行人的路径点。

plot(scenario,'Waypoints','on')
while advance(scenario)
    pause(0.001)
end

输入参数

collapse all

ac — Actor
Actor object | Vehicle object

属于 drivingScenario 对象的参与者，指定为 Actor 或 Vehicle 对象。要创建这些对象，分别使用 actor 和 vehicle 函数。

waypoints — 轨迹路线点
real-valued N-by-2 matrix | real-valued N-by-3 matrix

轨迹路径点，指定为实值N × 2或N × 3矩阵，其中N为路径点的个数。

  • 如果 waypoints 是一个N × 2矩阵，则每个矩阵行表示路径点的 (x, y) 坐标。每个航路点的 z 坐标为零。

  • 如果 waypoints 是一个N × 3矩阵，则每个矩阵行表示路径点的 (x, y, z) 坐标。

路径点是在世界坐标系中表示的，单位为米。

Example: [1 0 0; 2 7 7; 3 8 8]

Data Types: single | double

speed — 参与者速度
30.0 | real-valued scalar | N-element real-valued vector

在 waypoints 中，参与者在每个路径点的速度被指定为实数标量或 N 元素实数向量。其中 N 是路径点的数量。

  • 当 speed 是标量时，速度在参与者运动过程中保持不变。

  • 当 speed 是向量时，向量值指定了每个路径点的速度。对于正向运动，请指定正的速度值。对于反向运动，请指定负的速度值。要改变运动方向，请在速度为 0 的路径点上分隔正速度和负速度。

速度在路径点之间进行插值。speed 可以在任意路径点为零，但不能在两个连续的路径点为零。单位为每秒米。

Example: [10 8 9] 指定速度为10米/秒、8米/秒和9米/秒。

Example: [10 0 -10] 指定前进时的速度为10米/秒，然后暂停，然后以相反的方向以10米/秒的速度行驶。

Data Types: single | double | int8 | int16 | int32 | int64 | uint8 | uint16 | uint32 | uint64

waittime — 参与者的暂停时间
0 (default) | N-element vector of nonnegative values

参与者的暂停时间，以非负值的N元素向量进行指定。N是路径点的数量。当您在特定路径点为参与者指定暂停时间时，必须将相应的速度值设置为 0。您可以在任何路径点将 waitime 设置为 0，但不能在连续的两个路径点上将 waittime 设置为非零值。单位为秒。

Data Types: single | double

yaw — 参与者的偏航方向角度
N-element real-valued vector

参与者在每个路径点的偏航方向角度，以N元素的实数向量进行指定，其中N是路径点的数量。单位为度，角度在逆时针方向上为正值。

如果不指定 yaw，则每个航路点的偏航是 NaN，这意味着偏航没有约束。

Example: [0 90] 指定在第一个航路点呈0度角、在第二个航路点呈90度角的参与者。

Example: [0 NaN] 在第一个路径点指定一个角度为0度的参与者。在第二个路径点上，参与者的偏航角度没有约束。

Data Types: single | double

算法

trajectory 函数用于创建参与者在场景中遵循的轨迹。轨迹由对象所经过的路径和路径上的速度组成。您可以使用N个二维或三维路径点来指定路径。每个路径点之间的N - 1个线段定义了一个曲线，其曲率随着沿着线段的距离线性变化。该函数通过匹配路径点两侧的曲率，将分段布洛伊德曲线拟合到路径点的 (x, y) 坐标上。对于非闭合曲线，第一个和最后一个路径点的曲率为零。如果第一个和最后一个路径点重合，则匹配端点之前和之后的曲率。轨迹的z坐标使用保持形状的分段三次样条曲线进行插值。

生成的轨迹在路径点之间的每个线段上产生分段恒定加速度的轨迹曲线。这些线段之间存在加速度不连续性。为了避免加速度的不连续性，可以使用 smoothTrajectory 函数来生成平滑的轨迹曲线。
