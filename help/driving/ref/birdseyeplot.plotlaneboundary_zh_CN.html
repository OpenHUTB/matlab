plotLaneBoundary

在鸟瞰图上显示车道边界

collapse all in page

语法

plotLaneBoundary(lbPlotter,boundaryCoords)
plotLaneBoundary(lbPlotter,boundaries)

描述

example

plotLaneBoundary(lbPlotter,boundaryCoords) 在鸟瞰图上显示来自边界坐标列表的车道边界。车道边界绘图器（lbPlotter）与 birdsEyePlot 对象相关联，并配置指定车道边界的显示。

删除与车道边界绘图器lbPlotter相关联的所有车道边界，请调用 clearData 函数并将 lbPlotter 指定为输入参数。

plotLaneBoundary(lbPlotter,boundaries) 从车道边界对象或车道边界对象数组 boundaries 中显示车道边界。

例子

创建并显示道路边界

Open Live Script

创建一个包含以场景的世界坐标指定的八字形道路的驾驶场景。将场景的世界坐标转换为自车坐标系。

创建一个空的驾驶场景。

scenario = drivingScenario;

向场景中添加一个八字形道路。显示场景。

roadCenters = [0  0  1
             20 -20  1
             20  20  1
            -20 -20  1
            -20  20  1
              0   0  1];

roadWidth = 3;
bankAngle = [0 15 15 -15 -15 0];
road(scenario,roadCenters,roadWidth,bankAngle);
plot(scenario)

向场景中添加一个自车。将车辆定位在世界坐标 (20, –20) 处，并将其方向设置为-15度的偏航角。

ego = actor(scenario,'ClassID',1,'Position',[20 -20 0],'Yaw',-15);

使用 roadBoundaries 函数获取自车坐标系中的道路边界。将自车指定为输入参数。

rbEgo1 = roadBoundaries(ego);

在鸟瞰图上显示结果。

bep = birdsEyePlot;
lbp = laneBoundaryPlotter(bep,'DisplayName','Road');
plotLaneBoundary(lbp,rbEgo1)

使用roadBoundaries函数获取世界坐标系中的道路边界。将场景指定为输入参数。

rbScenario = roadBoundaries(scenario);

使用 driving.scenario.roadBoundariesToEgo 函数将道路边界转换为自车坐标系中的坐标。

rbEgo2 = driving.scenario.roadBoundariesToEgo(rbScenario,ego);

在鸟瞰图上显示道路边界。

bep = birdsEyePlot;
lbp = laneBoundaryPlotter(bep,'DisplayName','Road boundaries');
plotLaneBoundary(lbp,{rbEgo2})

输入参数

collapse all

lbPlotter — Lane boundary plotter
LaneBoundaryPlotter object

车道边界绘图器是一个 LaneBoundaryPlotter 对象，它被指定为 birdsEyePlot 对象的 Plotters 属性的一部分，并配置了在鸟瞰图中显示指定车道边界的方式。要创建这个对象，请使用 laneBoundaryPlotter 函数。

boundaryCoords — Lane boundary coordinates
cell array of M-by-2 real-valued matrices

车道边界坐标是一个由Mx2的实数矩阵组成的单元数组。每个矩阵表示不同车道边界的坐标。M是车道边界中的坐标数，对于每个车道边界可能不同。每一行表示曲线的 (X, Y) 位置。正X方向指向车辆中心的前方。正Y方向指向车辆原点的左侧，车辆原点是后轴的中心，如车辆坐标系图中所示。

boundaries — Lane boundaries
lane boundary object | array of lane boundary objects

车道边界是指定为车道边界对象或车道边界对象数组的形式。有效的车道边界对象包括parabolicLaneBoundary, cubicLaneBoundary 和 clothoidLaneBoundary。如果指定了车道边界对象数组，则所有对象必须是相同类型的。高度表示的Z数据将被忽略。
