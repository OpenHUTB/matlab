meshPlotter

用于鸟瞰图的网格绘图器

collapse all in page

语法

mPlotter = meshPlotter(bep)
mPlotter = meshPlotter(bep,Name,Value)

描述

example

mPlotter = meshPlotter(bep) 创建一个 MeshPlotter 对象，用于配置在鸟瞰图上显示 meshes。MeshPlotter 对象存储在输入的 birdsEyePlot 对象（bep）的 Plotters 属性中。要显示对象的网格表示，请使用 plotMesh 函数。

mPlotter = meshPlotter(bep,Name,Value) 使用一个或多个 Name,Value 对参数设置属性。例如，meshPlotter(bep,'FaceAlpha',1) 将网格面设置为完全不透明。

例子

collapse all

在驾驶场景中显示参与者的网格表示

Open Script

使用参与者的网格表示而不是立方体表示在驾驶场景中显示参与者。

创建一个驾驶场景，并向场景中添加一个长度为25米的直线道路。

scenario = drivingScenario;
roadcenters = [0 0 0; 25 0 0];
road(scenario,roadcenters);

向场景中添加一个行人和一辆车辆。使用预先构建的网格来指定参与者的网格尺寸。

  • 将行人的网格指定为 driving.scenario.pedestrianMesh 对象。

  • 将车辆的网格指定为 driving.scenario.carMesh 对象。

p = actor(scenario,'ClassID',4, ...
            'Length',0.2,'Width',0.4, ...
            'Height',1.7,'Mesh',driving.scenario.pedestrianMesh);

v = vehicle(scenario,'ClassID',1, ...
            'Mesh',driving.scenario.carMesh);

为行人和车辆添加轨迹。

  • 指定行人以每秒1米的速度穿过道路。

  • 指定车辆以每秒10米的速度沿着道路行驶。

waypointsP = [15 -3 0; 15 3 0];
speedP = 1;
smoothTrajectory(p,waypointsP,speedP);

wayPointsV = [v.RearOverhang 0 0; (25 - v.Length + v.RearOverhang) 0 0];
speedV = 10;
smoothTrajectory(v,wayPointsV,speedV)

为车辆添加一个自车视角的绘图。打开网格的显示。

chasePlot(v,'Meshes','on')

创建一个鸟瞰图，用于显示网格。同时创建一个网格绘图器和车道边界绘图器。然后运行仿真循环。

  1. 获取车辆所在道路的道路边界。

  2. 获取参与者网格的顶点、面和颜色，相对于车辆的位置。

  3. 在鸟瞰图上绘制道路边界和参与者网格。

  4. 暂停场景，以便给绘图留出更新的时间。追踪绘图每次推进场景时都会更新。

bep = birdsEyePlot('XLim',[-25 25],'YLim',[-10 10]);
mPlotter = meshPlotter(bep);
lbPlotter = laneBoundaryPlotter(bep);
legend('off')

while advance(scenario)

   rb = roadBoundaries(v);

   [vertices,faces,colors] = targetMeshes(v);

   plotLaneBoundary(lbPlotter,rb)
   plotMesh(mPlotter,vertices,faces,'Color',colors)

   pause(0.01)
end

输入参数

collapse all

bep — Bird’s-eye plot
birdsEyePlot object

Bird’s-eye plot, specified as a birdsEyePlot object.

Name-Value Arguments

将可选的参数对指定为 Name1=Value1,...,NameN=ValueN的形式，其中 Name 是参数名称，Value 是相应的值。Name-value 参数必须出现在其他参数之后，但是参数对的顺序无关紧要。

在R2021a之前，使用逗号将每个名称和值分隔开，并将 Name 用引号括起来。

Example: meshPlotter('FaceAlpha',0.5) 将网格的面设置为50%的透明度。

FaceAlpha — Transparency of mesh faces
0.75 (default) | scalar in the range [0, 1]

网格面的透明度，指定为逗号分隔的参数对，包括 'FaceAlpha' 和取值范围为[0, 1]的标量。值为 0 表示网格面完全透明，值为 1 表示网格面完全不透明。

Tag — Tag associated with plotter object
'PlotterN' (default) | character vector | string scalar

与绘图器对象关联的标签，指定为逗号分隔的参数对，包括 'Tag' 和字符向量或字符串标量。默认值为 'PlotterN'，其中 N 是与输入 birdsEyePlot 对象关联的第 N 个绘图器对应的整数。

输出参数

collapse all

mPlotter — Mesh plotter
MeshPlotter object

网格绘图器，返回为 MeshPlotter 对象。您可以通过更改其属性值来修改此对象。属性名称对应于 meshPlotter 函数的名称-值对参数。

mPlotter 存储在输入 birdsEyePlot 对象（bep）的 Plotters 属性中。要绘制网格，请使用 plotMesh 函数。
