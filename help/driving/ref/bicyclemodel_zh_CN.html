Bicycle Model

实现一个单轨迹的3自由度刚体车辆模型，用于计算纵向、横向和偏航运动

expand all in page

描述

Bicycle Model 模块实现了一个刚性的双轴单轨迹车辆模型，用于计算纵向、横向和偏航运动。该模块考虑了车体质量、空气阻力以及由于加速度和转向而导致的轴之间的重量分配。有两种类型的 Bicycle Model 模块。

Block Implementation

Bicycle Model - Velocity Input

  • 该模块假设外部纵向速度是准稳态的，因此纵向加速度近似为零。

  • 由于运动是准稳态的，该模块仅通过使用轮胎滑移角和线性侧向刚度来计算横向力。

Bicycle Model - Force Input

  • 该模块使用外部纵向力来加速或制动车辆。

  • 该模块通过使用轮胎滑移角和线性侧向刚度来计算横向力。

为了计算前轴和后轴上的法向力，该模块使用刚体车辆运动、悬挂系统力和风阻力。该模块解析刚体车身框架上的力和力矩分量。

Wrap Euler angles, wrapAng — Selection
off (default) | on

Wrap the Euler angles to the interval [-pi, pi]. For vehicle maneuvers that might undergo vehicle yaw rotations that are outside of the interval, consider deselecting the parameter if you want to:

  • Track the total vehicle yaw rotation.

  • Avoid discontinuities in the vehicle state estimators.
