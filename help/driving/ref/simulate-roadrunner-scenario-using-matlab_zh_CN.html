Simulate a RoadRunner Scenario Using MATLAB Functions

此示例使用:

  • Automated Driving Toolbox
  • RoadRunner
  • RoadRunner Scenario
Open Live Script

此示例演示如何使用 MATLAB 函数在 RoadRunner 场景中运行和可视化场景。您可以使用 MATLAB 函数以编程方式控制 RoadRunner 场景。可以执行的常见编程任务包括：

  • 打开并关闭 RoadRunner 场景应用程序。

  • 打开、关闭和保存场景、方案和项目。

  • 导入和导出方案。

RoadRunner 方案使您能够以交互方式设计和模拟方案中的代理。若要验证这些代理的行为，自动执行运行和分析方案模拟结果的过程通常很有帮助。在此示例中，您将学习如何使用自动驾驶工具箱®启动 RoadRunner 场景、配置和运行模拟，然后绘制模拟结果

若要运行此示例，必须：

  • 拥有自动驾驶工具箱®许可证。

  • 拥有RoadRunner许可证并安装产品。

  • 拥有 RoadRunner 场景许可证并安装产品。

  • 已创建 RoadRunner 项目文件夹。

设置环境以启动Roadrunner方案

要使用 MATLAB 函数以编程方式控制 RoadRunner 场景，请使用该对象。默认情况下，从您正在使用的平台（Windows® 或 Linux®）的默认安装文件夹中打开 RoadRunner。以下是按平台划分的默认安装位置：

  • Windows – C：\Program Files\RoadRunner R20NNx\bin\win64

  • Linux， Ubuntu® – /usr/local/RoadRunner_R20NNx/bin/glnxa64

R20NNx 是您正在使用的 MATLAB 版本。

如果您的 RoadRunner 场景安装位于与默认位置不同的位置，请使用 MATLAB API 自定义 RoadRunner 场景安装文件夹的默认值。

打开RoadRunner Scenario

您可以使用该函数创建对象并启动 RoadRunner 方案会话。该函数需要一个参数来指定 RoadRunner 项目的位置。RoadRunner 项目文件夹通常包含以下子文件夹：Assets, Exports, Project, Scenarios, Scenes.

通过指定创建项目的位置，使用roadrunner函数在RoadRunner中打开项目。此示例假定 RoadRunner 安装在其在 Windows 中的默认位置。

指定现有项目的路径。例如，此代码显示位于 C:\RR\MyProjectroadrunnerrrApp上的项目的路径。该函数返回一个对象，该对象提供用于执行基本工作流任务（如打开、关闭和保存场景和项目）的函数。

rrProj = "C:\RR\MyProject";
rrApp = roadrunner(rrProj,InstallationFolder="C:\Program Files\RoadRunner R2022b\bin\win64");

使用openScenario函数，通过指定rrApp对象和要打开的特定场景文件名，打开RoadRunner Scenario中的现有场景。例如，打开TrajectoryCutIn场景文件，该场景是RoadRunner项目中默认包含的一个场景。该函数通过MATLAB在RoadRunner Scenario应用程序中打开所需的场景。

openScenario(rrApp,"TrajectoryCutIn.rrscenario");

模拟场景

在将场景加载到RoadRunner Scenario后，可以使用createSimulation函数创建一个仿真对象，以实现仿真任务的自动化。仿真对象使您能够以编程方式与场景仿真进行交互。

将rrApp对象指定为createSimulation函数的输入参数。该函数将创建一个名为rrSim的仿真对象。

rrSim = createSimulation(rrApp);
Connection status: 1
Connected to RoadRunner Scenario server on localhost:59151, with client id {1c6fd5fc-736d-4cd9-9d60-84bc96444b95} 
使用set函数，并将rrSim对象、要设置的变量名和该变量的值作为输入参数，将仿真时间设置为最大10秒。
maxSimulationTimeSec = 10;set(rrSim,'MaxSimulationTime',maxSimulationTimeSec);

要启用仿真结果记录功能，使用set函数，并指定rrSim对象、要设置的变量名以及该变量的值为'on'。这将激活记录功能，使您能够在以后检索和绘制仿真结果。

set(rrSim,"Logging","on");

启动仿真。使用while循环来监视仿真的状态，并等待仿真完成。

set(rrSim,"SimulationCommand","Start");
while strcmp(get(rrSim,"SimulationStatus"),"Running")
    pause(1);
end

绘制车辆代理速度

在这个部分，您可以从仿真中获取演员的记录速度，并将它们的大小绘制成与仿真时间的关系图。

获取场景的记录结果。使用get函数，并将rrSim对象和"SimulationLog"作为输入参数。该函数将返回rrLog中的仿真日志，其中包含有关场景仿真的信息。

rrLog = get(rrSim,"SimulationLog");

TrajectoryCutIn场景包含两个演员。红色轿车的Actor ID设置为1，白色轿车的Actor ID设置为2。从仿真日志中获取这些演员的速度记录，以及相应的仿真时间。

velocityAgent1 = get(rrLog,'Velocity','ActorID',1);
velocityAgent2 = get(rrLog,'Velocity','ActorID',2);
time = [velocityAgent1.Time];

该函数将红色轿车和白色轿车的速度作为向量返回，并分别存储在变量velMagAgent1和velMagAgent2中。通过使用norm函数计算每个演员的速度大小。

velMagAgent1 = arrayfun(@(x) norm(x.Velocity,2),velocityAgent1);
velMagAgent2 = arrayfun(@(x) norm(x.Velocity,2),velocityAgent2);

使用plot函数将代理的速度与仿真时间关联起来，并对图表、x轴和y轴进行标注。

figure
hold on
plot(time,velMagAgent1,"r")
plot(time,velMagAgent2,"b")
grid on
title("Agent Velocities from RoadRunner Scenario")
ylabel("Velocity (m/sec)")
xlabel("Time (sec)")
legend("Actor ID = 1","Actor ID = 2")

注意到演员的速度与它们在RoadRunner Scenario的逻辑编辑器中的规格相对应。

绘制代理速度

绘制RoadRunner场景中的车道，并在地图上叠加车辆的位置。

通过使用getMap函数从RoadRunner获取HD Map规格。请注意，该函数返回一个结构，其中一个字段包含有关车道的信息。

hdMap = getMap(rrSim);
lanes = hdMap.map.lanes;

使用for循环遍历每个车道规格，并绘制车道坐标。

figure
hold on
for i = 1:numel(lanes)
    control_points = lanes(i).geometry.values;
    x_coordinates = arrayfun(@(cp) cp.x,control_points);
    y_coordinates = arrayfun(@(cp) cp.y,control_points);
    plot(x_coordinates, y_coordinates, 'black');
end
axis equal

提取车辆的位置并将其绘制在车道上。

poseActor1 = rrLog.get('Pose','ActorID',1);
positionActor1_x = arrayfun(@(x) x.Pose(1,4),poseActor1);
positionActor1_y = arrayfun(@(x) x.Pose(2,4),poseActor1);
plot(positionActor1_x,positionActor1_y,"r","LineWidth",2)

poseActor2 = rrLog.get('Pose','ActorID',2);
positionActor2_x = arrayfun(@(x) x.Pose(1,4),poseActor2);
positionActor2_y = arrayfun(@(x) x.Pose(2,4),poseActor2);
plot(positionActor2_x,positionActor2_y,"b","LineWidth",2)

title("Agent Positions from RoadRunner Scenario")
ylabel("Y (m)")
xlabel("X (m)")

figure(gcf)

关闭方案会话

停止与 RoadRunner 场景交互，请关闭模拟。然后关闭应用程序。

close(rrApp)

关闭打开的图形。

close all

进一步探索

在此示例中，您了解了使用 MATLAB 以编程方式连接到 RoadRunner 场景的基本功能。要进一步扩展此脚本，您可以：

  • 改变场景中的场景和车辆参与者。

  • 开发 MATLAB 和 Simulink 行为，发布参与者行为，在 RoadRunner 场景模拟中模拟行为，以及控制模拟和访问模拟参数。
