    获取参与者的运行时属性

    collapse all in page

      语法

      runtimeAttr = getAttribute(actorSim,'RuntimeAttrName')

      描述

      example

      runtimeAttr = getAttribute(actorSim,'RuntimeAttrName')

      返回Simulink.ActorSimulation对象actorSim的指定运行时属性。当您在MATLAB® System object™的stepImpl方法中使用这个函数时，该函数将返回前一个时间步中的状态，但参与者 ID 除外。

      Examples

      collapse all

      获取参与者的运行时属性

      为具有多个参与者的场景创建 Simulink.ScenarioSimulation 对象。

      rrApp = roadrunner('C:\Project\TestHighwayRoute'); 
openScenario(rrApp,'myScenario1'); 
sim = createSimulation(rrApp);

      开始模拟。

      set(sim,'SimulationCommand','Start');

      暂停模拟。

      set(sim,'SimulationCommand','Pause');

      从模拟中获取所有Actor。

      actorsim = get(sim, 'ActorSimulation');

      从列表中获取前两个参与者。

      one = actorsim(1);
two = actorsim(2);

      获取第一个演员的 ID。

      id = getAttribute(one,'ID')

      输入参数

      collapse all

      actorSim — Actor
      Simulink.ActorSimulation object

      从中返回运行时属性的参与者, 指定为 Simulink.ActorSimulation 对象。

      Example: actorSim

      RuntimeAttrName — 运行时属性
      ID | Pose | Velocity | AngularVelocity | WheelPoses | MapLocation| Children | Parent

      要检查的运行时属性的名称，指定为这些字符串之一

      • ID

      • Pose

      • Velocity

      • AngularVelocity

      • WheelPoses

      • MapLocation

      • Children

      • Parent

      输出参数

      collapse all

      runtimeAttr — 参与者运行时属性
      uint64 | double | struct

      actor 的运行时属性，作为对应于输入参数 RuntimeAttrName 的数据类型返回。

      执行时属性 描述
      ID 执行组件标识符，返回为 .uint64
      Pose 参与者在 RoadRunner 场景坐标系中的位置和方向，以 4 x 4 数组的形式返回。
      Velocity 执行组件在 x、y 和 z 方向上的速度以 1×3 向量返回。单位为米/秒。
      AngularVelocity 参与者在 x、y 和 z 方向上的角速度以 1×3 矢量的形式返回。单位单位为弧度 /秒。
      WheelPoses

      车辆类型Actor的每个车轮的运行时姿势，以多维4×4×N数组的形式返回。N是车辆型演员的车轮数。WheelPoses被返回用于车轮 从前轴的最左侧开始，向右移动。 第一个轴完成后，在向右移动之前会考虑第二个轴上最左侧的车轮，依此类推。
      MapLocation 车辆型参与者相对于其最近车道的位置，作为结构struct返回。
      Children

      指定 actor 的直系子项（如果有）作为 Simulink.ActorSimulation 对象的数组返回。

      如果 指定的 actor 没有任何子 actor，则空数组为 返回。
      Parent

      指定 actor 的直接父级作为 Simulink.ActorSimulation 对象的数组返回。

      如果指定的参与者本身是父级，则返回与世界参与者对应的 Simulink.ActorSimulation 对象。

      下表描述了地图位置结构的字段。

      字段名称 描述
      IsOnLane 指示车辆是否位于车道上的布尔值。如果车辆的模型原点位于该车道的左右边界之间，则认为车辆位于该车道的左侧和右侧边界之间。
      LocationOnLane 车辆的当前车道，作为结构返回。如果车辆横跨多条车道，则返回最对齐的车道。.

      该表描述了地图位置结构的 LocationOnLane 结构的字段。

      字段名称 描述
      LaneID 最对齐的通道的 UUID 标识符，以字符串形式返回。
      Position 车辆在车道中心折线上的位置（以 s 值为单位）。s 值是 [0， 1] 中的双精度值，表示三维折线上的位置，0 在折线的开头，1 在折线的末尾。
      Angle 车辆行驶方向与车道切线之间的夹角位置，以弧度为单位。
