parkingSpace

定义停车场的停车位

expand all in page

描述

parkingSpace 对象定义了在驾驶场景中要添加到停车场的停车位。您可以定义停车位的尺寸和角度，停车位的类型，停车车道标记的颜色，宽度和强度。要在将停车位添加到停车场之前将其可视化，请使用 plot 函数。要用创建的停车位填充停车场，请使用 parkingLot 或 insertParkingSpaces 函数。

创建

语法

space = parkingSpace
space = parkingSpace(Name=Value)

描述

example

space = parkingSpace 创建一个直线停车位，宽度为2.6米，长度为5.5米，带有白色车道标线。

example

space = parkingSpace(Name=Value) 使用名称-值参数设置属性。例如， space = parkingSpace(Type="Accessible",Width=3,Angle=60) 创建一个3米宽、60度角的无障碍停车位。

属性

expand all

Type — 停车位类型
"Parking" (default) | "NoParking" | "Accessible"

停车位类型，指定为“可停车”、“不可停车”或“无障碍”。此表显示了每种类型的默认停车位。

"Parking" "NoParking" "Accessible"

Width — 停车位宽度(米)
2.6 (default) | real scalar in range [0.5, 100]

停车位的宽度，以米为单位，指定为在范围[0.5, 100]内的实数标量。

Example: 3.6

Length — 停车位长度(米)
5.5 (default) | real scalar in range [0.5, 100]

停车位的长度，以米为单位，指定为在范围[0.5, 100]内的实数标量。

Example: 7.0

Angle — 停车位的角度（度）
90 (default) | real scalar in range [20, 160]

停车位的角度，以度为单位，以实标量表示，范围为[20,160]。Angle 按顺时针方向从车位的上边缘到左边缘测量。此表显示了样本 Angle 值及其相应的图。

Angle=60 Angle=90 Angle=120

Example: 45

MarkingColor — 停车位车道标线的颜色
RGB triplet | hexadecimal color code | color name | short color name

停车位车道标线的颜色，可以指定为RGB三元组、十六进制颜色代码、颜色名称或简短颜色名称。同一种颜色适用于该停车位的所有标线。

对于自定义颜色，请指定一个RGB三元组或十六进制颜色代码。

  • RGB三元组是一个包含三个元素的行向量，这些元素指定颜色的红色、绿色和蓝色分量的强度。强度必须在[0,1]范围内，例如，[0.4 0.6 0.7]。

  • 十六进制颜色代码是以井号 (#) 开头，后跟三个或六个十六进制数字的字符向量或字符串标量。这些数字的范围可以从 0 到 F。这些代码对大小写不敏感。因此，颜色代码 "#FF8800", "#ff8800", "#F80" 和 "#f80" 是等效的。

或者，您可以通过名称指定一些常见的颜色。下表列出了命名的颜色选项、相应的RGB三元组和十六进制颜色代码。

Color Name Short Name RGB Triplet Hexadecimal Color Code Appearance
"red" "r" [1 0 0] "#FF0000"

"green" "g" [0 1 0] "#00FF00"

"blue" "b" [0.5234 0.7695 0.8516] "#85C4D9"

"cyan" "c" [0 1 1] "#00FFFF"

"magenta" "m" [1 0 1] "#FF00FF"

"yellow" "y" [0.98 0.86 0.36] "#FADB5C"

"black" "k" [0 0 0] "#000000"

"white" "w" [1 1 1] "#FFFFFF"

默认的标线颜色取决于由 Type 指定的停车位类型。

Type MarkingColor Default
"Parking" [1 1 1] (white)
"NoParking" [0.98 0.86 0.36] (yellow)
"Accessible" [0.5234 0.7695 0.8516] (blue)

Example: [0.8 0.8 0.8]

MarkingWidth — 停车位车道标线的宽度（米）
0.15 (default) | positive real scalar

停车位车道标线的宽度，以米为单位，指定为正实数标量。同一宽度适用于该停车位的所有标线。

标线宽度（MarkingWidth）必须小于或等于由宽度（Width）指定的停车位宽度。

Example: 0.2

MarkingStrength — 停车位车道标线颜色的饱和度强度
1 (default) | real scalar in range [0, 1]

停车位车道标线颜色的饱和度强度，以实数标量表示，取值范围为[0, 1]。值为 0 表示标线颜色完全不饱和，即标线颜色与地面颜色相同。值 1 表示标线颜色完全饱和。

Example: 0.75

对象函数

plot 绘制停车位

例子

创建包含多种停车位类型的停车场

Open Live Script

创建一个包含停车位、禁停区域和无障碍停车位混合的停车场。

定义在停车场中使用的停车位。使用默认设置。绘制该停车位。

space = parkingSpace;
plot(space,Origin="off")

定义在停车场中使用的禁停区域。指定白色的颜色和宽度，宽度比默认停车位的宽度小一米。绘制该区域。

noSpace = parkingSpace(Type="NoParking",Width=(space.Width - 1),MarkingColor="White");
plot(noSpace,Origin="off")

定义在停车场中使用的无障碍停车位。指定宽度，宽度比默认停车位的宽度大一米。绘制该停车位。

accessibleSpace = parkingSpace(Type="Accessible",Width=(space.Width + 1));
plot(accessibleSpace,Origin="off")

创建一个包含非矩形布局的停车场的驾驶场景。绘制停车场，并显示可以添加停车位的边缘编号。

scenario = drivingScenario;
vertices = [0 0; 32 -10.5; 32 -50.5; 25 -57.5; 0 -57.5];
lot = parkingLot(scenario,vertices);
plot(scenario,ParkingLotEdges="on")

在停车场的前三条边上插入默认停车位。为了避免停车位重叠，对这些插入进行以下调整：

  • 在边缘1上，只插入10个停车位。

  • 在边缘2上，将停车位从边缘的第一个顶点向后偏移5米。

  • 在边缘3上，将停车位从边缘的第一个顶点向后偏移3米。

numSpaces = 10;
insertParkingSpaces(lot,space,numSpaces,Edge=1)
insertParkingSpaces(lot,space,Edge=2,Offset=5)
insertParkingSpaces(lot,space,Edge=3,Offset=3)

在停车场的中心位置，插入一个 2×10 的停车位网格，其中包含8列默认停车位，1列非停车区和1列无障碍停车位。

insertParkingSpaces(lot,[space noSpace accessibleSpace],[8 1 1],Position=[12 -18],Rows=2)

使用预定义的布局创建停车场

Open Live Script

通过使用预定义的布局以及配置这些布局的选项，您可以创建各种不同的停车场。

定义用于填充停车场的停车位。修改停车位的宽度、长度或角度，以及车道标线的宽度和强度。绘制停车位。

width = 2.6; % m
length = 5.5; % m
angle = 90; % deg
markingWidth = 0.15;
markingStrength = 1;

space = parkingSpace(Width=width, ...
                    Length=length, ...
                    Angle=angle, ...
                    MarkingWidth=markingWidth, ...
                    MarkingStrength=markingStrength);
plot(space)

创建一个包含 50×50米停车场的驾驶场景。指定预定义的停车场布局和最小行驶车道宽度。在给定最小车道宽度约束的情况下，生成的停车场填满尽可能多的适合布局的停车位。修改车位、布局类型、车道宽度，观察对停车场的影响。例如:

  • 当你增加停车位的大小或最小车道宽度时，停车场中间的停车位网格数量就会减少。

  • 如果您选择了 HorizontalWithEdges 或 VerticalWithEdges 布局，其中一侧的停车位数量会比其他侧少。这一侧包含一个宽度为 DrivingLaneWidth 的驾驶车道，用于车辆进入停车场。

scenario = drivingScenario;
vertices = [0 0; 50 0; 50 -50; 0 -50];
parkingLayout = "Vertical";
drivingLaneWidth = 2.6; % m

parkingLot(scenario,vertices, ...
    ParkingSpace=space, ...
    ParkingLayout=parkingLayout, ...
    DrivingLaneWidth=drivingLaneWidth);

plot(scenario)
