chasePlot

自我中心的投影透视图绘制

collapse all in page

语法

chasePlot(ac)
chasePlot(ac,Name,Value)

描述

example

chasePlot(ac) 从参与者 ac 的视角绘制一个驾驶场景。这个绘图被称为追逐图，具有自我中心的投影透视视角，视角位于参与者后方。

example

chasePlot(ac,Name,Value) s使用一个或多个名称-值对来指定选项。例如，您可以在绘图中显示道路中心和参与者的航路点。

例子

collapse all

仿真汽车在S形弯道上行驶

Open Script

仿真一个驾驶场景，其中一辆汽车在S形弯道上行驶。创建并绘制车道边界。

创建一个具有S形弯道的驾驶场景。

scenario = drivingScenario('StopTime',3);
roadcenters = [-35 20 0; -20 -20 0; 0 0 0; 20 20 0; 35 -20 0];

创建车道并将它们添加到道路上。

lm = [laneMarking('Solid','Color','w'); ...
    laneMarking('Dashed','Color','y'); ...
    laneMarking('Dashed','Color','y'); ...
    laneMarking('Solid','Color','w')];
ls = lanespec(3,'Marking',lm);
road(scenario,roadcenters,'Lanes',ls);

添加一个自车，并根据自车的路径点指定其轨迹。默认情况下，汽车的速度为每秒30米。

car = vehicle(scenario, ...
    'ClassID',1, ...
    'Position',[-35 20 0]);
waypoints = [-35 20 0; -20 -20 0; 0 0 0; 20 20 0; 35 -20 0];
smoothTrajectory(car,waypoints);

绘制场景和相应的追踪图。

plot(scenario)


chasePlot(car)

运行仿真循环。

  1. 初始化鸟瞰图，并创建一个轮廓绘制器、左车道和右车道边界绘制器，以及一个道路边界绘制器。

  2. 获取道路边界和矩形轮廓。

  3. 获取车辆左右两侧的车道边界。

  4. 推进仿真并更新绘图器。

bep = birdsEyePlot('XLim',[-40 40],'YLim',[-30 30]);
olPlotter = outlinePlotter(bep);
lblPlotter = laneBoundaryPlotter(bep,'Color','r','LineStyle','-');
lbrPlotter = laneBoundaryPlotter(bep,'Color','g','LineStyle','-');
rbsEdgePlotter = laneBoundaryPlotter(bep);
legend('off');
while advance(scenario)
    rbs = roadBoundaries(car);
    [position,yaw,length,width,originOffset,color] = targetOutlines(car);
    lb = laneBoundaries(car,'XDistance',0:5:30,'LocationType','Center', ...
        'AllBoundaries',false);
    plotLaneBoundary(rbsEdgePlotter,rbs)
    plotLaneBoundary(lblPlotter,{lb(1).Coordinates})
    plotLaneBoundary(lbrPlotter,{lb(2).Coordinates})
    plotOutline(olPlotter,position,yaw,length,width, ...
        'OriginOffset',originOffset,'Color',color)
end

在驾驶场景仿真中显示目标轮廓

Open Live Script

创建一个驾驶场景，并展示随着仿真的推进目标轮廓如何变化。

创建一个驾驶场景，包括两条相交的直路。第一段道路长45米。第二条直路长32米，两边有护栏，并与第一段道路相交。一辆以每秒12.0米的速度行驶的汽车在第一段道路上接近一个以每秒2.0米的速度穿过十字路口的行人。

scenario = drivingScenario('SampleTime',0.1,'StopTime',1);
road1 = road(scenario,[-10 0 0; 45 -20 0]);
road2 = road(scenario,[-10 -10 0; 35 10 0]);
barrier(scenario,road1)
barrier(scenario,road1,'RoadEdge','left')
ped = actor(scenario,'ClassID',4,'Length',0.4,'Width',0.6,'Height',1.7);
car = vehicle(scenario,'ClassID',1);
pedspeed = 2.0;
carspeed = 12.0;
smoothTrajectory(ped,[15 -3 0; 15 3 0],pedspeed);
smoothTrajectory(car,[-10 -10 0; 35 10 0],carspeed);

为车辆创建一个以自我为中心的追逐情节。

chasePlot(car,'Centerline','on')

创建一个空的鸟瞰图，并添加一个轮廓绘制器和车道边界绘制器。然后，运行仿真。在每个仿真步骤中：

  • 更新追逐情节，显示道路边界和目标轮廓。

  • 更新鸟瞰图，显示更新后的道路边界和目标轮廓。绘图的视角始终与自车相关。

bepPlot = birdsEyePlot('XLim',[-50 50],'YLim',[-40 40]);
outlineplotter = outlinePlotter(bepPlot);
laneplotter = laneBoundaryPlotter(bepPlot);
legend('off')

while advance(scenario)
    rb = roadBoundaries(car);
    [position,yaw,length,width,originOffset,color] = targetOutlines(car);
    [bposition,byaw,blength,bwidth,boriginOffset,bcolor,barrierSegments] = targetOutlines(car,'Barriers');
    plotLaneBoundary(laneplotter,rb)
    plotOutline(outlineplotter,position,yaw,length,width, ...
        'OriginOffset',originOffset,'Color',color)
    plotBarrierOutline(outlineplotter,barrierSegments,bposition,byaw,blength,bwidth, ...
        'OriginOffset',boriginOffset,'Color',bcolor)
    pause(0.01)
end

输入参数

collapse all

ac — Actor
Actor object | Vehicle object

Actor是属于 drivingScenario 对象的一个对象，可以是 Actor 或 Vehicle 对象。要创建这些对象，请分别使用 actor 和 vehicle 函数。

Name-Value Arguments

将可选的参数对指定为 Name1=Value1,...,NameN=ValueN 的形式，其中 Name 是参数名称，Value 是相应的值。Name-value 参数必须出现在其他参数之后，但是参数对的顺序无关紧要。

在R2021a之前，使用逗号将每个名称和值分隔开，并将 Name 用引号括起来。

Example: chasePlot(ac,'Centerline','on','RoadCenters,'on') 显示每个道路段的中心线和道路中心点。

Parent — Axes in which to draw plot
Axes object

要绘制图形的坐标轴，指定为由'Parent' 和一个 Axes 对象组成的逗号分隔对。如果您不指定 Parent，则会创建一个新的图形。

Centerline — Display center line of roads
'off' (default) | 'on'

显示道路的中心线，指定为由 'Centerline' 和'off' 或 'on' 组成的逗号分隔对。中心线沿着每个道路段的中间走向。中心线在交叉口或道路分叉等区域是不连续的。

RoadCenters — Display road centers
'off' (default) | 'on'

显示道路中心，指定为由'RoadCenters'和 'off' 或'on'组成的逗号分隔对。道路中心定义了绘图中显示的道路。

Waypoints — Display actor waypoints
'off' (default) | 'on'

显示参与者的路径点，指定为由'Waypoints' 和 'off' 或'on'组成的逗号分隔对。路径点定义了参与者的轨迹。

Meshes — Display actor meshes
'off' (default) | 'on'

显示参与者的网格而不是立方体，指定为由'Meshes' 和 'off' 或 'on' 组成的逗号分隔对。

ViewHeight — Height of plot viewpoint
1.5 × actor height (default) | positive real scalar

绘图视点的高度，指定为由 'ViewHeight' 和一个正实数标量组成的逗号分隔对。该高度是相对于参与者底部的。单位为米。

ViewLocation — Location of plot viewpoint
2.5 × actor length (default) | [x, y] real-valued vector

绘图视点的位置，指定为由 'ViewLocation' 和一个实值向量 [x, y] 组成的逗号分隔对。该位置是相对于参与者的坐标系中的立方体中心的。视点的默认位置是在立方体中心的后方，[2.5*actor.Length 0]。单位为米。

ViewRoll — Roll angle orientation of plot viewpoint
0 (default) | real scalar

绘图视点的滚转角度方向，指定为由 'ViewRoll' 和实数标量组成的逗号分隔对。单位为度。

ViewPitch — Pitch angle orientation of plot viewpoint
0 (default) | real scalar

绘图视点的俯仰角方向，指定为由'ViewPitch'和实数标量组成的逗号分隔对。单位为度。

ViewYaw — Yaw angle orientation of plot viewpoint
0 (default) | real scalar

绘图视点的偏航角方向，指定为由'ViewYaw' 和实数标量组成的逗号分隔对。单位为度。
