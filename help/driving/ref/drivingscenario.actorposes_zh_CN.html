actorPoses

驾驶场景中参与者的位置、速度和方向

collapse all in page

语法

poses = actorPoses(scenario)

描述

example

poses = actorPoses(scenario) 返回驾驶场景 scenario 中所有参与者的当前姿态（位置、速度和方向）。参与者包括 Actor 对象、Vehicle 对象和 Barrier 段，您可以分别使用 actor, vehicle 和 barrier 函数创建它们。参与者的姿态是以场景坐标表示的。

例子

collapse all

创建具有多个参与者和道路的驾驶场景

Open Live Script

创建一个驾驶场景，包含一条弯曲的道路、两条直线道路和两个参与者：一辆汽车和一辆自行车。两个参与者沿着道路行驶60秒。

创建驾驶场景对象。

scenario = drivingScenario('SampleTime',0.1','StopTime',60);

使用道路中心点创建弯曲道路，该道路的弧线遵循800米半径的圆弧。弧线起始角度为0°，结束角度为90°，采用5°增量进行采样。

angs = [0:5:90]';
R = 800;
roadcenters = R*[cosd(angs) sind(angs) zeros(size(angs))];
roadwidth = 10;
cr = road(scenario,roadcenters,roadwidth);

在每个端点处使用道路中心点添加两条默认宽度的直线道路。在第一条直线道路上的两侧道路边缘添加障碍物。

roadcenters = [700 0 0; 100 0 0];
sr1 = road(scenario,roadcenters);
barrier(scenario,sr1)
barrier(scenario,sr1,'RoadEdge','left')
roadcenters = [400 400 0; 0 0 0];
road(scenario,roadcenters);

获取道路边界。

rbdry = roadBoundaries(scenario);

在场景中添加一辆汽车和一辆自行车。将汽车放置在第一条直线道路的起始位置。

car = vehicle(scenario,'ClassID',1,'Position',[700 0 0], ...
    'Length',3,'Width',2,'Height',1.6);

将自行车放置在道路更远的位置。

bicycle = actor(scenario,'ClassID',3,'Position',[706 376 0]', ...
    'Length',2,'Width',0.45,'Height',1.5);

绘制场景。

plot(scenario,'Centerline','on','RoadCenters','on');
title('Scenario');

展示参与者的姿势和简介。 

allActorPoses = actorPoses(scenario)
allActorPoses=242×1 struct array with fields:
    ActorID
    Position
    Velocity
    Roll
    Pitch
    Yaw
    AngularVelocity


allActorProfiles = actorProfiles(scenario)
allActorProfiles=242×1 struct array with fields:
    ActorID
    ClassID
    Length
    Width
    Height
    OriginOffset
    MeshVertices
    MeshFaces
    RCSPattern
    RCSAzimuthAngles
    RCSElevationAngles

由于这个场景中存在障碍物，每个障碍物段被视为一个参与者，actorPoses 和 actorProfiles 函数返回所有静止和非静止参与者的姿势。要仅获取车辆和自行车等非静止参与者的姿势和简介，请首先使用 scenario.Actors.ActorID 属性获取它们对应的参与者ID。

movableActorIDs = [scenario.Actors.ActorID];

然后，使用这些ID来筛选出仅包含非静止参与者的姿势和简介。

movableActorPoseIndices = ismember([allActorPoses.ActorID],movableActorIDs);

movableActorPoses = allActorPoses(movableActorPoseIndices)
movableActorPoses=2×1 struct array with fields:
    ActorID
    Position
    Velocity
    Roll
    Pitch
    Yaw
    AngularVelocity


movableActorProfiles = allActorProfiles(movableActorPoseIndices)
movableActorProfiles=2×1 struct array with fields:
    ActorID
    ClassID
    Length
    Width
    Height
    OriginOffset
    MeshVertices
    MeshFaces
    RCSPattern
    RCSAzimuthAngles
    RCSElevationAngles

输入参数

collapse all

scenario — Driving scenario
drivingScenario object

驾驶场景，指定为一个 drivingScenario 对象。

输出参数

collapse all

poses — Actor poses
structures | array of structures

参与者姿势，以场景坐标表示，以结构体或结构体数组的形式返回。姿势包括参与者的位置、速度和方向。

poses 中的每个结构体具有以下字段。

Field Description
ActorID

场景定义的参与者标识符，指定为正整数。
Position

参与者的位置，以实数向量的形式指定，格式为 [x y z]。单位为米。
Velocity

参与者在x、y和z方向上的速度 (v) ，以实数向量的形式指定，格式为 [vx vy vz]。单位为米每秒。
Roll

参与者的滚动角度，以实数标量形式指定。单位为度。
Pitch

参与者的俯仰角度，以实数标量形式指定。单位为度。
Yaw

参与者的偏航角度，以实数标量形式指定。单位为度。
AngularVelocity

参与者在x、y和z方向上的角速度 (ω) ，以实数向量的形式指定，格式为 [ωx ωyωz]。单位为度每秒。

有关这些结构体字段的完整定义，请参阅 actor, vehicle 和barrier 函数。
