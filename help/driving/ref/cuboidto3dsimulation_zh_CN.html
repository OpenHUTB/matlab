Cuboid To 3D Simulation

将参与者从长方体坐标转换为三维仿真坐标

expand all in page
  • 库:

  • Automated Driving Toolbox / Driving Scenario and Sensor Modeling

描述

The Cuboid To 3D Simulation 模块将世界坐标中的长方体参与者姿势转换为 Simulation 3D Vehicle with Ground Following 模块使用的 X, Y, 和 Yaw 坐标。使用转换后的值在三维仿真环境中为使用 Driving Scenario Designer 应用程序创建的参与者设置车辆位置。场景的地面地形决定了车辆的横滚（x轴旋转）、俯仰（y轴旋转）和仰角（z轴位置）。 

可以指定包含单个参与者姿势或多个参与者姿势的总线。默认情况下，该块会转换总线中第一个参与者的姿势。指定要转换其姿势的参与者，请指定该参与者的 ActorID。

在长方体和三维仿真驾驶场景中，坐标系相同，但车辆的原点不同。在长方体驾驶场景中，车辆原点位于地面上，后轴中心下方。该模块将该原点转换为三维仿真环境中使用的原点，该原点位于车辆的几何中心下方。该表显示了两种环境之间的原点差异。

端口

输入

expand all

Actor — 世界坐标系中立方体参与者姿势
包含MATLAB® 结构 Simulink® 总线

长方体参与者在世界坐标中的姿势，指定为包含 MATLAB 结构体的 Simulink 总线。

要获取此结构输入，请使用 Scenario Reader 模块从场景中读取参与者。默认情况下，场景读取器块以自我车辆坐标输出参与者。要将这些姿势从自我车辆转换为世界坐标，请使用 Vehicle To World 模块。

该总线中的结构可以包含单个参与者姿势或多个参与者姿势。

单姿态结构

要指定单个参与者姿势，该结构必须包含这些字段。

字段 描述
ActorID

场景定义的参与者标识符，指定为正整数。
Position

参与者的位置，指定为 [x y z] 形式的实值向量。单位为米。
Velocity

参与者在 x、y 和 z 方向上的速度 (v)，指定为 [vx vy vz] 形式的实值向量。单位为米每秒。
Roll

参与者的俯仰角，指定为实值标量。单位为度。
Pitch

参与者的偏航角，指定为实值标量。单位为度。
Yaw

参与者的偏航角，指定为实值标量。单位为度。
AngularVelocity

参与者在 x、 y和 z 方向上的角速度 (ω) ，指定为 [ωx ωy ωz]。单位为度每秒。

多姿态结构

要指定多个参与者姿态，该结构必须包含这些字段。

字段 描述 类型
NumActors 参与者数目 非负整数
Time 当前仿真时间 实值标量
Actors 参与者姿态 参与者姿态结构NumActors-长度的数组

Actors 中每个参与者的姿态结构都必须有这些字段。

字段 描述
ActorID

由场景所定义的参与者标识符，指定为正整数。
Position

参与者的位置，指定为 [x y z]形式的实值向量。单位为米。
Velocity

参与者在x、y和 z方向上的速度 (v) ，指定为 [vx vy vz]形式的实值向量。单位为米每秒。
Roll

参与者的横滚角，指定为实值标量。单位为度。
Pitch

参与者的俯仰角，指定为实值标量。单位为度。
Yaw

参与者的偏航角，指定为实值标量。单位为度。
AngularVelocity

参与者在x、 y和 z方向上的角速度 (ω) ，指定为 [ωx ωy ωz] 形式的实值向量。单位为度每秒。

该模块仅转换 Actors 数组中的一个姿态。指定要转换的姿势，请选择 Specify Actor ID,，然后 ActorID 使用 ActorID used for conversion 参数的指定参与者 ID。

输出

expand all

X — 三维仿真坐标中参与者的纵向位置
数值标量

三维仿真坐标中参与者的纵向位置，以数值标量形式返回。单位为米。

在此坐标系中，从X轴正方向看时（顺着坐标轴的方向看），Y轴正向指向左，Z轴指向上。

要在三维仿真环境中指定车辆的X位置，请将此端口连接到 Simulation 3D Vehicle with Ground Following 的X输入端口。

Y — 三维仿真坐标中参与者的横向位置
数值标量

三维仿真坐标中参与者的横向位置，以数值标量形式返回。单位为米。

在此坐标系中，从X轴正方向看时 ，Y轴正向指向左，Z轴指向上。

要在三维仿真环境中指定车辆的Y位置，请将此端口连接到 Simulation 3D Vehicle with Ground Following 的Y输入端口。

Yaw — 三维仿真坐标中参与者的偏航方向角
数值标量

三维仿真坐标中参与者围绕Z轴的偏航方向角，以数值标量形式返回。单位为度。

在该坐标系中，当从Z轴正方向看时 ，偏航为顺时针正。但是，如果从二维自上而下的角度查看仿真，则偏航为逆时针正，因为正在沿着负Z轴查看场景。

要在三维仿真环境中指定车辆的偏航方向角，请将此端口连接到 Simulation 3D Vehicle with Ground Following 的偏航输入端口。

参数

expand all

Specify Actor ID — 启用长方体参与者的 ID 规范
off (默认) | on

选择此参数可启用 ActorID used for conversion parameter，可以指定长方体参与者姿态的  ActorID 来转换三维仿真坐标。

如果清除此参数，则该模块转换输入 Actor 总线中的第一个参与者姿态。

ActorID used for conversion —长方体参与者的 ActorID 值

1 (默认) | 正整数

用于转换三维仿真坐标的长方体参与者 ActorID 值，指定为正整数。该参数必须是从输入 Actor 总线而来的有效 ActorID。

依赖

要启用此参数，请选择 Specify Actor ID。

Simulate using — 要运行的仿真类型
Interpreted execution (默认) | Code generation

  • Interpreted execution — 使用 MATLAB 解释器对模型进行仿真。此选项可缩短启动时间。在 Interpreted execution 模式下，可以调试模块的源代码。

  • Code generation — 使用生成的 C/C++ 代码仿真模型。第一次运行仿真时，Simulink 会为该模块生成 C/C++ 代码。只要模型不改变，C 代码就可以重复用于后续仿真。此选项需要额外的启动时间。
