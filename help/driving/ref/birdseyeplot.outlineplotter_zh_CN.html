outlinePlotter

用于鸟瞰图的轮廓绘图器

collapse all in page

语法

olPlotter = outlinePlotter(bep)
olPlotter = outlinePlotter(bep,Name,Value)

描述

olPlotter = outlinePlotter(bep) 创建一个 OutlinePlotter 对象，用于配置在鸟瞰图上显示对象轮廓。OutlinePlotter 对象存储在 birdsEyePlot 对象（bep）的 Plotters 属性中。要显示驾驶场景中参与者的轮廓，首先使用 targetOutlines 获取参与者和障碍物的尺寸。然后，在创建轮廓绘图器对象后，分别使用 plotOutline 和 plotBarrierOutline 函数在鸟瞰图中显示所有参与者和障碍物的轮廓。

example

olPlotter = outlinePlotter(bep,Name,Value) 使用一个或多个 Name,Value 对参数设置属性。例如，outlinePlotter(bep,'FaceAlpha',0) 将每个轮廓内的区域设置为完全透明。

例子

collapse all

在鸟瞰图上绘制目标的轮廓

Open Live Script

创建一个驾驶场景。创建一个长度为25米的道路段，其左边缘有一个障碍物。添加一个以每秒1米的速度横穿道路的行人，以及一个以每秒10米的速度沿道路行驶的车辆。

scenario = drivingScenario;

r = road(scenario,[0 0 0; 25 0 0]);
barrier(scenario,r,'RoadEdge','left')

p = actor(scenario,'ClassID',4,'Length',0.2,'Width',0.4,'Height',1.7);
v = vehicle(scenario,'ClassID',1);

smoothTrajectory(p,[15 -3 0; 15 3 0],1)
smoothTrajectory(v,[v.RearOverhang 0 0; 25-v.Length+v.RearOverhang 0 0],10)

使用追踪图来从车辆的视角显示场景。

chasePlot(v,'Centerline','on')

创建一个鸟瞰图、轮廓绘制器（outline plotter）和车道边界绘制器（lane boundary plotter）。

bep = birdsEyePlot('XLim',[-25 25],'YLim',[-10 10]);
olPlotter = outlinePlotter(bep);
lbPlotter = laneBoundaryPlotter(bep);
legend('off')

运行仿真循环。使用目标的轮廓更新绘制器。

while advance(scenario)
   % Obtain the road boundaries and rectangular outlines.
   rb = roadBoundaries(v);
   [position,yaw,length,width,originOffset,color] = targetOutlines(v);
   [bposition,byaw,blength,bwidth,boriginOffset,bcolor,numBarrierSegments] = targetOutlines(v,'Barriers');

   % Update the bird's-eye plotters with the road, actors and barriers.
   plotLaneBoundary(lbPlotter,rb);
   plotOutline(olPlotter,position,yaw,length,width, ...
               'OriginOffset',originOffset,'Color',color);
   plotBarrierOutline(olPlotter,numBarrierSegments,bposition,byaw,blength,bwidth, ...
               'OriginOffset',boriginOffset,'Color',bcolor); 
   % Allow time for plot to update.
   pause(0.01)
end

输入参数

collapse all

bep — Bird’s-eye plot
birdsEyePlot object

Bird’s-eye plot, specified as a birdsEyePlot object.

Name-Value Arguments

指定可选参数对，格式为 Name1=Value1,...,NameN=ValueN，其中 Name 是参数名，Value 是对应的值。Name-Value参数必须在其他参数之后出现，但是对参数对的顺序没有要求。

在R2021a之前，使用逗号将每个名称和值分隔开，并将 Name 用引号括起来。

Example: outlinePlotter('FaceAlpha',1) 将每个轮廓内的区域设置为完全不透明。

FaceAlpha — Transparency of area within each outline
0.75 (default) | real scalar

每个轮廓内区域的透明度，由逗号分隔的参数对'FaceAlpha'和取值范围为 [0, 1] 的实数标量指定。值为 0 时，区域完全透明。值为 1 时，区域完全不透明。

Tag — Tag associated with plotter object
'PlotterN' (default) | character vector | string scalar

与绘图器对象相关联的标签，由逗号分隔的参数对 'Tag' 和字符向量或字符串标量指定。默认值为'PlotterN'，其中 N 是与输入 birdsEyePlot 对象相关联的第  N  个绘图器对应的整数。

输出参数

collapse all

olPlotter — Outline plotter
OutlinePlotter object

轮廓绘图器，返回为一个 OutlinePlotter 对象。您可以通过更改其属性值来修改此对象。属性名称对应于 outlinePlotter 函数的名称-值对参数。

olPlotter 存储在 birdsEyePlot 对象的 Plotters 属性中。要绘制驾驶场景中的参与者的轮廓，请首先使用 targetOutlines 获取参与者和障碍物的尺寸。然后，在调用 outlinePlotter 创建绘图器对象后，分别使用 plotOutline 和 plotBarrierOutline 在鸟瞰图中绘制所有参与者和障碍物的轮廓。

版本历史

Introduced in R2017b
