roadGroup

向驾驶场景中添加道路交叉口或路口

collapse all in page

语法

roadGroup(scenario,rg)

描述

example

roadGroup(scenario,rg) 可以根据指定的道路段创建一个道路交叉口或路口，并将其添加到驾驶场景 scenario 中。RoadGroup 对象 rg 指定了连接在交叉口处相遇的道路的道路段。

例子

collapse all

创建一个具有三岔口的道路网络

Open Live Script

三岔口是指两条相邻的道路以钝角与第三条道路相交的 Y 型交叉口，如图所示。为了连接这三条道路，您需要添加三个道路段来创建一个 Y 型交叉口。

向驾驶场景中添加三条道路

创建一个空的驾驶场景。

scenario = drivingScenario;

指定道路上的车道数量和每条车道的宽度。

ls = lanespec(2,'Width',5);

定义三条道路的道路中心，并将它们添加到驾驶场景中。第一条道路位于场景画布的左侧，呈对角线方向向左倾斜；第二条道路位于场景画布的右侧，呈对角线方向向右倾斜；第三条道路垂直朝向。

% Add the first road
roadCenters = [-20 0; 6 0];
road(scenario,roadCenters,'Name','Road 1','Lanes',ls);

% Add the second road
roadCenters = [23 7; 50 33];
road(scenario,roadCenters,'Name','Road 2','Lanes',ls);

% Add the third road
roadCenters = [23 -7; 50 -33];
road(scenario,roadCenters,'Name','Road 3','Lanes',ls);

绘制场景。

figure
plot(scenario)

创建 Y 型交叉口以连接道路

创建一个 RoadGroup 对象。为构成 Y 型交叉口的每个道路段指定宽度。

rg = driving.scenario.RoadGroup('Name','Y-Junction');
roadWidth = 10;

为三个道路段指定道路中心，并使用 road 函数将这些道路段添加到 RoadGroup 对象中。这些道路段相互交叉。

% Add the first road segment
roadCenters = [23 7; 14 1; 6 0];
road(rg,roadCenters,roadWidth,'Name','Segment 1');

% Add the second road segment
roadCenters = [23 -7; 14 -1; 6 0];
road(rg,roadCenters,roadWidth,'Name','Segment 2');

% Add the third road segment
roadCenters = [23 7; 21 4; 21 -4; 23 -7];
road(rg,roadCenters,roadWidth,'Name','Segment 3');

将 Y 型交叉口添加到驾驶场景中

使用 roadGroup 函数将存储在 RoadGroup 对象中的道路段添加到驾驶场景中。这些道路段形成一个连接驾驶场景中三条道路的 Y 型交叉口。

roadGroup(scenario,rg);

输入参数

collapse all

scenario — Driving scenario
drivingScenario object

驾驶场景，指定为 drivingScenario 对象。

rg — 道路段的规格说明
RoadGroup object

形成交叉口的道路段的规格，指定为 RoadGroup 对象。添加单独的道路段以连接在交叉口相遇的每对进入道路。使用 road 函数将道路段添加到 RoadGroup 对象中。

注意

  • 在将道路段添加到 RoadGroup 对象后，您无法更改它们的属性。

限制

  • 该函数不支持在交叉口中的车道标线。

  • 场景绘图不显示交叉口中的道路中心。

建议

  • 使用 roadGroup 函数向 RoadGroup 对象添加至少一个道路段，以创建一个交叉口。

  • T要在交叉口中创建平滑的路面形状，您需要将 RoadGroup 对象中的道路段的规格（道路中心、宽度、高度和倾斜角）与汇合在交叉口处的进入道路的规格相匹配。

  • 在创建交叉口时，该函数仅考虑每个道路段的第一个车道规格，因此请为 RoadGroup 对象的每个道路段指定单个车道规格。
