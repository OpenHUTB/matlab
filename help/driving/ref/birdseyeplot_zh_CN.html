birdsEyePlot

绘制车辆周围的检测、轨迹和传感器覆盖范围

expand all in page

描述

birdsEyePlot 对象显示了一个鸟瞰图，展示了自车附近的二维驾驶场景。您可以将此图与能够检测物体和车道的传感器一起使用。

要在鸟瞰图上显示驾驶场景的各个方面，可以执行以下操作：

  1. 创建一个 birdsEyePlot 对象。

  2. 为您想要绘制的驾驶场景方面创建绘图器。

  3. 使用相应的绘图器和它们的绘图函数将这些方面显示在鸟瞰图上。

这个表格显示了根据您想要绘制的驾驶场景方面所使用的绘图函数。

Driving Scenario Aspect to Plot Plotter Creation Function Plotter Display Function
Sensor coverage areas coverageAreaPlotter plotCoverageArea
Sensor detections detectionPlotter, rangeDetectionPlotter plotDetection, plotRangeDetection
Lane boundaries laneBoundaryPlotter plotLaneBoundary
Lane markings laneMarkingPlotter plotLaneMarking, plotParkingLaneMarking
Object meshes meshPlotter plotMesh
Object outlines outlinePlotter plotOutline, plotBarrierOutline
Ego vehicle path pathPlotter plotPath

Object tracking results trackPlotter plotTrack

要了解如何配置和使用鸟瞰图的示例，请参阅 Visualize Sensor Coverage, Detections, and Tracks。

Creation

语法

bep = birdsEyePlot
bep = birdsEyePlot(Name,Value)

描述

bep = birdsEyePlot 在一个新的图形窗口中创建一个鸟瞰图。

example

bep = birdsEyePlot(Name,Value) 使用一个或多个 Name,Value 键值对参数设置 properties 。例如，birdsEyePlot('XLimits',[0 60],'YLimits',[-20 20]) 显示了车辆前方 60 米范围内以及车辆两侧 20 米范围内的区域。请将每个属性名称用引号括起来。

Properties

expand all

Parent — Axes on which to plot
axes handle

要绘制的坐标轴，可以指定为一个坐标轴句柄。默认情况下，birdsEyePlot 对象使用当前的坐标轴句柄，该句柄由 gca 函数返回。

Plotters — Plotters created for bird's-eye plot
array of plotter objects

为鸟瞰图创建的绘图器，指定为绘图器对象的数组。

XLimits — X-axis range
real-valued vector of the form [Xmin Xmax]

鸟瞰图的X轴范围，以车辆坐标表示，指定为形式为 [XminXmax]的实数向量。单位为米。如果您不指定 XLimits，则图形将使用父坐标轴的默认值。

X轴是垂直的，并且在自车的前进方向上为正方向。原点位于自车后轴的中心位置。

有关鸟瞰图中使用的坐标系的更多详细信息，请参阅 Vehicle Coordinate System。

YLimits — Y-axis range
real-valued vector of the form [Ymin Ymax]

鸟瞰图的Y轴范围，以车辆坐标表示，指定为形式为 [YminYmax] 的实数向量。单位为米。如果您不指定 YLimits，则图形将使用父坐标轴的默认值。

Y轴水平运行，并且在面向前方时，自车的左侧为正方向。原点位于自车后轴的中心位置。

有关在 birdsEyePlot 对象中使用的坐标系的更多详细信息，请参阅 Vehicle Coordinate System。

例子

collapse all

创建并显示鸟瞰图

Open Live Script

创建一个鸟瞰图，其中X轴的范围从0到90米，Y轴的范围从-35到35米。

bep = birdsEyePlot('XLim',[0 90],'YLim',[-35 35]);

显示一个覆盖范围，其视场角为35度，范围为60米。

caPlotter = coverageAreaPlotter(bep,'DisplayName','Radar coverage area');
mountPosition = [1 0];
range = 60;
orientation = 0;
fieldOfView = 35;
plotCoverageArea(caPlotter,mountPosition,range,orientation,fieldOfView);

显示雷达检测结果，坐标分别为 (30, –5), (50, –10) 和 (40, 7).。 

radarPlotter = detectionPlotter(bep,'DisplayName','Radar detections');
plotDetection(radarPlotter,[30 -5; 50 -10; 40 7]);

创建一个带有覆盖范围和检测点的鸟瞰图

Open Live Script

创建一个鸟瞰图，其中X轴范围为0到90米，Y轴范围为-35到35米。配置绘图以包括雷达覆盖范围和检测点绘图器。设置这些绘图器的显示名称。

bep = birdsEyePlot('XLim',[0 90],'YLim',[-35 35]);
coverageAreaPlotter(bep,'DisplayName','Radar coverage area');
detectionPlotter(bep,'DisplayName','Radar detections');

使用 findPlotter 通过它们的显示名称来定位这些绘图器。

caPlotter = findPlotter(bep,'DisplayName','Radar coverage area');
radarPlotter = findPlotter(bep,'DisplayName','Radar detections');

绘制覆盖范围和检测到的物体。

plotCoverageArea(caPlotter,[1 0],30,0,35);
plotDetection(radarPlotter,[30 5; 30 -10; 30 15]);

清除绘图中的数据。

clearPlotterData(bep);

限制

在鸟瞰图中，无法使用矩形缩放功能，即通过绘制矩形来放大图形的某个部分。

Vehicle Coordinate System

birdsEyePlot 使用车辆坐标系 (XV, YV)，其中：

  • XV 轴从自车向前指向。

  • YV 轴指向左侧，以面向前方时的视角为参考。

原点位于自车的旋转中心。该点位于道路表面，在自车后轴的中心下方。

有关车辆坐标系的更多详细信息，请参阅 Coordinate Systems in Automated Driving Toolbox。
