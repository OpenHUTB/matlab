transformImage

将图像转换为鸟瞰图

collapse all in page

语法

J = transformImage(birdsEye,I)

描述

example

J = transformImage(birdsEye,I) 将输入图像 I 转换为鸟瞰图像 J。birdsEyeView 对象 birdsEye 的 OutputView 和 ImageSize 属性分别确定要转换的 I 的部分和 J 的大小。

例子

collapse all

将道路图像转换为鸟瞰图像

Open Live Script

从车辆上安装的前置摄像头获取的图像创建一个鸟瞰图像。使用车辆和图像坐标系统在鸟瞰图中显示点。

定义相机内参并创建包含这些内参的对象。

focalLength = [309.4362 344.2161];
principalPoint = [318.9034 257.5352];
imageSize = [480 640];

camIntrinsics = cameraIntrinsics(focalLength,principalPoint,imageSize);

将相机的高度设置为离地面约2米。将相机的俯仰角设置为向下14度。

height = 2.1798;
pitch = 14;

创建一个包含相机配置的对象。

sensor = monoCamera(camIntrinsics,height,'Pitch',pitch);

定义相机前方要转换为鸟瞰图的区域。设置一个相机前方范围为3到30米，相机两侧各6米的区域。

distAhead = 30;
spaceToOneSide = 6;
bottomOffset = 3;

outView = [bottomOffset,distAhead,-spaceToOneSide,spaceToOneSide];

将输出图像的宽度设置为250像素。通过将长度设置为 NaN，自动从宽度计算输出长度。

outImageSize = [NaN,250];

使用先前定义的参数创建一个执行鸟瞰图转换的对象。

birdsEye = birdsEyeView(sensor,outView,outImageSize);

加载由传感器捕获的图像。

I = imread('road.png');
figure
imshow(I)
title('Original Image')

将输入图像转换为鸟瞰图像。

BEV = transformImage(birdsEye,I);

在鸟瞰图像中，将一个20米的标记物放置在传感器正前方。使用 vehicleToImage 函数来指定标记物在车辆坐标中的位置。在鸟瞰图像上显示标记物。

imagePoint = vehicleToImage(birdsEye,[20 0]);
annotatedBEV = insertMarker(BEV,imagePoint);
annotatedBEV = insertText(annotatedBEV,imagePoint + 5,'20 meters');

figure
imshow(annotatedBEV)
title('Bird''s-Eye-View Image: vehicleToImage')

在原始鸟瞰图像中定义一个位置，这次使用图像坐标。使用 imageToVehicle 函数将图像坐标转换为车辆坐标。显示标记物与车辆前部之间的距离。

imagePoint2 = [120 400];
annotatedBEV = insertMarker(BEV,imagePoint2);

vehiclePoint = imageToVehicle(birdsEye,imagePoint2);
xAhead = vehiclePoint(1);
displayText = sprintf('%.2f meters',xAhead);
annotatedBEV = insertText(annotatedBEV,imagePoint2 + 5,displayText);

figure
imshow(annotatedBEV)
title('Bird''s-Eye-View Image: imageToVehicle')

输入参数

collapse all

birdsEye — Object for transforming image to bird's-eye view
birdsEyeView object

将图像转换为鸟瞰图的对象，指定为 birdsEyeView 对象。

I — Input image
truecolor image | grayscale image

输入图像，指定为真彩色或灰度图像。 birdsEye 的 OutputView 属性确定要转换为鸟瞰图的I的部分。

I 不应包含镜头畸变。可以使用 undistortImage 函数来去除镜头畸变。在高端光学中，可以忽略畸变。

输出参数

collapse all

J — Bird’s-eye-view image
truecolor image | grayscale image

鸟瞰图像，以真彩色或灰度图像的形式返回。birdsEye 的 ImageSize 属性确定了 J 的大小。
