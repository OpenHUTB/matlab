driving.scenario.truckMesh

驾驶场景中卡车的网格表示

collapse all in page

语法

mesh = driving.scenario.truckMesh

描述

example

mesh = driving.scenario.truckMesh 创建一个卡车的网格表示，以 extendedObjectMesh 对象 mesh 的形式返回。

例子

collapse all

使用卡车网格生成激光雷达点云

Open Live Script

将预先构建的卡车网格添加到驾驶场景中以生成点云。然后，使用 lidarPointCloudGenerator 系统对象来生成卡车网格的点云。

创建并显示预先构建的卡车网格。 

mesh = driving.scenario.truckMesh;
egoMesh = driving.scenario.carMesh;
show(mesh);

pause(1);

创建一个驾驶场景。

s = drivingScenario;

在驾驶场景中添加一条直路。该道路每个方向上有一条车道。

road(s,[0 0 0; 30 0 0],'Lanes',lanespec([1 1]));

将一辆汽车设置为自车，并将一辆卡车设置为非自车参与者。

egoVehicle = vehicle(s,'ClassID',1,'Mesh',egoMesh);
smoothTrajectory(egoVehicle,[1 -2 0; 21.3 -2 0],20);
truck = vehicle(s,'Position',[15 2 0],'Yaw',180,'ClassID',2,'Mesh',mesh);

绘制驾驶场景。使用名称-值对参数 'Meshes','on' 来显示绘图中参与者的网格。

plot(s,'Meshes','on');

创建一个 lidarPointCloudGenerator 系统对象。将该系统对象的参与者配置设置为驾驶场景中的参与者配置。

lidar = lidarPointCloudGenerator;
lidar.ActorProfiles = actorProfiles(s);

生成驾驶场景的激光雷达点云。

player = pcplayer([-20 20],[-10 10],[0 4]);
while advance(s)
 tgts = targetPoses(egoVehicle);
 rdmesh = roadMesh(egoVehicle);
 [ptCloud,isValidTime] = lidar(tgts,rdmesh,s.SimulationTime);
    if isValidTime
    view(player,ptCloud);
    end
end

输出参数

collapse all

mesh — 卡车的网格表示
extendedObjectMesh object

将卡车的网格表示返回为一个 extendedObjectMesh 对象。该网格的原点位于其几何中心处。

您可以使用这个预建的卡车网格作为起点来开发自己的网格。在 MATLAB® 命令行中输入以下内容：

edit driving.scenario.truckMesh
