driving.scenario.carMesh

汽车在驾驶场景中的网格表示

collapse all in page

语法

mesh = driving.scenario.carMesh

描述

example

mesh = driving.scenario.carMesh 创建一个以 extendedObjectMesh 对象 mesh 形式的汽车网格表示。

例子

collapse all

使用汽车网格生成激光雷达点云

Open Live Script

将预构建的汽车网格添加到驾驶场景中。然后，使用 lidarPointCloudGenerator 系统对象来生成汽车网格的点云。

创建并显示预构建的汽车网格。 

mesh = driving.scenario.carMesh;
show(mesh);

创建一个驾驶场景。

s = drivingScenario;

在驾驶场景中添加一条直道。该道路每个方向都有一条车道。

road(s,[0 0 0; 25 0 0],'Lanes',lanespec([1 1]));

将一辆汽车同时设置为自车和非自车参与者。

egoVehicle = vehicle(s,'ClassID',1,'Mesh',mesh);
smoothTrajectory(egoVehicle,[1 -2 0; 21.3 -2 0],20);
car = vehicle(s,'Position',[15 2 0],'Yaw',180,'ClassID',1,'Mesh',mesh);

绘制驾驶场景。设置名称-值对 'Meshes','on' 以显示绘图中参与者的网格。

plot(s,'Meshes','on');

创建一个 lidarPointCloudGenerator 系统对象。将该系统对象的参与者配置设置为驾驶场景中的参与者配置。

lidar = lidarPointCloudGenerator;
lidar.ActorProfiles = actorProfiles(s);

生成驾驶场景的激光雷达点云。

player = pcplayer([-20 20],[-10 10],[0 4]);
 while advance(s)
 tgts = targetPoses(egoVehicle);
 rdmesh = roadMesh(egoVehicle);
 [ptCloud,isValidTime] = lidar(tgts,rdmesh,s.SimulationTime);
    if isValidTime
    view(player,ptCloud);
    end
 end

输出参数

collapse all

mesh — 车辆的网格表示
extendedObjectMesh object

车辆的网格表示，以 extendedObjectMesh 对象的形式返回。网格的原点位于其几何中心处。

您可以使用这个预构建的车辆网格作为起点来开发自己的网格。在 MATLAB® 命令行中输入:

edit driving.scenario.carMesh
