roadBoundaries

获取道路边界

collapse all in page

语法

rbdry = roadBoundaries(scenario)
rbdry = roadBoundaries(ac)

描述

example

rbdry = roadBoundaries(scenario) 返回驾驶场景 scenario 的道路边界 rbdry。

rbdry = roadBoundaries(ac) 返回参与者 ac 在驾驶场景中所遵循的道路边界。

例子

collapse all

创建具有多个参与者和道路的驾驶场景

Open Live Script

创建一个驾驶场景，其中包含一条弯曲的道路、两条直线道路和两个参与者：一辆汽车和一辆自行车。这两个参与者都在道路上行驶60秒。

创建驾驶场景对象。

scenario = drivingScenario('SampleTime',0.1','StopTime',60);

使用道路中心点创建弯曲的道路，该道路沿着半径为800米的圆弧运动。该圆弧从0°开始，到90°结束，采样间隔为5°。

angs = [0:5:90]';
R = 800;
roadcenters = R*[cosd(angs) sind(angs) zeros(size(angs))];
roadwidth = 10;
cr = road(scenario,roadcenters,roadwidth);

添加两条默认宽度的直线道路，使用每个端点的道路中心点。在第一条直线道路上，两侧道路边缘添加路障。

roadcenters = [700 0 0; 100 0 0];
sr1 = road(scenario,roadcenters);
barrier(scenario,sr1)
barrier(scenario,sr1,'RoadEdge','left')
roadcenters = [400 400 0; 0 0 0];
road(scenario,roadcenters);

获取道路边界。

rbdry = roadBoundaries(scenario);

将一辆汽车和一辆自行车添加到场景中。将汽车位置设置在第一条直线道路的起始位置。

car = vehicle(scenario,'ClassID',1,'Position',[700 0 0], ...
    'Length',3,'Width',2,'Height',1.6);

将自行车放置在道路上的较远位置。

bicycle = actor(scenario,'ClassID',3,'Position',[706 376 0]', ...
    'Length',2,'Width',0.45,'Height',1.5);

绘制场景。

plot(scenario,'Centerline','on','RoadCenters','on');
title('Scenario');

显示参与者的姿势和配置文件。 

allActorPoses = actorPoses(scenario)
allActorPoses=242×1 struct array with fields:
    ActorID
    Position
    Velocity
    Roll
    Pitch
    Yaw
    AngularVelocity


allActorProfiles = actorProfiles(scenario)
allActorProfiles=242×1 struct array with fields:
    ActorID
    ClassID
    Length
    Width
    Height
    OriginOffset
    MeshVertices
    MeshFaces
    RCSPattern
    RCSAzimuthAngles
    RCSElevationAngles

由于这个场景中有路障，每个路障段被视为一个参与者，actorPoses 和 actorProfiles 函数会返回所有静止和非静止参与者的姿势。为了只获取车辆和自行车等非静止参与者的姿势和配置文件，请首先使用 scenario.Actors.ActorID 属性获取它们对应的参与者ID。

movableActorIDs = [scenario.Actors.ActorID];

然后，使用这些ID来筛选只包含非静止参与者的姿势和配置文件。

movableActorPoseIndices = ismember([allActorPoses.ActorID],movableActorIDs);

movableActorPoses = allActorPoses(movableActorPoseIndices)
movableActorPoses=2×1 struct array with fields:
    ActorID
    Position
    Velocity
    Roll
    Pitch
    Yaw
    AngularVelocity


movableActorProfiles = allActorProfiles(movableActorPoseIndices)
movableActorProfiles=2×1 struct array with fields:
    ActorID
    ClassID
    Length
    Width
    Height
    OriginOffset
    MeshVertices
    MeshFaces
    RCSPattern
    RCSAzimuthAngles
    RCSElevationAngles

创建并显示道路边界

Open Live Script

创建一个驾驶场景，其中包含一个在场景的世界坐标系中指定的八字形道路。将场景的世界坐标转换为自车的坐标系。

创建一个空的驾驶场景。

scenario = drivingScenario;

将一个八字形道路添加到场景中。显示场景。

roadCenters = [0  0  1
             20 -20  1
             20  20  1
            -20 -20  1
            -20  20  1
              0   0  1];

roadWidth = 3;
bankAngle = [0 15 15 -15 -15 0];
road(scenario,roadCenters,roadWidth,bankAngle);
plot(scenario)

将一个自车添加到场景中。将车辆位置设置为世界坐标 (20, -20)，并将其朝向设置为 -15 度的偏航角。

ego = actor(scenario,'ClassID',1,'Position',[20 -20 0],'Yaw',-15);

使用 roadBoundaries 函数获取自车坐标系中的道路边界。将自车指定为输入参数。

rbEgo1 = roadBoundaries(ego);

在鸟瞰图上显示结果。

bep = birdsEyePlot;
lbp = laneBoundaryPlotter(bep,'DisplayName','Road');
plotLaneBoundary(lbp,rbEgo1)

使用 roadBoundaries 函数获取世界坐标系中的道路边界。将场景指定为输入参数。

rbScenario = roadBoundaries(scenario);

使用 driving.scenario.roadBoundariesToEgo 函数将道路边界转换为自车坐标系中的坐标。

rbEgo2 = driving.scenario.roadBoundariesToEgo(rbScenario,ego);

在鸟瞰图上显示道路边界。

bep = birdsEyePlot;
lbp = laneBoundaryPlotter(bep,'DisplayName','Road boundaries');
plotLaneBoundary(lbp,{rbEgo2})

输入参数

collapse all

scenario — Driving scenario
drivingScenario object

驾驶场景，以 drivingScenario 对象的形式指定。

ac — Actor
Actor object | Vehicle object

属于 drivingScenario 对象的参与者，以 Actor 或 Vehicle 对象的形式指定。要创建这些对象，请分别使用 actor 和 vehicle 函数。

输出参数

collapse all

rbdry — 道路边界
cell array

道路边界，以单元数组的形式返回。单元数组中的每个单元包含一个实值的 N×3 矩阵，表示场景中的一个道路边界，其中 N 是道路边界的数量。矩阵的每一行对应于道路边界顶点的 (x, y, z) 坐标。

当输入参数是驾驶场景时，道路坐标是相对于驾驶场景的世界坐标系的。当输入参数是参与者时，道路坐标是相对于参与者的坐标系的。
