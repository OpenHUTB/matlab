Simulation 3D Pedestrian

在三维环境中实现行人

expand all in page
  • 库:

  • Automated Driving Toolbox / Simulation 3D

描述

Simulation 3D Pedestrian 模块在三维仿真环境中实现行人。 该模块使用输入 (X, Y) 位置和行人的偏航角来调整仰角、翻滚角、俯仰角，以便行人遵循地面地形。该模块确定行人的速度和方向，并适当地将运动调整为闲置、步行或跑步。将此块用于自动驾驶应用。

要使用此模块，请确保模型中存在 Simulation 3D Scene Configuration 模块。如果将 Simulation 3D Pedestrian 模块的 Sample time 参数设置为 -1, 则该模块将继承在 Simulation 3D Scene Configuration 模块中指定的采样时间 。

该模块使用 SAE J670 [1] 和 ISO 8855 [2]中定义的行人 Z-向上右手（right-handed, RH）笛卡尔坐标系。坐标系是惯性坐标系，最初与行人的脚对齐：

  • X-轴沿着行人的纵轴并指向前方。

  • Y-轴沿着行人的横轴并指向左侧。

  • Z-轴指向上方。

当从 Z-轴、Y-轴和 X-轴的正方向看时，Z-轴、Y-轴和X-轴的偏航角、俯仰角和滚动角分别在顺时针方向上为正。行人被放置在场景的世界坐标系中。有关更多详细信息，请参阅 Coordinate Systems for Unreal Engine Simulation in Automated Driving Toolbox.

端口

输入

expand all

Y — 行人的横向位置
标量

行人沿场景Y-轴的横向位置。Y位于惯性Z-向上坐标系中。单位为米。

初始位置（Initial position）[X, Y, Z] (m) 参数的Y值必须与仿真开始时该端口的值匹配。

要沿整个行人路径指定 Y 端口的多个位置，首先在 MATLAB®中定义Y路点的时间序列。然后，使用 From Workspace (Simulink) 模块将这些路点提供给Y。要了解如何选择和指定路点，请参阅 Select Waypoints for Unreal Engine Simulation 示例。

X — 行人的纵向位置
标量

行人沿场景X轴的纵向位置。X位于惯性Z-向上坐标系中。单位为米。

初始位置 （Initial position）[X, Y, Z] (m) 参数的 X值必须与仿真开始时该端口的值匹配。 

要沿整个行人路径指定 X 端口的多个位置，首先在 MATLAB 中定义 X 路点的时间序列。然后，使用 From Workspace (Simulink) 模块将这些路点提供给 X。要了解如何选择和指定路点，请参阅 为虚幻引擎仿真选择路点 示例。

Yaw — 行人的偏航方向角
标量

行人沿场景Z-轴的偏航方向角。偏航位于Z-向上坐标系中。单位为度。

初始旋转（Initial rotation）[Roll, Pitch, Yaw] (deg) 参数的偏航值必须与仿真开始时该端口的值匹配。

要沿整个行人路径在偏航端口指定多个方向角，首先在 MATLAB 中定义偏航路点的时间序列。然后，使用 From Workspace (Simulink) 模块将这些路径点提供给Yaw。要了解如何选择和指定路点，请参阅 为虚幻引擎仿真选择路点 示例。

输出

expand all

Location — 行人的位置
1×3 实值向量

行人在场景中的位置(X, Y, Z)，以实值向量 1×3 的形式返回。该位置基于行人原点，该原点位于地面上，位于行人的几何中心。位置（Location）值位于惯性Z-向上世界坐标系中。单位为米。

依赖

要启用此端口，请在 Ground Truth 选项卡上选择输出位置Output location (m) 和 orientation (rad)。

数据类型: double

Orientation — 行人的朝向
1×3 实值向量

行人分别绕场景的Z轴、Y轴和 X轴的偏航、俯仰和滚动方向角，以实值 1×3 向量形式返回。该方向基于行人原点，该原点位于地面上行人的几何中心。朝向（Orientation）值位于惯性Z-向上坐标系中。单位为弧度。

依赖

要启用此端口，请在 Ground Truth 选项卡上选择 Output location (m) 和 orientation (rad)。

数据类型: double

参数

expand all

行人参数

Type — 行人类型
Male 1 (默认) | Male 2 | Male 3 | Female 1 | Female 2 | Female 3 | Child

行人类型，指定为以下选项之一：

  • Male 1

  • Male 2

  • Male 3

  • Female 1

  • Female 2

  • Female 3

  • Child

Scale — 行人缩放因子
[1, 1, 1] (默认) | 1×3 实值向量

行人沿X-轴、 Y-轴和Z-轴的缩放比例因子。

Initial position [X, Y, Z] (m) — 初始行人位置
[0, 0, 0] (默认) | 1×3 实值向量

沿场景的X轴、 Y轴和Z轴的初始行人位置。该位置位于惯性Z-向上坐标系中。单位为米。

设置该参数的X和Y值以匹配仿真开始时的X和Y输入端口值。

Initial rotation [Roll, Pitch, Yaw] (deg) — 行人旋转的初始角度
[0, 0, 0] (default) | 1×3 实值向量

行人旋转的初始角度。旋转角度由行人的翻滚(roll)、俯仰(pitch)和偏航(yaw)来定义。单位为度。

设置该参数的偏航值以匹配模拟开始时的偏航（Yaw）输入端口值。

Name — 行人名称
SimulinkPedestrian1 (默认) | 行人名称

行人名称。默认情况下，在模型中使用该模块时，该模块会将 Name 参数设置为 SimulinkPedestrianX。 X 的值取决于模型中 Simulation 3D Pedestrian 模块的数量

行人名称显示为与行人同一模型内任何 Automated Driving Toolbox™ Simulation 3D 传感器的父名称 Parent name 参数中的选项。通过 Parent name 参数，可以选择要安装传感器的行人。

Sample time — 采样时间
-1 (默认) | 正标量

采样时间 Ts, ，以秒为单位。图形帧速率是采样时间的倒数。

如果将采样时间设置为 -1，则该模块将使用在 Simulation 3D Scene Configuration 模块中指定的采样时间。

真值

Output location (m) and orientation (rad) — 输出行人的位置和方向
off (默认) | on

选择此参数可分别输出行人在 Location 和 Orientation 端口的位置和方向。
