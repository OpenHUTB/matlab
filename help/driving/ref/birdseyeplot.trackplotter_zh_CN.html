trackPlotter

鸟瞰图的轨迹绘图器

collapse all in page

语法

tPlotter = trackPlotter(bep)
tPlotter = trackPlotter(bep,Name,Value)

描述

example

tPlotter = trackPlotter(bep) 创建一个 TrackPlotter 对象，用于配置在鸟瞰图上显示轨迹的方式。TrackPlotter 对象存储在输入的 birdsEyePlot 对象（bep）的 Plotters 属性中。要显示轨迹，请使用 plotTrack 函数。

tPlotter = trackPlotter(bep,Name,Value) 使用一个或多个 Name,Value 对参数设置属性。例如，trackPlotter(bep,'DisplayName','Tracks') 设置在鸟瞰图图例中显示的名称。

例子

collapse all

在鸟瞰图上创建并显示带标签的轨迹

Open Live Script

创建一个鸟瞰图，其x轴范围为0到90米，y轴范围为-35到35米。创建一个轨迹绘图器，显示每个轨迹的最多七个历史值，并将标签偏移3米放置在轨迹前方。

bep = birdsEyePlot('XLim',[0 90],'YLim',[-35 35]);
tPlotter = trackPlotter(bep,'DisplayName','Tracks','HistoryDepth',7,'LabelOffset',[3 0]);

设置三个带标签的轨迹的位置和速度。

positions = [30, 5; 30, 5; 30, 5];
velocities = [3, 0; 3, 2; 3, -3];
labels = {'T1','T2','T3'};

显示10次试验的轨迹。鸟瞰图显示了先前指定的七个历史值。

for i=1:10
   plotTrack(tPlotter,positions,velocities,labels);
   positions = positions + velocities;
end

输入参数

collapse all

bep — Bird’s-eye plot
birdsEyePlot object

Bird’s-eye plot, specified as a birdsEyePlot object.

Name-Value Arguments

将可选的参数对指定为Name1=Value1,...,NameN=ValueN的形式，其中 Name 是参数名称，Value 是相应的值。Name-value 参数必须出现在其他参数之后，但是参数对的顺序无关紧要。

在R2021a之前，使用逗号将每个名称和值分隔开，并将 Name 用引号括起来。

Example: trackPlotter('Marker','*') 将轨迹的标记符号设置为星号。

DisplayName — Plotter name to display in legend
'' (default) | character vector | string scalar

要在图例中显示的绘图器名称，指定为由 'DisplayName' 和字符向量或字符串标量组成的逗号分隔对。如果您不指定名称，鸟瞰图不会为该绘图器显示图例条目。

HistoryDepth — Number of previous track updates to display
0 (default) | integer in the range [0, 100]

要显示的先前轨迹更新数量，指定为由 'HistoryDepth' 和范围为 [0, 100] 的整数组成的逗号分隔对。当您将此值设置为 0 时，鸟瞰图不会显示任何先前的更新。

Marker — Marker symbol for tracks
'square' (default) | '+' | '*' | '.' | 'x' | ...

轨迹的标记符号，指定为由 'Marker' 和表中的标记符号之一组成的逗号分隔对。

Marker Description Resulting Marker

"+" Plus sign

"*" Asterisk

"." Point

"x" Cross

"|" Vertical line

"square" Square

"diamond" Diamond

"v" Downward-pointing triangle

">" Right-pointing triangle

"<" Left-pointing triangle

"pentagram" Pentagram

"hexagram" Hexagram

"none" No markers Not applicable

MarkerSize — Size of marker for tracks
10 (default) | positive integer

轨迹的标记符号大小，指定为由 'MarkerSize' 和以点为单位的正整数组成的逗号分隔对。

MarkerEdgeColor — Marker outline color for tracks
[0 0 0] (black) (default) | RGB triplet | hexadecimal color code | color name | short color name

轨迹的标记符号轮廓颜色，指定为由 'MarkerEdgeColor' 和 RGB 三元组、十六进制颜色代码、颜色名称或简短颜色名称组成的逗号分隔对。

对于自定义颜色，请指定一个RGB三元组或十六进制颜色代码。

  • RGB三元组是一个包含三个元素的行向量，其元素指定颜色的红色、绿色和蓝色分量的强度。强度必须在范围 [0,1]内，例如，[0.4 0.6 0.7]。

  • 十六进制颜色代码是一个以井号 (#) 开头的字符向量或字符串标量，后面跟着三个或六个十六进制数字，这些数字的范围可以从 0 到 F。这些值不区分大小写。因此，颜色代码 "#FF8800","#ff8800", "#F80"和 "#f80" 是等效的。

另外，您可以通过名称指定一些常见的颜色。下表列出了命名颜色选项、对应的RGB三元组和十六进制颜色代码。

FontSize — Font size for labeling tracks
10 points (default) | positive integer

标记轨道的字体大小，由逗号分隔的一对组成，其中包括 'FontSize' 和正整数（以字体点为单位）。

LabelOffset — Gap between label and positional point
[0 0] (default) | real-valued vector of the form [x y]

标签和位置点之间的间隔，由逗号分隔的一对组成，包括 'LabelOffset' 和形式为[x y]的实值向量。单位为米。

VelocityScaling — Scale factor for magnitude length of velocity vectors
1 (default) | positive real scalar

速度矢量的大小比例因子，由逗号分隔的一对组成，包括 'VelocityScaling' 和正实数标量。鸟瞰图将矢量的大小渲染为M × VelocityScaling，其中 M 是速度的大小。

Tag — Tag associated with plotter object
'PlotterN' (default) | character vector | string scalar

与绘图器对象相关联的标签，由逗号分隔的一对组成，包括 'Tag' 和字符向量或字符串标量。默认值为 'PlotterN'，其中 N 是与输入的 birdsEyePlot 对象相关联的第 N 个绘图器的整数对应值。

输出参数

collapse all

tPlotter — Track plotter
TrackPlotter object

轨迹绘图器，返回为 TrackPlotter 对象。你可以通过修改其属性值来修改这个对象。属性名称对应于 trackPlotter 函数的名称-值对参数。

tPlotter 存储在输入的 birdsEyePlot 对象（bep）的 Plotters 属性中。要绘制轨迹，请使用 plotTrack 函数。
