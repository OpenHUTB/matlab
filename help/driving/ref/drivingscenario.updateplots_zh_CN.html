updatePlots

更新驾驶场景绘图

collapse all in page

语法

updatePlots(scenario)

描述

example

updatePlots(scenario) 用于更新驾驶场景 scenario 的所有现有绘图显示。每次调用 advance 函数推进模拟时，驾驶场景绘图会自动更新。如果您更新了任何参与者属性并希望刷新绘图而不必调用 advance 函数，请使用 updatePlots 函数。

例子

更新驾驶场景绘图

Open Live Script

在更改驾驶场景的各个方面后，更新驾驶场景的绘图。

创建一个包含一辆车辆在一条直线、长度为25米的道路段上的驾驶场景，并绘制该场景。

scenario = drivingScenario;
roadcenters = [0 0 0; 25 0 0];
road(scenario,roadcenters);
  
v = vehicle(scenario,'ClassID',1);
v.Position = [1 0 0];

plot(scenario)

使用追逐绘图从车辆的视角绘制驾驶场景。

chasePlot(v)

为车辆设置一个新的位置。

v.Position = [12 0 0];

更新两个绘图，以显示车辆的新位置。
