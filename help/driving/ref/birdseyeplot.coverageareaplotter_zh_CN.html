coverageAreaPlotter

鸟瞰图的覆盖区域绘图器

collapse all in page

语法

caPlotter = coverageAreaPlotter(bep)
caPlotter = coverageAreaPlotter(bep,Name,Value)

描述

caPlotter = coverageAreaPlotter(bep) 创建一个 CoverageAreaPlotter 对象，用于配置在鸟瞰图上显示传感器覆盖区域。CoverageAreaPlotter 对象存储在输入的 birdsEyePlot 对象 bep 的 Plotters 属性中。要显示传感器覆盖区域，请使用 plotCoverageArea 函数。

example

caPlotter = coverageAreaPlotter(bep,Name,Value) 使用一个或多个 Name,Value 键值对参数设置属性。例如，coverageAreaPlotter(bep,'DisplayName','Coverage area') 设置在鸟瞰图图例中显示的名称。

例子

collapse all

创建并显示在鸟瞰图上的覆盖区域

Open Live Script

创建一个鸟瞰图，x轴范围从0到90米，y轴范围从-35到35米。创建一个覆盖区域绘图器，将覆盖区域显示为红色。

bep = birdsEyePlot('XLim',[0 90],'YLim',[-35 35]);
caPlotter = coverageAreaPlotter(bep,'DisplayName','Radar coverage area','FaceColor','r');

显示一个视场角为35度、范围为60米的覆盖区域。将覆盖区域传感器安装在距离原点前方1米的位置。将传感器的方向角设置为0度。

mountPosition = [1 0];
range = 60;
orientation = 0;
fieldOfView = 35;
plotCoverageArea(caPlotter,mountPosition,range,orientation,fieldOfView);

输入参数

collapse all

bep — Bird’s-eye plot
birdsEyePlot object

鸟瞰图，指定为 birdsEyePlot 对象。

Name-Value Arguments

将可选的参数对指定为 Name1=Value1,...,NameN=ValueN 的形式，其中 Name 是参数名称，Value 是对应的值。名称-值参数必须出现在其他参数之后，但是参数对的顺序不重要。

在R2021a之前，使用逗号将每个名称和值分隔开，并将 Name 用引号括起来。

Example: coverageAreaPlotter('FaceColor','red') 将传感器覆盖区域的填充颜色设置为红色。

DisplayName — Plotter name to display in legend
'' (default) | character vector | string scalar

绘图器在图例中显示的名称，指定为由 'DisplayName' 和字符向量或字符串标量组成的逗号分隔对。如果您不指定名称，鸟瞰图不会为绘图器显示图例条目。

FaceColor — Fill color of coverage areas
[0 0 0] (black) (default) | RGB triplet | hexadecimal color code | color name | short color name

覆盖区域的填充颜色，指定为由 'FaceColor' 和 RGB 三元组、十六进制颜色代码、颜色名称或短颜色名称组成的逗号分隔对。

对于自定义颜色，请指定一个 RGB 三元组或十六进制颜色代码。

  • RGB 三元组是一个包含三个元素的行向量，这些元素指定颜色的红色、绿色和蓝色分量的强度。强度必须在[0,1]范围内，例如，[0.4 0.6 0.7]。

  • 十六进制颜色代码是以井号 (#) 开头，后面跟着三个或六个十六进制数字的字符向量或字符串标量。这些数字的取值范围是从 0 到 F。这些代码对大小写不敏感。因此，颜色代码 "#FF8800", "#ff8800", "#F80" 和 "#f80" 是等价的。

EdgeColor — Border color of coverage areas
[0 0 0] (black) (default) | RGB triplet | hexadecimal color code | color name | short color name

覆盖区域的边框颜色，可以使用逗号分隔的键值对 'EdgeColor' 来指定，其取值可以是 RGB 三元组、十六进制颜色代码、颜色名称或简短颜色名称。

对于自定义颜色，请指定一个 RGB 三元组或十六进制颜色代码。

  • RGB 三元组是一个包含三个元素的行向量，这些元素指定颜色的红色、绿色和蓝色分量的强度。强度必须在 [0,1] 范围内，例如，[0.4 0.6 0.7]。

FaceAlpha — Transparency of coverage areas
0.1 (default) | scalar in the range [0, 1]

覆盖区域的透明度可以通过指定 'FaceAlpha' 和一个范围为[0, 1]的标量来设置。当值为 0 时，覆盖区域完全透明；当值为 1 时，覆盖区域完全不透明。

Tag — Tag associated with plotter object
'PlotterN' (default) | character vector | string scalar

与绘图器对象相关联的标签可以通过指定 'Tag' 和字符向量或字符串标量的逗号分隔对来设置。默认值为 'PlotterN'，其中 N 是与输入的 birdsEyePlot 对象相关联的第 N 个绘图器对应的整数。

输出参数

collapse all

caPlotter — Coverage area plotter
CoverageAreaPlotter object

输出参数为 CoverageAreaPlotter 对象，可以通过更改其属性值来修改该对象。属性名称对应于 coverageAreaPlotter 函数的名称-值对参数。

caPlotter 存储在输入的 birdsEyePlot 对象（bep）的 Plotters 属性中。要绘制覆盖区域，请使用 plotCoverageArea 函数。
