clearPlotterData

清除鸟瞰图中的数据

collapse all in page

语法

clearPlotterData(bep)

描述

example

clearPlotterData(bep) 函数用于清除指定鸟瞰图中显示的所有绘图器数据。但是，图例条目和覆盖区域不会从图中清除。

要清除特定绘图器的数据，请使用 clearData 函数。

例子

collapse all

创建带有覆盖区域和检测绘图器的鸟瞰图

Open Live Script

创建一个鸟瞰图，x轴范围为0到90米，y轴范围为-35到35米。配置该图以包含一个雷达覆盖区域绘图器和一个检测绘图器。设置这些绘图器的显示名称。

bep = birdsEyePlot('XLim',[0 90],'YLim',[-35 35]);
coverageAreaPlotter(bep,'DisplayName','Radar coverage area');
detectionPlotter(bep,'DisplayName','Radar detections');

使用 findPlotter 函数通过显示名称定位绘图器。

caPlotter = findPlotter(bep,'DisplayName','Radar coverage area');
radarPlotter = findPlotter(bep,'DisplayName','Radar detections');

绘制覆盖区域和检测到的物体。

plotCoverageArea(caPlotter,[1 0],30,0,35);
plotDetection(radarPlotter,[30 5; 30 -10; 30 15]);

清除图形中的数据。

clearPlotterData(bep);

输入参数

collapse all

bep — Bird’s-eye plot
birdsEyePlot object

鸟瞰图，指定为 birdsEyePlot 对象。

版本历史

Introduced in R2017a
