plotCoverageArea

在鸟瞰图上显示传感器的覆盖区域

collapse all in page

语法

plotCoverageArea(caPlotter,position,range,orientation,fieldOfView)

描述

example

plotCoverageArea(caPlotter,position,range,orientation,fieldOfView) 函数在鸟瞰图上显示自车传感器的覆盖区域。需要指定传感器的位置、范围、方向角和视野角度。覆盖区域绘图器 caPlotter 关联了一个 birdsEyePlot 对象，并配置传感器覆盖区域的显示方式。

例子

collapse all

显示雷达传感器的覆盖区域

Open Live Script

创建一个鸟瞰图，其中x轴范围从0到90米，y轴范围从-35到35米。

bep = birdsEyePlot('XLim',[0 90],'YLim',[-35 35]);

为鸟瞰图创建一个覆盖区域绘图器。

caPlotter = coverageAreaPlotter(bep,'DisplayName','Radar coverage area');

显示一个具有35度视野和60米范围的覆盖区域。将覆盖区域传感器安装在原点前1米的位置。将传感器的方向角设置为0度。

mountPosition = [1 0];
range = 60;
orientation = 0;
fieldOfView = 35;

绘制覆盖区域。

plotCoverageArea(caPlotter,mountPosition,range,orientation,fieldOfView);

显示来自车辆四个角落的传感器覆盖区域

Open Live Script

创建一个鸟瞰图，x 轴范围从-100米到100米，y 轴范围从-100米到100米

bep = birdsEyePlot('XLim',[-100 100],'YLim',[-100 100]);

为车辆上的每个传感器位置创建具有唯一显示名称和填充颜色的覆盖区域绘图器。

rearLeftPlotter = coverageAreaPlotter(bep,'DisplayName','Rear left','FaceColor','r');
rearRightPlotter = coverageAreaPlotter(bep,'DisplayName','Rear right','FaceColor','b');
frontLeftPlotter = coverageAreaPlotter(bep,'DisplayName','Front left','FaceColor','y');
frontRightPlotter = coverageAreaPlotter(bep,'DisplayName','Front right','FaceColor','g');

设置传感器的位置、范围、方向和视野。传感器的最大范围为90米，视野为30度。绘制覆盖区域。

plotCoverageArea(rearLeftPlotter,[0 0.9],90,120,30);
plotCoverageArea(rearRightPlotter,[0 -0.9],90,-120,30);
plotCoverageArea(frontLeftPlotter,[2.8 0.9],90,60,30);
plotCoverageArea(frontRightPlotter,[2.8 -0.9],90,-60,30);

输入参数

collapse all

caPlotter — Coverage area plotter
CoverageAreaPlotter object

覆盖区域绘图器，以 CoverageAreaPlotter 对象的形式指定。该对象存储在 birdsEyePlot 对象的 Plotters 属性中，并配置bird's-eye plot中覆盖区域的显示。要创建此对象，请使用 coverageAreaPlotter 函数。

position — Position of sensor
real-valued vector of the form [XOriginOffset YOriginOffset]

传感器在车辆坐标系中的位置，以实值向量的形式表示，形式为 [XOriginOffset YOriginOffset]。单位为米。

  • XOriginOffset 指定传感器相对于原点向前的距离。

  • YOriginOffset 指定传感器相对于原点向左的距离。

原点位于后轴的中心，如车辆坐标系图中所示。

range — Range of sensor
positive real scalar

传感器的范围，指定为正实数标量。单位为米。

orientation — Orientation angle of sensor
real scalar

传感器相对于自车X轴的方向角，指定为实数标量。单位为度。orientation（方向角）在逆时针方向（向左）为正。

fieldOfView — Field of view of sensor
positive real scalar

传感器覆盖区域的视野，指定为正实数标量。单位为度。
