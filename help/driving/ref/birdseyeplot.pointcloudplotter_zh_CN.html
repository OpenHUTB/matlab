pointCloudPlotter

用于鸟瞰图的点云绘图器

collapse all in page

语法

pcPlotter = pointCloudPlotter(bep)
pcPlotter = pointCloudPlotter(bep,Name,Value)

描述

example

pcPlotter = pointCloudPlotter(bep) 创建一个点云绘图器对象，用于配置在鸟瞰图上显示激光雷达点云数据。该点云绘图器对象存储在输入的鸟瞰图对象（bep）的 Plotters 属性中。要绘制激光雷达点云数据，请使用 plotPointCloud 函数。

pcPlotter = pointCloudPlotter(bep,Name,Value) 使用一个或多个名称-值对参数指定选项。例如，'DisplayName','Point Cloud' 将在鸟瞰图图例中显示的显示名称设置为 "Point Cloud"。

例子

collapse all

生成多个参与者的激光雷达点云数据

Open Live Script

通过使用 lidarPointCloudGenerator 系统对象，为具有多个参与者的驾驶场景生成激光雷达点云数据。使用 drivingScenario 对象创建驾驶场景，其中包含自车、行人和其他两辆车。

创建并绘制一个包含多辆车辆的驾驶场景

创建一个驾驶场景。

scenario = drivingScenario;

向驾驶场景中添加一条直线道路。该道路在每个方向上有一条车道。

roadCenters = [0 0 0; 70 0 0];
laneSpecification = lanespec([1 1]);
road(scenario,roadCenters,'Lanes',laneSpecification);

向驾驶场景中添加一个自车。

egoVehicle = vehicle(scenario,'ClassID',1,'Mesh',driving.scenario.carMesh);
waypoints = [1 -2 0; 35 -2 0];
smoothTrajectory(egoVehicle,waypoints,10);

向驾驶场景中添加一辆卡车、一个行人和一辆自行车，并绘制场景。

truck = vehicle(scenario,'ClassID',2,'Length', 8.2,'Width',2.5,'Height',3.5, ...
  'Mesh',driving.scenario.truckMesh);
waypoints = [70 1.7 0; 20 1.9 0];
smoothTrajectory(truck,waypoints,15);

pedestrian = actor(scenario,'ClassID',4,'Length',0.24,'Width',0.45,'Height',1.7, ...
  'Mesh',driving.scenario.pedestrianMesh);
waypoints = [23 -4 0; 10.4 -4 0];
smoothTrajectory(pedestrian,waypoints,1.5);

bicycle = actor(scenario,'ClassID',3,'Length',1.7,'Width',0.45,'Height',1.7, ...
  'Mesh',driving.scenario.bicycleMesh);
waypoints = [12.7 -3.3 0; 49.3 -3.3 0];
smoothTrajectory(bicycle,waypoints,5);

plot(scenario,'Meshes','on')

生成并绘制激光雷达点云数据

创建一个 lidarPointCloudGenerator 系统对象。

lidar = lidarPointCloudGenerator;

将参与者配置文件和驾驶场景中的自车参与者ID添加到系统对象中。

lidar.ActorProfiles = actorProfiles(scenario);
lidar.EgoVehicleActorID = egoVehicle.ActorID;

绘制点云数据。

bep = birdsEyePlot('Xlimits',[0 70],'YLimits',[-30 30]);
plotter = pointCloudPlotter(bep);
legend('off');
while advance(scenario)
    tgts = targetPoses(egoVehicle);
    rdmesh = roadMesh(egoVehicle);
    [ptCloud,isValidTime] = lidar(tgts,rdmesh,scenario.SimulationTime);
    if isValidTime
        plotPointCloud(plotter,ptCloud);
    end
end

输入参数

collapse all

bep — Bird’s-eye plot
birdsEyePlot object

Bird’s-eye plot, specified as a birdsEyePlot object.

Name-Value Arguments

将可选的参数对指定为Name1=Value1,...,NameN=ValueN，其中 Name 是参数名称，Value 是相应的值。名称-值参数必须在其他参数之后出现，但是参数对的顺序无关紧要。

在R2021a之前，使用逗号将每个名称和值分隔开，并用引号将 Name 括起来。

Example: 'DisplayName','Point Cloud' 将在鸟瞰图图例中显示的显示名称设置为"Point Cloud"。

DisplayName — Plotter name to display in legend
'' (default) | character vector | string scalar

要在图例中显示的绘图器名称，指定为由 'DisplayName' 和字符向量或字符串标量组成的逗号分隔对。如果不指定名称，则鸟瞰图不会为绘图器显示图例项。

Data Types: char | string

PointSize — Size of marker for points in point cloud
6 (default) | positive integer

点云中点的标记大小，指定为由 'PointSize' 和以点为单位的正整数组成的逗号分隔对。

Color — Point fill color
'none' (default) | RGB triplet | hexadecimal color code | color name | short color name

点的填充颜色，指定为由'Color' 和RGB三元组、十六进制颜色代码、颜色名称或简短颜色名称组成的逗号分隔对。

对于自定义颜色，请指定一个RGB三元组或十六进制颜色代码。

  • RGB三元组是一个包含三个元素的行向量，其元素指定颜色的红色、绿色和蓝色分量的强度。强度必须在[0,1]范围内，例如，[0.4 0.6 0.7]。

  • 十六进制颜色代码是一个以井号 (#) 开头的字符向量或字符串标量，后面跟着三个或六个十六进制数字，范围从 0 到 F。这些值不区分大小写。因此，颜色代码"#FF8800", "#ff8800", "#F80"和 "#f80" 是等效的。

另外，您还可以通过名称指定一些常见的颜色。下表列出了命名颜色选项、对应的RGB三元组和十六进制颜色代码。

Tag — Tag associated with plotter object
'PlotterN' (default) | character vector | string scalar

与绘图器对象相关联的标签，指定为由 'Tag' 和字符向量或字符串标量组成的逗号分隔对。默认值为 'PlotterN'，其中 N 是与输入 birdsEyePlot 对象关联的第 N 个绘图器对应的整数。

输出参数

collapse all

pcPlotter — Point cloud plotter
pointCloudPlotter object

点云绘图器，返回为 pointCloudPlotter 对象。您可以通过更改其属性值来修改此对象。

pcPlotter 存储在输入 bep 的 Plotters 属性中。要绘制点云数据，请使用 plotPointCloud 函数。
