targetPoses

目标位置和方向相对于自我车辆

collapse all in page

语法

poses = targetPoses(ac)
poses = targetPoses(ac,range)

描述

example

poses = targetPoses(ac) 返回驾驶场景中所有目标相对于自我车辆参与者 ac 的姿势。参见 Ego Vehicle and Targets 了解更多详细信息。

example

poses = targetPoses(ac,range) 返回车辆参与者周围指定范围内目标的姿态。

在自我车辆和场景坐标之间转换目标姿态

Open Live Script

在一个简单的驾驶场景中，获得目标车辆在自我车辆坐标系中的姿态。然后将这些姿势转换回驾驶场景的世界坐标。

创建一个驾驶场景。

scenario = drivingScenario;

创建目标参与者。

actor(scenario,'ClassID',1, ...
    'Position',[10 20 30], ...
    'Velocity',[12 113 14], ...
    'Yaw',54, ...
    'Pitch',25, ...
    'Roll',22, ...
    'AngularVelocity',[24 42 27]);

actor(scenario,'ClassID',1, ...
    'Position',[17 22 12], ...
    'Velocity',[19 13 15], ...
    'Yaw',45, ...
    'Pitch',52, ...
    'Roll',2, ...
    'AngularVelocity',[42 24 29]);

添加一个自我车辆参与者。

egoActor = actor(scenario,'ClassID',1, ...
    'Position',[1 2 3], ...
    'Velocity',[1.2 1.3 1.4], ...
    'Yaw',4, ...
    'Pitch',5, ...
    'Roll',2, ...
    'AngularVelocity',[4 2 7]);

使用 actorPoses 函数返回场景中所有参与者的姿势。姿态属性(位置、速度和方向)在驾驶场景的世界坐标中。将目标参与者保存到一个单独的变量中，并检查第一个目标参与者的姿势。

allPoses = actorPoses(scenario);
targetPosesScenarioCoords = allPoses(1:2);
targetPosesScenarioCoords(1)
ans = struct with fields:
            ActorID: 1
           Position: [10 20 30]
           Velocity: [12 113 14]
               Roll: 22
              Pitch: 25
                Yaw: 54
    AngularVelocity: [24 42 27]

使用 driving.scenario.targetsToEgo 函数将目标姿势转换为自我参与者的自我中心坐标。检查第一个参与者的姿势。

targetPosesEgoCoords = driving.scenario.targetsToEgo(targetPosesScenarioCoords,egoActor);
targetPosesEgoCoords(1)
ans = struct with fields:
            ActorID: 1
           Position: [7.8415 18.2876 27.1675]
           Velocity: [18.6826 112.0403 9.2960]
               Roll: 16.4327
              Pitch: 23.2186
                Yaw: 47.8114
    AngularVelocity: [-3.3744 47.3021 18.2569]

或者，使用 targetPoses 函数来获取车辆坐标中所有目标参与者的姿势。显示第一个目标姿势，它与先前计算的姿势相匹配。

targetPosesEgoCoords = targetPoses(egoActor);
targetPosesEgoCoords(1)
ans = struct with fields:
            ActorID: 1
            ClassID: 1
           Position: [7.8415 18.2876 27.1675]
           Velocity: [18.6826 112.0403 9.2960]
               Roll: 16.4327
              Pitch: 23.2186
                Yaw: 47.8114
    AngularVelocity: [-3.3744 47.3021 18.2569]

使用 driving.scenario.targetsToScenario 将目标姿态转换回场景的世界坐标。显示与原始目标姿势匹配的第一个目标姿势。

targetPosesScenarioCoords = driving.scenario.targetsToScenario(targetPosesEgoCoords,egoActor);
targetPosesScenarioCoords(1)
ans = struct with fields:
            ActorID: 1
            ClassID: 1
           Position: [10.0000 20.0000 30.0000]
           Velocity: [12.0000 113.0000 14.0000]
               Roll: 22
              Pitch: 25.0000
                Yaw: 54
    AngularVelocity: [24.0000 42.0000 27.0000]

获取传感器范围内的目标姿态

Open Live Script

获取安装在自车上的传感器的最大范围内的目标姿态。

创建一个驾驶场景。该场景包括一条75米长的直路、一个自车和两辆目标车辆。

  • 最近的目标车辆距离自车45米，与自车处于同一车道。

  • 最远的目标车辆距离自车65米，与自车处于相对车道。

绘制驾驶场景。

scenario = drivingScenario;

roadCenters = [0 0 0; 75 0 0];
laneSpecification = lanespec([1 1]);
road(scenario,roadCenters,'Lanes',laneSpecification,'Name','Road');

egoVehicle = vehicle(scenario, ...
    'ClassID',1, ...
    'Position',[4 -2 0], ...
    'Name','Ego');

vehicle(scenario, ...
    'ClassID',1, ...
    'Position',[45 -1.7 0], ...
    'Name','Near Target');

vehicle(scenario, ...
    'ClassID',1, ...
    'Position',[65 2 0], ...
    'Yaw',-180, ...
    'Name','Far Target');

plot(scenario)

在自车的前保险杠上安装一个视觉传感器。将传感器配置为最大检测范围为50米。

sensor = visionDetectionGenerator('SensorIndex',1, ...
    'SensorLocation',[3.7 0], ...
    'MaxRange',50);

绘制车辆的轮廓和传感器的覆盖范围。最近的目标车辆在传感器的范围内，但最远的目标车辆不在范围内。

bep = birdsEyePlot;

olPlotter = outlinePlotter(bep);
[position,yaw,length,width,originOffset,color] = targetOutlines(egoVehicle);
plotOutline(olPlotter,position,yaw,length,width, ...
    'OriginOffset',originOffset,'Color',color)

caPlotter = coverageAreaPlotter(bep,'DisplayName','Coverage area','FaceColor','blue');
mountPosition = sensor.SensorLocation;
range = sensor.MaxRange;
orientation = sensor.Yaw;
fieldOfView = sensor.FieldOfView(1);
plotCoverageArea(caPlotter,mountPosition,range,orientation,fieldOfView);

获取传感器范围内的目标姿态。输出结构中只包含最近的目标的姿态，该目标距离自车不到50米。

poses = targetPoses(egoVehicle,range)
poses = struct with fields:
            ActorID: 2
            ClassID: 1
           Position: [41 0.3000 0]
           Velocity: [0 0 0]
               Roll: 0
              Pitch: 0
                Yaw: 0
    AngularVelocity: [0 0 0]

输入参数

collapse all

ac — Actor
Actor object | Vehicle object

属于 drivingScenario 对象的参与者，可以指定为 Actor 或 Vehicle 对象。要创建这些对象，请分别使用 actor 和 vehicle 函数。

range — 自车周围的圆形范围
nonnegative real scalar

围绕自车参与者的圆形范围，指定为非负实数标量。targetPoses 函数仅返回位于此范围内的目标的姿态。单位为米。

输出参数

collapse all

poses — 目标姿态
structure | array of structures

目标姿态以自车坐标系返回，可以作为结构体或结构体数组返回。不包括自车参与者 ac 的姿态。

目标姿态定义了目标在自车坐标系中的位置、速度和方向。目标姿态还包括参与者位置和方向的变化速率。

每个姿态结构具有以下字段。

Field Description
ActorID

场景定义的参与者标识符，指定为正整数。
ClassID 分类标识符，指定为非负整数。0 表示未知或未分配类别的对象。
Position

参与者的位置，以[x y z]形式的实值向量指定。单位为米。
Velocity

参与者在x、y、z方向上的速度(v)，表示为[vx vy vz]形式的实值向量。单位是米每秒。
Roll

参与者的滚动角度，指定为实标量。单位是度。
Pitch

参与者的俯仰角，用实标量指定。单位是度。
Yaw

参与者的偏航角，用实标量指定。单位是度。
AngularVelocity

参与者在x、y和z方向上的角速度 (ω) ，以[ωx ωy ωz]形式的实值向量指定。单位为度/秒。

要获取这些结构字段的完整定义，请参阅 actor 和 vehicle 函数。

自我车辆和目标

在驾驶场景中，您可以指定一个参与者作为所有其他参与者的观察者，类似于汽车驾驶员观察所有其他汽车的方式。观察者参与者被称为自我参与者或更具体地说是自我车辆。从自我车辆的视角来看，所有其他参与者（如车辆和行人）都是被观察的参与者，称为目标。自我车辆坐标系以自我车辆为中心并与其对齐。驾驶场景的坐标是世界坐标系。
