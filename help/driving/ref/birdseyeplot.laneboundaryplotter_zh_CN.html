laneBoundaryPlotter

用于鸟瞰图的车道边界绘图工具

collapse all in page

语法

lbPlotter = laneBoundaryPlotter(bep)
lbPlotter = laneBoundaryPlotter(bep,Name,Value)

描述

example

lbPlotter = laneBoundaryPlotter(bep) 创建一个 LaneBoundaryPlotter 对象，用于配置鸟瞰图上车道边界的显示。LaneBoundaryPlotter 对象存储在输入的 birdsEyePlot 对象（bep）的 Plotters 属性中。要显示车道边界，请使用 plotLaneBoundary 函数。

lbPlotter = laneBoundaryPlotter(bep,Name,Value) 使用一个或多个 Name,Value 对参数设置属性。例如，laneBoundaryPlotter(bep,'DisplayName','Lane boundaries') 设置出现在鸟瞰图图例中的显示名称。

例子

collapse all

在鸟瞰图上创建并显示车道边界

Open Live Script

创建左车道和右车道边界。

leftlb = parabolicLaneBoundary([-0.001,0.01,-1.8]);
rightlb = parabolicLaneBoundary([-0.001,0.01,1.8]);

创建一个鸟瞰图，其中x轴范围从0到30米，y轴范围从-5到5米。

bep = birdsEyePlot('XLimits',[0 30],'YLimits',[-5 5]);

创建一个车道边界绘图工具。

lbPlotter = laneBoundaryPlotter(bep,'DisplayName','Lane boundaries');

在鸟瞰图上显示车道边界。

plotLaneBoundary(lbPlotter,[leftlb rightlb]);

输入参数

collapse all

bep — Bird’s-eye plot
birdsEyePlot object

鸟瞰图，指定为 birdsEyePlot 对象。

Name-Value Arguments

将可选的参数对指定为 Name1=Value1,...,NameN=ValueN，其中 Name 是参数名称，Value 是相应的值。Name-Value参数必须在其他参数之后出现，但是对于参数对的顺序没有要求。

在R2021a之前，使用逗号将每个名称和值分隔开，并将 Name 用引号括起来。

Example: laneBoundaryPlotter('Color','red') sets the color of lane boundaries to red.

DisplayName — Plotter name to display in legend
'' (default) | character vector | string scalar

图形绘制器在图例中显示的名称，以 'DisplayName' 和字符向量或字符串标量的逗号分隔对的形式指定。如果您不指定名称，鸟瞰图不会为绘制器显示图例条目。

Color — Lane boundary color
[0 0 0] (black) (default) | RGB triplet | hexadecimal color code | color name | short color name

车道边界颜色，以 'Color' 和 RGB 三元组、十六进制颜色代码、颜色名称或简短颜色名称的逗号分隔对的形式指定。

对于自定义颜色，请指定RGB三元组或十六进制颜色代码。

  • RGB三元组是一个由三个元素组成的行向量，其元素指定颜色的红色、绿色和蓝色分量的强度。强度必须在 [0,1] 范围内，例如，[0.4 0.6 0.7]。

  • 十六进制颜色代码是以井号 (#) 开头，后跟三个或六个十六进制数字的字符向量或字符串标量。这些数字的取值范围是 0 到 F。颜色代码不区分大小写。因此，颜色代码 "#FF8800", "#ff8800", "#F80" 和 "#f80" 是等效的。

或者，您可以通过名称指定一些常见的颜色。下表列出了命名颜色选项、等效的RGB三元组和十六进制颜色代码。

LineStyle — Lane boundary line style
'-' (default) | '--' | ':' | '-.' | 'none'

车道边界线样式，指定为逗号分隔的参数对，包括 'LineStyle' 和下表中列出的选项之一。

Line Style Description Resulting Line
"-" Solid line

"--" Dashed line

":" Dotted line

"-." Dash-dotted line

"none" No line No line

Tag — Tag associated with plotter object
'PlotterN' (default) | character vector | string scalar

与绘图器对象相关联的标签，指定为逗号分隔的参数对，包括 'Tag' 和字符向量或字符串标量。默认值为 'PlotterN'，其中 N 是与输入的 birdsEyePlot 对象相关联的第 N 个绘图器的整数。

输出参数

collapse all

lbPlotter — Lane boundary plotter
LaneBoundaryPlotter object

车道边界绘制器，返回一个 LaneBoundaryPlotter 对象。您可以通过更改其属性值来修改此对象。属性名称对应于 laneBoundaryPlotter 函数的名称-值对参数。

lbPlotter 存储在输入的 birdsEyePlot 对象（ bep）的 Plotters 属性中。要绘制车道边界，请使用 plotLaneBoundary 函数。
