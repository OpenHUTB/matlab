plotPath

在鸟瞰图上显示参与者路径

collapse all in page

语法

plotPath(pPlotter,pathCoords)

描述

example

plotPath(pPlotter,pathCoords) 在鸟瞰图上显示来自路径坐标列表的参与者路径。路径绘图器对象 pPlotter 与 birdsEyePlot 对象相关联，并配置指定路径的显示。

要删除与路径绘图器 pPlotter 关联的所有路径，请调用 clearData 函数并将 pPlotter 指定为输入参数。

例子

collapse all

绘制自车的路径

Open Live Script

创建一个宽度为3米的车道。

lb = parabolicLaneBoundary([-0.001,0.01,1.5]);
rb = parabolicLaneBoundary([-0.001,0.01,-1.5]);

手动计算从x轴0米到30米的车道边界模型。

xWorld = (0:30)';
yLeft = computeBoundaryModel(lb,xWorld);
yRight = computeBoundaryModel(rb,xWorld);

创建一个鸟瞰图和车道边界绘图器，并在鸟瞰图上显示车道信息。

bep = birdsEyePlot('XLimits',[0 30],'YLimits',[-5 5]);
lanePlotter = laneBoundaryPlotter(bep,'DisplayName','Lane boundaries');
plotLaneBoundary(lanePlotter,{[xWorld,yLeft],[xWorld,yRight]});

创建一个路径绘图器。创建并显示一个自车路径，该路径通过车道的中心行驶。

yCenter = (yLeft + yRight)/2;
egoPathPlotter = pathPlotter(bep,'DisplayName','Ego vehicle path');
plotPath(egoPathPlotter,{[xWorld,yCenter]});

输入参数

collapse all

pPlotter — Path plotter
PathPlotter object

路径绘图器是一个 PathPlotter 对象，存储在 birdsEyePlot 对象的 Plotters 属性中，用于配置在鸟瞰图中显示指定参与者路径的方式。要创建该对象，请使用 pathPlotter 函数。

pathCoords — Path coordinates
cell array of M-by-2 real-valued matrices

路径坐标是一个由M×2实值矩阵组成的单元数组。每个矩阵表示不同路径的坐标。M是路径中的坐标数，对于每个路径可以是不同的。每个矩阵的第一列和第二列表示路径曲线的 (X, Y) 位置。正X方向指向车辆中心的前方。正Y方向指向车辆坐标系原点的左侧，该原点为车辆后轴的中心，如车辆坐标系示意图所示。

路径坐标是相对于自车的坐标。
