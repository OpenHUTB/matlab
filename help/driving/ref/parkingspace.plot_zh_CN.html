plot

绘制停车位

collapse all in page

语法

plot(space)
plot(space,Origin=visibility)

描述

example

plot(space) 绘制一个鸟瞰图，展示一个停车位的尺寸、角度、颜色和类型。在将其添加到停车场之前，可以使用此绘图来检查停车位的属性。

example

plot(space,Origin=visibility) 将停车位原点的可见性设置为 "on" 或 "off"。

例子

collapse all

使用预定义的布局创建停车场

Open Live Script

通过使用预定义的布局以及配置这些布局的选项，您可以探索可以创建的各种停车场。

定义用于填充停车场的停车位。修改停车位的宽度、长度或角度，以及停车位标线的宽度和强度。绘制停车位。

width = 2.6; % m
length = 5.5; % m
angle = 90; % deg
markingWidth = 0.15;
markingStrength = 1;

space = parkingSpace(Width=width, ...
                    Length=length, ...
                    Angle=angle, ...
                    MarkingWidth=markingWidth, ...
                    MarkingStrength=markingStrength);
plot(space)

创建一个包含50米乘50米停车场的驾驶场景。指定一个预定义的停车场布局和一个最小的行车道宽度。在给定最小行车道宽度的约束条件下，生成的停车场将填满尽可能多的适合布局的停车位。修改停车位、布局类型和行车道宽度，并观察对停车场的影响。例如：

  • 当你增加停车位的大小或最小行车道宽度时，适合停车场中间的停车格数目会减少。

  • 如果选择了 HorizontalWithEdges 或 VerticalWithEdges 布局，其中一条边上的停车位数量会比其他边少。这条边包含一个宽度为 DrivingLaneWidth 的行车道，使车辆可以进入停车场。

scenario = drivingScenario;
vertices = [0 0; 50 0; 50 -50; 0 -50];
parkingLayout = "Vertical";
drivingLaneWidth = 2.6; % m

parkingLot(scenario,vertices, ...
    ParkingSpace=space, ...
    ParkingLayout=parkingLayout, ...
    DrivingLaneWidth=drivingLaneWidth);

plot(scenario)

创建包含多种停车位类型的停车场

Open Live Script

创建一个包含停车位、禁停区域和无障碍停车位混合的停车场。

定义在停车场中使用的停车位。使用默认设置。绘制该停车位。

space = parkingSpace;
plot(space,Origin="off")

定义在停车场中使用的禁停区域。指定一个白色的颜色，并且宽度比默认停车位的宽度小1米。绘制该区域。

noSpace = parkingSpace(Type="NoParking",Width=(space.Width - 1),MarkingColor="White");
plot(noSpace,Origin="off")

定义在停车场中使用的无障碍停车位。指定一个比默认停车位宽度大1米的宽度。绘制该停车位。

accessibleSpace = parkingSpace(Type="Accessible",Width=(space.Width + 1));
plot(accessibleSpace,Origin="off")

创建一个包含非矩形布局的停车场的驾驶场景。绘制停车场并显示可以添加停车位的边缘编号。

scenario = drivingScenario;
vertices = [0 0; 32 -10.5; 32 -50.5; 25 -57.5; 0 -57.5];
lot = parkingLot(scenario,vertices);
plot(scenario,ParkingLotEdges="on")

在停车场的前三条边上插入默认停车位。为了避免停车位重叠，对插入进行以下调整：

  • 沿着边缘1，只插入10个停车位。

  • 沿着边缘2，将停车位从该边的第一个顶点向后偏移5米。

  • 沿着边缘3，将停车位从该边的第一个顶点向后偏移3米。

numSpaces = 10;
insertParkingSpaces(lot,space,numSpaces,Edge=1)
insertParkingSpaces(lot,space,Edge=2,Offset=5)
insertParkingSpaces(lot,space,Edge=3,Offset=3)

在停车场的中心插入一个2x10的停车位网格，其中包含8列默认停车位、1列禁停区域和1列无障碍停车位。

insertParkingSpaces(lot,[space noSpace accessibleSpace],[8 1 1],Position=[12 -18],Rows=2)

输入参数

collapse all

space — 停车位
parkingSpace object

停车位，指定为一个 parkingSpace 对象。

visibility — 停车位原点的可见性
"on" (default) | "off"

停车位起点的可见性，指定为 "on" 和 "off"。起点位于停车位的左上角。

Data Types: char | string
