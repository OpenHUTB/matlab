targetOutlines

参与者所观察到的目标的轮廓

collapse all in page

语法

[position,yaw,length,width,originOffset,color] = targetOutlines(ac)
[position,yaw,length,width,originOffset,color,numBarrierSegments] = targetOutlines(ac,'Barriers')

描述

example

[position,yaw,length,width,originOffset,color] = targetOutlines(ac) 返回驾驶场景中所有非自车和非障碍物目标参与者的定向矩形轮廓。这些轮廓是从指定的自车参与者 ac 所看到的。有关更多详细信息，请参阅 Ego Vehicle and Targets 。

目标轮廓是目标参与者长方体在自车的局部坐标系的 (x,y) 平面上的投影。目标轮廓的组成部分包括 position, yaw, length, width, originOffset 和 color 的输出参数。

您可以将返回的轮廓作为输入参数传递给 birdsEyePlot 的轮廓绘制器(outline plotter)。首先，调用 outlinePlotter 函数创建绘制器对象。然后，使用 plotOutline 函数在鸟瞰图中绘制所有参与者的轮廓。

[position,yaw,length,width,originOffset,color,numBarrierSegments] = targetOutlines(ac,'Barriers') 返回驾驶场景中所有障碍物的定向矩形轮廓。额外的输出参数 numBarrierSegments 包含每个障碍物中的线段数量。

您可以将返回的障碍物轮廓作为输入参数传递给 birdsEyePlot 的轮廓绘制器(outline plotter)。首先，调用 outlinePlotter 函数创建绘制器对象。然后，使用 plotBarrierOutline 函数在鸟瞰图中绘制所有障碍物的轮廓。

例子

collapse all

在驾驶场景模拟中显示目标轮廓

Open Live Script

创建一个驾驶场景，并显示目标轮廓如何随着模拟的进展而变化。

创建一个由两条交叉的直路组成的驾驶场景。第一段路长45米。第二条笔直的道路长32米，两边都有泽西栅栏，与第一条道路相交。沿着第一条道路以每秒12.0米的速度行驶的汽车以每秒2.0米的速度接近穿过十字路口的奔跑的行人。

scenario = drivingScenario('SampleTime',0.1,'StopTime',1);
road1 = road(scenario,[-10 0 0; 45 -20 0]);
road2 = road(scenario,[-10 -10 0; 35 10 0]);
barrier(scenario,road1)
barrier(scenario,road1,'RoadEdge','left')
ped = actor(scenario,'ClassID',4,'Length',0.4,'Width',0.6,'Height',1.7);
car = vehicle(scenario,'ClassID',1);
pedspeed = 2.0;
carspeed = 12.0;
smoothTrajectory(ped,[15 -3 0; 15 3 0],pedspeed);
smoothTrajectory(car,[-10 -10 0; 35 10 0],carspeed);

为车辆创建一个以自我为中心的追逐情节。

chasePlot(car,'Centerline','on')

创建一个空的鸟瞰图，并添加轮廓绘图仪和车道边界绘图仪。然后，运行模拟。在每个模拟步骤中:

  • 更新追逐图以显示道路边界和目标轮廓。

  • 更新鸟瞰图以显示更新后的道路边界和目标轮廓。情节视角总是与自我载体有关。

bepPlot = birdsEyePlot('XLim',[-50 50],'YLim',[-40 40]);
outlineplotter = outlinePlotter(bepPlot);
laneplotter = laneBoundaryPlotter(bepPlot);
legend('off')

while advance(scenario)
    rb = roadBoundaries(car);
    [position,yaw,length,width,originOffset,color] = targetOutlines(car);
    [bposition,byaw,blength,bwidth,boriginOffset,bcolor,barrierSegments] = targetOutlines(car,'Barriers');
    plotLaneBoundary(laneplotter,rb)
    plotOutline(outlineplotter,position,yaw,length,width, ...
        'OriginOffset',originOffset,'Color',color)
    plotBarrierOutline(outlineplotter,barrierSegments,bposition,byaw,blength,bwidth, ...
        'OriginOffset',boriginOffset,'Color',bcolor)
    pause(0.01)
end

输入参数

collapse all

ac — Actor
Actor object | Vehicle object

属于 drivingScenario 对象的参与者，指定为 Actor 或 Vehicle 对象。要创建这些对象，分别使用 actor 和 vehicle 函数。

输出参数

collapse all

position — 目标的旋转中心
real-valued N-by-2 matrix

目标的旋转中心，作为实值N × 2矩阵返回。N是目标的数量。每行包含目标旋转中心的x坐标和y坐标。单位是米。

yaw — 目标的偏航角度
real-valued N-element vector

目标围绕旋转中心的偏航角，作为实值N元矢量返回。N是目标的数量。偏航角以逆时针方向测量，如上图所示。单位是度。

length — 目标矩形轮廓的长度
positive, real-valued N-element vector

目标矩形轮廓的长度，作为一个正的实值N元素向量返回。N是目标的数量。单位是米。

width — 目标矩形轮廓的宽度
positive, real-valued N-element vector

目标矩形轮廓的宽度，作为一个正的实值N元素向量返回。N是目标的数量。单位是米。

originOffset — 旋转中心相对于几何中心的偏移量
real-valued N-by-2 matrix

目标的旋转中心与其几何中心的偏移量，作为实值N × 2矩阵返回。N是目标的数量。每行包含定义此偏移量的 x 坐标和 y 坐标。在车辆目标中，旋转中心或原点位于地面上，直接位于后轴中心下方。单位是米。

color — 目标颜色的RGB表示
nonnegative, real-valued N-by-3 matrix

目标颜色的RGB表示，作为非负的实值N × 3矩阵返回。N是目标参与者的数量。

numBarrierSegments — 每个障碍物中的障碍段数量
nonnegative, real-valued N-by-1 vector

每个障碍物中的障碍物段数，以非负的实值 N×1 向量的形式返回。N 是场景中障碍物的数量。只有在输入时提供了 'Barriers' 标志时，才会返回此参数。

自我车辆和目标

在驾驶场景中，您可以指定一个参与者作为所有其他参与者的观察者，类似于汽车的驾驶员如何观察所有其他汽车。观察者行为人被称为自我行为人，或者更具体地说，是自我载体。从自我载体的角度来看，所有其他参与者(如车辆和行人)都是被观察的参与者，称为目标。自我车辆坐标以自我车辆为中心和方向。驾驶场景的坐标为世界坐标。
