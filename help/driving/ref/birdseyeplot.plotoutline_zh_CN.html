plotOutline

在鸟瞰图上显示物体轮廓

collapse all in page

语法

plotOutline(olPlotter,positions,yaw,length,width)
plotOutline(___,Name,Value)

描述

plotOutline(olPlotter,positions,yaw,length,width) 在鸟瞰图上显示立方体物体的矩形轮廓。指定每个立方体的位置、偏航角度、长度和宽度。轮廓绘图器（olPlotter）与 birdsEyePlot 对象相关联，并配置指定轮廓的显示方式。

要删除与轮廓绘图器 olPlotter 相关联的所有轮廓，请调用 clearData 函数，并将 olPlotter 指定为输入参数。

要显示驾驶场景中的车辆的轮廓，首先使用targetOutlines 获取车辆的尺寸。然后，在调用outlinePlotter创建绘图器对象之后，使用 plotOutline 函数在鸟瞰图中显示所有车辆的轮廓，除了障碍物。使用 plotBarrierOutline 函数来显示障碍物的轮廓。

example

plotOutline(___,Name,Value) 使用一个或多个 Name,Value 对参数以及前面语法中的输入参数来指定选项。

例子

collapse all

在鸟瞰图上绘制目标的轮廓

Open Live Script

创建一个驾驶场景。创建一个长度为25米的道路段，其左边缘有一个障碍物。添加一个以 1 m/s 的速度横穿道路的行人，以及一个以10 m/s 的速度沿着道路行驶的车辆。

scenario = drivingScenario;

r = road(scenario,[0 0 0; 25 0 0]);
barrier(scenario,r,'RoadEdge','left')

p = actor(scenario,'ClassID',4,'Length',0.2,'Width',0.4,'Height',1.7);
v = vehicle(scenario,'ClassID',1);

smoothTrajectory(p,[15 -3 0; 15 3 0],1)
smoothTrajectory(v,[v.RearOverhang 0 0; 25-v.Length+v.RearOverhang 0 0],10)

使用追踪绘图来从车辆的视角显示该场景。

chasePlot(v,'Centerline','on')

创建一个鸟瞰图、轮廓绘图器和车道边界绘图器。

bep = birdsEyePlot('XLim',[-25 25],'YLim',[-10 10]);
olPlotter = outlinePlotter(bep);
lbPlotter = laneBoundaryPlotter(bep);
legend('off')

运行仿真循环。使用目标的轮廓更新绘图器。

while advance(scenario)
   % Obtain the road boundaries and rectangular outlines.
   rb = roadBoundaries(v);
   [position,yaw,length,width,originOffset,color] = targetOutlines(v);
   [bposition,byaw,blength,bwidth,boriginOffset,bcolor,numBarrierSegments] = targetOutlines(v,'Barriers');

   % Update the bird's-eye plotters with the road, actors and barriers.
   plotLaneBoundary(lbPlotter,rb);
   plotOutline(olPlotter,position,yaw,length,width, ...
               'OriginOffset',originOffset,'Color',color);
   plotBarrierOutline(olPlotter,numBarrierSegments,bposition,byaw,blength,bwidth, ...
               'OriginOffset',boriginOffset,'Color',bcolor); 
   % Allow time for plot to update.
   pause(0.01)
end

输入参数

collapse all

olPlotter — Outline plotter
OutlinePlotter object

轮廓绘图器是一个 OutlinePlotter 对象，存储在 birdsEyePlot 对象的 Plotters 属性中，用于配置在鸟瞰图中显示指定轮廓的方式。要创建该对象，请使用 outlinePlotter 函数。

positions — Positions of detected objects
M-by-2 real-valued matrix

在车辆坐标系中指定的检测到的物体位置，表示为一个大小为M×2的实值矩阵，其中包含了 (X, Y) 位置信息。M 是检测到的物体数量。正X方向指向车辆中心的前方。正Y方向指向车辆原点的左侧，车辆原点是后轴的中心，如下图所示的车辆坐标系。

yaw — Angles of rotation
M-element real-valued vector

对象轮廓的旋转角度，指定为一个M元素的实数向量，其中M是对象的数量。

length — Lengths of outlines
M-element real-valued vector

对象轮廓的长度，指定为一个MM元素的实数向量，其中M是对象的数量。

width — Widths of outlines
M-element real-valued vector

对象轮廓的宽度，指定为一个M元素的实数向量，其中M是对象的数量。

Name-Value Arguments

将可选的参数对指定为 Name1=Value1,...,NameN=ValueN 的形式，其中 Name 是参数名称，Value 是相应的值。Name-value 参数必须出现在其他参数之后，但是参数对的顺序无关紧要。

在R2021a之前，使用逗号将每个名称和值分隔开，并用引号括住 Name。

Example: 'Marker','x'

OriginOffset — Rotational centers of rectangles relative to origin
M-by-2 real-valued matrix

相对于原点的矩形的旋转中心，指定为由'OriginOffset'和一个M×2的实数矩阵组成的逗号分隔的参数对。M是对象的数量。每一行对应于要围绕其旋转相应矩形的旋转中心，指定为从该矩形的几何中心的位移(X,Y) 。

Color — Outline color
M-by-3 matrix of RGB triplets

轮廓颜色，指定为由'Color' 和一个M×3的RGB三元组矩阵组成的逗号分隔的参数对。M是对象的数量。如果您不指定此参数，函数将对每个对象使用默认的颜色映射。

Example: 'Color',[0 0.5 0.75; 0.8 0.3 0.1]
