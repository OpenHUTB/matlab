Simulink.ScenarioLog

查看任意时间步的场景仿真结果

    描述

    Simulink.ScenarioLog 对象存储仿真所有时间步骤的场景仿真结果。使用 Simulink.ScenarioLog 对象可以：

    • 在任何时间步骤查看参与者信息，例如运行时属性和操作。

    • 查看任意时间步的仿真信息。

    • 查看仿真过程中报告的诊断信息。

    • 在任意时间步查看用户定义的操作参数的值。

    创建

    使用 Simulink.ScenarioLog 对象的 get 函数检索代表仿真日志的 Simulink.ScenarioSimulation 对象。仅当使用 Simulink.ScenarioSimulation 对象的 set 函数启用了日志记录时，才能获取仿真的日志。只有在仿真停止播放后才能获取仿真日志。

    对象函数

    get 获取记录场景的仿真数据

    示例

    获取仿真日志

    创建一个 Simulink.ScenarioSimulation 对象 Sim。

    rrApp = roadrunner('C:\RoadRunnerProject');
openScenario(rrApp,'TrajectoryCutIn');
Sim = createSimulation(rrApp);

    启用仿真日志记录。

    set(Sim,'Logging','On');

    开始仿真。

    set(Sim,'SimulationCommand', 'Start');

    仿真停止播放后获取仿真日志。

    simLog = get(Sim, 'SimulationLog');

    获取第一个和最后一个时间步参与者的姿态

    从仿真日志中获取所有参与者的 ID。

    actorIDs = get(simLog, 'ActorIDs')
    actorIDs =

     1     2     3

    在仿真运行的所有时间步骤中获取 ID 为 1 的仿真的姿态。

    poseActor1 = get(simLog,'Pose','ActorID',1)
    poseActor1 = 

  1×887 struct array with fields:

    Time
    Pose

    在仿真运行的第一个时间步获取参与者 ID 为 1 的姿态。

    poseActor1FirstTimeStep = poseActor1(1)
