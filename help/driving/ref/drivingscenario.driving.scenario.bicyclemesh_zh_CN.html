driving.scenario.bicycleMesh

驾驶场景中自行车的网格表示

collapse all in page

语法

mesh = driving.scenario.bicycleMesh

描述

example

mesh = driving.scenario.bicycleMesh 创建了一个自行车的网格表示，作为一个 extendedObjectMesh 对象，命名为 mesh。

例子

collapse all

使用自行车网格生成激光雷达点云

Open Live Script

将预建的自行车网格添加到驾驶场景中。然后，使用 lidarPointCloudGenerator 系统对象来生成自行车网格的点云。

创建并显示预建的自行车网格。

mesh = driving.scenario.bicycleMesh;
egoMesh = driving.scenario.carMesh;
figure
show(mesh)

ans = 
  Axes with properties:

             XLim: [-0.5000 0.5000]
             YLim: [-0.2000 0.2000]
           XScale: 'linear'
           YScale: 'linear'
    GridLineStyle: '-'
         Position: [0.1300 0.1100 0.7750 0.8150]
            Units: 'normalized'

  Show all properties

创建一个驾驶场景。

s = drivingScenario;

在驾驶场景中添加一条直路。该道路有一条车道，分别向两个方向行驶。

road(s,[0 0 0; 30 0 0],'Lanes',lanespec([1 1]));

在驾驶场景中添加一辆汽车作为自车，并添加一辆自行车作为非自车参与者。

egoVehicle = vehicle(s,'ClassID',1,'Mesh',egoMesh);
bicycle = vehicle(s,'Position',[15 2 0],'Yaw',180,'ClassID',3,'Mesh',mesh);
smoothTrajectory(egoVehicle,[1 -2 0; 21.3 -2 0],20);

绘制驾驶场景。设置名称-值对 'Meshes','on' 以显示绘图中参与者的网格。

figure;
plot(s,'Meshes','on');

创建一个 lidarPointCloudGenerator System对象。将该System对象的参与者配置设置为驾驶场景中的参与者配置。

lidar = lidarPointCloudGenerator;
lidar.ActorProfiles = actorProfiles(s);

生成驾驶场景的激光雷达点云。

player = pcplayer([-20 20],[-10 10],[0 4]);
while advance(s)
 tgts = targetPoses(egoVehicle);
 rdmesh = roadMesh(egoVehicle);
 [ptCloud,isValidTime] = lidar(tgts,rdmesh,s.SimulationTime);
    if isValidTime
    view(player,ptCloud);
    end
end

输出参数

collapse all

mesh — Mesh representation of bicycle
extendedObjectMesh object

自行车的网格表示，以 extendedObjectMesh 对象的形式返回。网格的原点位于其几何中心。

您可以使用这个预建的自行车网格作为起点，开发自己的网格。在MATLAB®命令行中输入：

edit driving.scenario.bicycleMesh
