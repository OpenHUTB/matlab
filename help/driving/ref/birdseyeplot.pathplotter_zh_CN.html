pathPlotter

用于鸟瞰图的路径绘制器

collapse all in page

语法

pPlotter = pathPlotter(bep)
pPlotter = pathPlotter(bep,Name,Value)

描述

example

pPlotter = pathPlotter(bep) 创建一个 PathPlotter 对象，用于在鸟瞰图上配置参与者路径的显示。PathPlotter 对象存储在输入的 birdsEyePlot 对象（bep）的 Plotters 属性中。要显示路径，请使用 plotPath 函数。

pPlotter = pathPlotter(bep,Name,Value) 使用一个或多个 Name,Value 对参数来设置属性。例如，pathPlotter(bep,'DisplayName','Actor paths') 设置在鸟瞰图图例中显示的名称。

例子

collapse all

绘制自车路径

Open Live Script

创建一个宽度为3米的车道。

lb = parabolicLaneBoundary([-0.001,0.01,1.5]);
rb = parabolicLaneBoundary([-0.001,0.01,-1.5]);

手动计算沿着 x 轴从 0 到 30 米的车道边界模型。

xWorld = (0:30)';
yLeft = computeBoundaryModel(lb,xWorld);
yRight = computeBoundaryModel(rb,xWorld);

创建一个鸟瞰图和车道边界绘图器。在鸟瞰图上显示车道信息。

bep = birdsEyePlot('XLimits',[0 30],'YLimits',[-5 5]);
lanePlotter = laneBoundaryPlotter(bep,'DisplayName','Lane boundaries');
plotLaneBoundary(lanePlotter,{[xWorld,yLeft],[xWorld,yRight]});

创建一个路径绘图器。创建并显示一辆自车沿着车道中心行驶的路径。

yCenter = (yLeft + yRight)/2;
egoPathPlotter = pathPlotter(bep,'DisplayName','Ego vehicle path');
plotPath(egoPathPlotter,{[xWorld,yCenter]});

输入参数

collapse all

bep — Bird’s-eye plot
birdsEyePlot object

Bird’s-eye plot, specified as a birdsEyePlot object.

Name-Value Arguments

将可选的参数对指定为 Name1=Value1,...,NameN=ValueN，其中 Name 是参数名称，Value是相应的值。Name-value参数必须出现在其他参数之后，但对参数对的顺序没有要求。

在R2021a之前，使用逗号将每个名称和值分隔开，并用引号将 Name 括起来。

Example: pathPlotter('Color','red') 将路径的颜色设置为红色。

DisplayName — Plotter name to display in legend
'' (default) | character vector | string scalar

要在图例中显示的绘图器名称，指定为由 'DisplayName' 和字符向量或字符串标量组成的逗号分隔对。如果您不指定名称，则鸟瞰图不会为绘图器显示图例条目。

Color — Path color
[0 0 0] (black) (default) | RGB triplet | hexadecimal color code | color name | short color name

路径颜色，指定为由 'Color' 和 RGB 三元组、十六进制颜色代码、颜色名称或短颜色名称组成的逗号分隔对。

对于自定义颜色，可以指定一个RGB三元组或一个十六进制颜色代码。

  • RGB三元组是一个包含三个元素的行向量，这些元素指定颜色的红色、绿色和蓝色分量的强度。强度必须在 [0,1] 范围内，例如，[0.4 0.6 0.7]。

  • 十六进制颜色代码是以井号 (#) 开头的字符向量或字符串标量，后跟三个或六个十六进制数字，范围从 0 到 F。这些值不区分大小写。因此，颜色代码"#FF8800", "#ff8800", "#F80" 和 "#f80" 是等效的。

另外，您可以通过名称指定一些常见的颜色。下表列出了命名颜色选项、对应的RGB三元组和十六进制颜色代码。

LineStyle — Path line style
':' (default) | '-' | '--' | '-.' | 'none'

路径线条样式，指定为逗号分隔的一对，包括'LineStyle'和该表中列出的选项之一。

Line Style Description Resulting Line
"-" Solid line

"--" Dashed line

":" Dotted line

"-." Dash-dotted line

"none" No line No line

Tag — Tag associated with plotter object
'PlotterN' (default) | character vector | string scalar

与绘图对象关联的标签，指定为逗号分隔的一对，包括 'Tag' 和字符向量或字符串标量。默认值为 'PlotterN'，其中 N 是与输入 birdsEyePlot 对象关联的第 N 个绘图对象的整数对应的值。

输出参数

collapse all

pPlotter — Path plotter
PathPlotter object

路径绘图器作为 PathPlotter 对象返回。您可以通过更改其属性值来修改此对象。属性名称对应于 pathPlotter 函数的名称-值对参数。

pPlotter 存储在输入的 birdsEyePlot 对象（bep）的 Plotters 属性中。要绘制路径，请使用 plotPath 函数。
