laneBoundaries

获取参与者车道的车道边界

collapse all in page

语法

lbdry = laneBoundaries(ac)
lbdry = laneBoundaries(ac,Name,Value)

描述

lbdry = laneBoundaries(ac) 返回自车参与者（ac）所行驶的车道的车道边界（lbdry）。车道边界以自车坐标系表示。

example

lbdry = laneBoundaries(ac,Name,Value) 使用一个或多个名称-值对指定选项。例如，laneBoundaries(ac,'AllLaneBoundaries',true) 返回自车参与者所行驶道路的所有车道边界。

例子

collapse all

仿真汽车在"S"形曲线上行驶

Open Script

仿真一个驾驶场景，其中一辆汽车在"S"形曲线上行驶。创建并绘制车道边界。

创建一个具有"S"形曲线的驾驶场景，其中包含一条道路。

scenario = drivingScenario('StopTime',3);
roadcenters = [-35 20 0; -20 -20 0; 0 0 0; 20 20 0; 35 -20 0];

创建车道并将它们添加到道路中。

lm = [laneMarking('Solid','Color','w'); ...
    laneMarking('Dashed','Color','y'); ...
    laneMarking('Dashed','Color','y'); ...
    laneMarking('Solid','Color','w')];
ls = lanespec(3,'Marking',lm);
road(scenario,roadcenters,'Lanes',ls);

添加一个自车，并根据其路径点指定其轨迹。默认情况下，汽车以每秒30米的速度行驶。

car = vehicle(scenario, ...
    'ClassID',1, ...
    'Position',[-35 20 0]);
waypoints = [-35 20 0; -20 -20 0; 0 0 0; 20 20 0; 35 -20 0];
smoothTrajectory(car,waypoints);

绘制场景和相应的追踪图。

plot(scenario)


chasePlot(car)

运行仿真循环。

  1. 初始化鸟瞰图并创建一个轮廓绘制器、左车道和右车道边界绘制器以及一个道路边界绘制器。

  2. 获取道路边界和矩形轮廓。

  3. 获取车辆左右两侧的车道边界。

  4. 推进仿真并更新绘图器。

bep = birdsEyePlot('XLim',[-40 40],'YLim',[-30 30]);
olPlotter = outlinePlotter(bep);
lblPlotter = laneBoundaryPlotter(bep,'Color','r','LineStyle','-');
lbrPlotter = laneBoundaryPlotter(bep,'Color','g','LineStyle','-');
rbsEdgePlotter = laneBoundaryPlotter(bep);
legend('off');
while advance(scenario)
    rbs = roadBoundaries(car);
    [position,yaw,length,width,originOffset,color] = targetOutlines(car);
    lb = laneBoundaries(car,'XDistance',0:5:30,'LocationType','Center', ...
        'AllBoundaries',false);
    plotLaneBoundary(rbsEdgePlotter,rbs)
    plotLaneBoundary(lblPlotter,{lb(1).Coordinates})
    plotLaneBoundary(lbrPlotter,{lb(2).Coordinates})
    plotOutline(olPlotter,position,yaw,length,width, ...
        'OriginOffset',originOffset,'Color',color)
end

输入参数

collapse all

ac — Actor
Actor object | Vehicle object

属于 drivingScenario 对象的参与者，可以指定为 Actor 或 Vehicle 对象。要创建这些对象，请分别使用 actor 和 vehicle 函数。

名称-值参数

将可选的参数对指定为 Name1=Value1,...,NameN=ValueN 的形式，其中 Name 是参数名称，Value 是相应的值。Name-value参数必须出现在其他参数之后，但是参数对的顺序无关紧要。

在R2021a之前，使用逗号将每个名称和值分隔开，并将 Name 用引号括起来。

Example: 'LocationType','center' 指定车道边界以车道标线为中心。

XDistance — 计算车道边界时与自车的距离
0 (default) | N-element real-valued vector

计算车道边界时与自车的距离，指定为逗号分隔的一对参数，包括 'XDistance' 和一个N元素的实数向量。N是距离值的数量。当从朝向后方的摄像头检测车道时，请指定负距离。当从朝向前方的摄像头检测车道时，请指定正距离。单位为米。

默认情况下，该函数计算与自车距离为 0 的车道边界，即自车原点左右的边界。

Example: 1:0.1:10 在自车前方1到10米的范围内每0.1米计算一个车道边界。

Example: linspace(-150,150,101) 在自车后方150米到自车前方150米的范围内计算101个车道边界。这些距离之间相隔3米，呈线性间隔。

LocationType — 车道边界位置
'Center' (default) | 'Inner'

车道边界位置在车道标线上，由 'LocationType' 和表中选项之一组成的逗号分隔的参数对来指定。

Lane Boundary Location Description Example
'Center' 车道边界位于车道标线的中心。

一条三车道道路有四个车道边界：每个车道标线一个。
'Inner' 车道边界放置在车道标线的内侧边缘。

一条三车道道路有六个车道边界：每个车道有两个。

AllBoundaries — 返回道路上的所有车道边界
false (default) | true

返回自车所行驶的所有车道边界，由 'Value' 和 false 或 true 组成的逗号分隔的参数对来指定。

车道边界相对于自车从左到右返回。当 'AllBoundaries' 为 false 时，只返回自车左右两侧的车道边界。

输出参数

collapse all

lbdry — 车道边界
array of lane boundary structures

车道边界以车道边界结构的数组形式返回。以下表格显示了每个结构的字段。

Field Description

Coordinates

车道边界坐标，用实值N × 3矩阵表示，其中N为车道边界坐标的个数。车道边界坐标定义了沿道路中心距离车辆指定纵向距离的边界上的点的位置。

  • 在MATLAB®中，通过使用 laneBoundaries 函数的'XDistance' 名称-值对参数来指定这些距离。

  • 在Simulink®中，可以通过使用场景读取器块的从自车计算边界的距离（单位：米）参数或仿真 3D 视觉检测生成器块的用于计算车道边界的父级距离参数来指定这些距离。

该矩阵还包括距离自车零距离处的边界坐标。这些坐标位于自车原点的左侧和右侧，自车原点位于后轴的中心下方。单位为米。

Curvature

 在 Coordinates 矩阵的每行处指定的车道边界曲率，表示为实值的N×1向量。N是车道边界坐标的数量。单位为弧度每米。

CurvatureDerivative

 在 Coordinates 矩阵的每行处指定的车道边界曲率的导数，表示为实值的 N×1 向量。N 是车道边界坐标的数量。单位为弧度每平方米。

HeadingAngle

 初始车道边界航向角，指定为实数标量。车道边界的航向角是相对于自车航向的角度。单位为度。

LateralOffset

自车位置与车道边界的横向偏移量，指定为实数标量。自车左侧的车道边界偏移量为正值，自车右侧的车道边界偏移量为负值。单位为米。在这张图像中，自车与左车道的偏移量为1.5米，与右车道的偏移量为2.1米。

BoundaryType

车道边界标记的类型，指定为以下值之一：

  • 'Unmarked' — 不存在物理车道标线

  • 'Solid' — 单一连续线

  • 'Dashed' — 由虚线组成的单一车道标线

  • 'DoubleSolid' — 两条连续线

  • 'DoubleDashed' — 两条虚线

  • 'SolidDashed' — 左侧为实线，右侧为虚线

  • 'DashedSolid' — 左侧为虚线，右侧为实线

Strength

车道边界标记的饱和度强度，指定为介于0到1之间的实数标量。值为 0 表示标记的颜色完全不饱和，呈现为灰色。值为 1 表示标记的颜色完全饱和。

Width

车道边界的宽度，指定为正实数标量。在双线车道标线中，两条线和线之间的间隔使用相同的宽度。单位为米。

Length

虚线中虚线的长度，指定为正实数标量。在双线车道标线中，两条线使用相同的长度。

Space

虚线中虚线之间的间隔长度，指定为正实数标量。在双线虚线车道标线中，两条线之间使用相同的间隔。
