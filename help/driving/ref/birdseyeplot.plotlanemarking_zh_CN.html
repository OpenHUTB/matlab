plotLaneMarking

在鸟瞰图上显示车道标线

collapse all in page

语法

plotLaneMarking(lmPlotter,lmv,lmf)

描述

example

plotLaneMarking(lmPlotter,lmv,lmf) 在鸟瞰图上显示车道标线顶点（lmv）和车道标线面（lmf）。车道标线绘图器（lmPlotter）与 birdsEyePlot 对象相关联，并配置显示指定的车道标线。

要删除与车道标线绘图器（lmPlotter）相关联的所有车道标线，请调用 clearData 函数并将 lmPlotter 指定为输入参数。

例子

collapse all

在汽车和行人场景中显示车道标线

Open Script

创建一个包含汽车和行人的直线道路驾驶场景。然后，在鸟瞰图上创建并显示道路的车道标线。

创建一个空的驾驶场景。

scenario = drivingScenario;

创建一个直线的25米道路段，其中有两个单行道车道。

lm = [laneMarking('Solid')
      laneMarking('Dashed','Length',2,'Space',4)
      laneMarking('Solid')];
l = lanespec(2,'Marking',lm);
road(scenario,[0 0 0; 25 0 0],'Lanes',l);

向驾驶场景中添加一个以每秒1米的速度穿过道路的行人，以及一个以每秒10米的速度沿着道路行驶的汽车。

ped = actor(scenario,'ClassID',4,'Length',0.2,'Width',0.4,'Height',1.7);
car = vehicle(scenario,'ClassID',1);
smoothTrajectory(ped,[15 -3 0; 15 3 0],1);
smoothTrajectory(car,[car.RearOverhang 0 0; 25-car.Length+car.RearOverhang 0 0],10);

显示场景和相应的追踪图。

plot(scenario)


chasePlot(car)

运行仿真。

  1. 创建一个鸟瞰图。

  2. 创建轮廓绘制器、车道边界绘制器和车道标线绘制器，用于鸟瞰图。

  3. 获取道路边界和目标轮廓。

  4. 获取车道标线的顶点和面。

  5. 显示车道边界和车道标线。

  6. 运行仿真循环。

bep = birdsEyePlot('XLim',[-25 25],'YLim',[-10 10]);
olPlotter = outlinePlotter(bep);
lbPlotter = laneBoundaryPlotter(bep);
lmPlotter = laneMarkingPlotter(bep,'DisplayName','Lanes');
legend('off');
while advance(scenario)
    rb = roadBoundaries(car);
    [position,yaw,length,width,originOffset,color] = targetOutlines(car);
    [lmv,lmf] = laneMarkingVertices(car);
    plotLaneBoundary(lbPlotter,rb);
    plotLaneMarking(lmPlotter,lmv,lmf);
    plotOutline(olPlotter,position,yaw,length,width, ...
        'OriginOffset',originOffset,'Color',color);
end

输入参数

collapse all

lmPlotter — Lane marking plotter
LaneMarkingPlotter object

车道标线绘制器是一个 LaneMarkingPlotter 对象，存储在 birdsEyePlot 的 Plotters 属性中，用于配置在鸟瞰图中显示指定车道标线的方式。要创建这个对象，请使用 laneMarkingPlotter 函数。

lmv — Lane marking vertices
V-by-3 real-valued matrix

车道标线顶点是一个V行3列的实数矩阵，其中 lmv 的每一行表示车道标线中一个顶点的x、y和 z 坐标。绘制器只使用x和y坐标。V是标线中顶点的数量。

要从驾驶场景中获取车道标线的顶点和面，可以使用 laneMarkingVertices 函数。

lmf — Lane marking faces
integer-valued matrix

车道标线面是一个整数值矩阵，每一行 lmf 都是一个面，定义了一个车道标线的顶点之间的连接关系。

要从驾驶场景中获取车道标线的顶点和面，可以使用 laneMarkingVertices 函数。
