遵循椭圆形轨道周围的路点

Open Example

此示例仿真一辆 3 自由度 (degree-of-freedom, DOF) 车辆绕指定路点的椭圆形轨道行驶。该模型包含航点并使用 MATLAB® 函数来确定下一个航向航点。

要创建自己的轨迹并在虚幻引擎中使用它，可以使用 RoadRunner 和 RoadRunner 插件。要在虚幻引擎中仿真赛道上的车辆，需要 Unreal Engine® 4 Projects 支持包中的 Vehicle Dynamics Blockset™ 接口。 有关详细信息，请参阅 Create and Use an Oval Track。

路点

模型工作区包含一个变量，TrackPoints，它指定 X, Y, 和椭圆形赛道（比如 Indy 500® 赛道）的参考姿态路点。这些点是 Z -向下 车辆坐标系中的 X 和 Y 的位置，单位为米。参考位姿以弧度为单位。

运行仿真

在“仿真”（Simulation）选项卡上，单击运行“Run”。仿真运行时，车辆位置（Vehicle Position）窗口会提供车辆绕轨道移动时的轨迹。

椭圆形轨道参考

椭圆轨道参考子系统（Oval Track Reference subsystem）使用 TrackPoints 路点来生成参考路径。该子系统还包括一个 MATLAB® Function 模块，可根据当前车辆位置和姿态确定下一个航向航路点。然后，参考模块向驾驶员模块提供车辆命令。
