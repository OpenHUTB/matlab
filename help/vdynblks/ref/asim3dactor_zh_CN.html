ASim3dActor

用作用户定义的虚幻引擎 C++ 或蓝图参与者基类的抽象类

expand all in page

描述

ASim3dActor 是一个抽象类，您可以将其用作用户定义的虚幻引擎® C++ 或蓝图参与者的基类。

在与 Simulink® 模型进行联合仿真期间，基类本质上是同步的。此外，Simulation 3D Actor Transform Set 模块可以控制基类。要扩展 ASim3dActor 的行为 ，您可以使用消息接口函数来重写类方法，以便它们向模型发送消息和从模型接收消息。

ASim3dActor 包含在虚幻引擎 4 项目的 Vehicle Dynamics Blockset™ Interface for Unreal Engine 4 Projects 中。有关支持包的信息，请参阅 Customize 3D Scenes for Vehicle Dynamics Simulations.

属性

expand all

Translation — 参与者平移
1×3 (默认) | 每个参与者的部件数量×3

该属性是受保护（protected）的。它用在派生的 C++ 类中。值由 Simulation 3D Actor Transform Set 模块进行设置。

参与者分别沿世界 X-轴、Y轴和 Z- 轴进行平移。以米为单位。数组维度是每个参与者的部件数×3。

数据类型：float

Rotation — 参与者旋转
1×3 (默认) | 每个参与者的部件数目×3

该属性是受保护（protected）。它用在派生的 C++ 类中。值由 Simulation 3D Actor Transform Set 模块进行设置。

参与者分别绕世界X轴、Y轴和Z轴在 [-pi/2, pi/2] 范围内旋转（以弧度为单位）。数组维度是每个参与者的部件数目×3。

数据类型：float

Scale — 参与者缩放比例
1×3 (默认) | 每个参与者部件数量×3

该属性是受保护（protected）。它用在派生的 C++ 类中。值由 Simulation 3D Actor Transform Set 模块进行设置。

参与者缩放比例。 数组维度是每个参与者的部件数目×3。

数据类型：float

对象函数

Sim3dSetup 在虚幻引擎三维仿真中设置参与者的 C++ 方法
Sim3dStep 在虚幻引擎三维仿真中步进参与者的 C++ 方法
Sim3dRelease 虚幻引擎三维仿真中释放参与者的 C++ 方法

版本历史

在 R2020b 中引入
