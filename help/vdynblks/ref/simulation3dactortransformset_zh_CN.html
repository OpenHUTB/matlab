Simulation 3D Actor Transform Set

设置参与者平移、旋转、缩放

expand all in page
  • 库：

  • Vehicle Dynamics Blockset / Vehicle Scenarios / Sim3D / Sim3D Core

    Aerospace Blockset / Animation / Simulation 3D

    Simulink 3D Animation / Simulation 3D

描述

The Simulation 3D Actor Transform Set 模块设置三维可视化环境中参与者的平移、旋转和缩放。

该模块使用最初与惯性世界坐标系对齐的车辆固定坐标系。

坐标轴 描述
X

车辆前进方向

翻滚 — 绕 X 轴右手旋转
Y

延伸到车辆右侧，最初与地平面平行

俯仰 — 绕 Y 轴右手旋转
Z

向上延伸

偏航 — 绕 Z 轴左手旋转

参与者是支持三维平移、旋转和缩放的场景对象。部件是参与者组件。组件本身并不存在；他们与一个参与者有联系。

提示

验证 Simulation 3D Actor Transform Set 模块是否在 Simulation 3D Scene Configuration 模块之前执行。这样，Simulation 3D Actor Transform Set 在虚幻引擎 ® 三维可视化环境接收信号数据之前准备好信号数据。要检查块执行顺序，请右键单击模块并选择“属性（Properties）”。在“常规（General）”选项卡上，确认以下“优先级（Priority）”设置：

  • Simulation 3D Scene Configuration — 0

  • Simulation 3D Actor Transform Set — -1

有关执行顺序的更多信息，请参阅 控制和显示执行顺序。

Translation — 参与者平移
数组

参与者平移，单位为米。数组维度是每个参与者的部件数量 ×3。

  • Translation(1,1)、Translation(1,2) 和 Translation(1,3) — 车辆分别沿世界X轴、Y轴和Z轴的位移。

  • Translation(...,1)、Translation(...,2)、和 Translation(...,3) — 在最初分别与世界X轴、 Y轴和Z轴对齐的车辆固定坐标系中， 参与者相对于车辆的位移。

例如，考虑一个具有车身和四个轮子的车辆参与者。信号 Translation：

  • 维度为 [5x3].

  • 包含根据相对于车辆车轴和车轮位置的平移信息。

Rotation — 参与者旋转
数组

参与者在 [-pi/2, pi/2] 范围内旋转（以弧度为单位）。数组维度是每个参与者的部件数量×3。

  • Rotation(1,1)、Rotation(1,2) 和 Rotation(1,3) — 车辆分别绕车辆固定的俯仰(Pitch)、翻滚(Roll)和偏航(Yaw) Y、Z和X轴旋转。

  • Rotation(...,1)、 Rotation(...,2)、和 Rotation(...,3) — 参与者分别绕车辆固定的俯仰、翻滚和偏航Y、Z和X轴旋转。

参数

expand all

参与者设置

Tag for actor in 3D scene, ActorTag — 名称
SimulinkActor1 (默认) | 字符向量

参与者名称。

参与者是支持三维 平移、旋转和缩放的场景对象。部件是参与者组件。组件本身并不存在；他们与一个参与者有联系。

该模块不支持同一参与者标签的多个实例。要在使用三维模块对（例如， Simulation 3D Actor Transform Get and Simulation 3D Actor Transform Set) 时引用相同的场景参与者，指定相同的 Tag for actor in 3D scene, ActorTag 参数。

Number of parts per actor to set, NumberOfParts — 名称
1 (默认) | scalar

Number of parts per actor. Actors are scene objects that support 3D translation, rotation, and scale. Parts are actor components. Components do not exist by themselves; they are associated with an actor. Typically, a vehicle actor with a body and four wheels has 5 parts.

The block does not support multiple instances of the same actor tag. To refer to the same scene actor when you use the 3D block pairs (e.g. Simulation 3D Actor Transform Get and Simulation 3D Actor Transform Set), specify the same Tag for actor in 3D scene, ActorTag parameter.

Initial Values

Initial array values to translate actor per part, Translation — 参与者初始位置
[0 0 0] (default) | array

Actor initial position, along world X-, Y-, and Z- axes, in m.

Array dimensions are number of parts per actor-by-3.

  • Translation(1,1), Translation(1,2), and Translation(1,3) — Vehicle displacement along world X-, Y, and Z- axes, respectively.

  • Translation(...,1), Translation(...,2), and Translation(...,3) — Actor displacement relative to vehicle, in vehicle-fixed coordinate system initially aligned with world X-, Y, and Z- axes, respectively.

For example, consider a vehicle actor with a vehicle body and four wheels. The parameter:

  • Dimensions are [5x3].

  • Contains translation information according to the axle and wheel locations, relative to vehicle.
