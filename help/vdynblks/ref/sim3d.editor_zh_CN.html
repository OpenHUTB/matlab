sim3d.Editor

虚幻引擎项目的接口

expand all in page

描述

使用类 sim3d.Editor 与虚幻® 编辑器交互。

要使用虚幻编辑器开发场景并与 Simulink®进行联合仿真，您需要 Vehicle Dynamics Blockset Interface for Unreal Engine 4 Projects 支持包。支持包包含一个虚幻引擎项目，允许您自定义 Vehicle Dynamics Blockset™ 场景。有关支持包的信息，请参阅 Customize 3D Scenes for Vehicle Dynamics Simulations.

创建

语法

sim3d.Editor(project)

描述

示例

MATLAB 为  sim3d.Editor(project) 中指定的虚幻编辑器工程创建一个 sim3d.Editor 对象。

输入参数

expand all

project — 项目路径和名称
字符串数

项目路径和名称

示例："C:\Local\AutoVrtlEnv\AutoVrtlEnv.uproject"

数据类型：string

属性

expand all

Uproject — 项目路径和名称
字符串数组

该属性是只读的。

带虚幻引擎工程文件扩展名（.uproject）的工程路径和名称。

示例："C:\Local\AutoVrtlEnv\AutoVrtlEnv.uproject"

数据类型：string

目标函数

open 打开虚幻编辑器

示例

collapse all

在虚幻编辑器中打开项目

在虚幻编辑器中打开虚幻引擎项目。

为位于 C:\Local\AutoVrtlEnv\AutoVrtlEnv.uproject  中的虚幻引擎项目创建 sim3d.Editor 类的实例。

editor = sim3d.Editor(fullfile("C:\Local\AutoVrtlEnv\AutoVrtlEnv.uproject"))

在虚幻编辑器中打开项目。

editor.open();
