Simulation 3D Message Get

从虚幻引擎可视化环境中检索数据

expand all in page
  • 库:

  • Vehicle Dynamics Blockset / Vehicle Scenarios / Sim3D / Sim3D Core

    Aerospace Blockset / Animation / Simulation 3D

描述

Simulation 3D Message Get 模块从虚幻引擎® 三维可视化环境中检索数据。在您的模型中，确保 Simulation 3D Scene Configuration 模块与 Simulation 3D Message Get 模块处于同一级别。

Tip

验证 Simulation 3D Scene Configuration 模块是否在 Simulation 3D Message Get 模块之前执行。这样，虚幻引擎三维可视化环境就会在 Simulation 3D Message Get 模块接收数据之前准备好。要检查模块执行顺序，请右键单击模块并选择“属性（Properties）”。在“常规（General）”选项卡上，确认以下优先级（Priority）设置：

  • Simulation 3D Scene Configuration — 0

  • Simulation 3D Message Get — 1

有关执行顺序的更多信息，请参阅 控制和显示执行顺序。

配置场景发送数据

要使用该模块，您必须在虚幻引擎环境中配置场景以将数据发送到 Simulink® 模型：

  1. 安装 用于车辆动力学仿真的自定义三维场景。

  2. 在虚幻® 编辑器中，按照以下一般工作流程将数据发送到 Simulink。有关详细信息，请参阅 开始于虚幻引擎可视化环境通信。

    虚幻引擎用户

    		 工作流程

    蓝图

    1. 实例化想要发送到 Simulink 模型对应数据的 Sim3DSet 参与者。该示例显示虚幻编辑器 Sim3DSet 数据类型。

    2. 指定与 Simulation 3D Message Get 模块 Signal name 参数匹配的参与者标签名称。

    3. 导航至关卡蓝图。

    4. 根据由 Simulation 3D Message Get 模块数据类型（Data type）和消息大小（Message size）参数指定的数据类型和大小，查找 Sim3DSet 参与者类的蓝图方法。 

      例如，在虚幻编辑器中，此图显示的 Write Array Boolean 是发送数组大小为 30 的布尔数据类型的 Sim3DSetBoolean 参与者类的方法。

    5. 编译并保存场景。

    笔记

    默认情况下，双车道变更场景（Double Lane Change）有一个 Sim3DSetBoolean 标签名为 NumOfConesHit 的参与者。

    C++ 类

    1. 为您希望 Simulink 模型与之交互的网格或资产创建一个新的参与者类。继承自 ASim3dActor。

    2. 在新的参与者类中：

      • 将指向信号名称的指针声明为类字段。

      • 获取类标签。

      • 创建一个信号写入器并在方法 Sim3dSetup 中分配指针。

      • 在方法 Sim3dStep中，调用 WriteSimulation3DMessage 函数将数据写入 Simulink 模型。

      • 在参与者的 Sim3dRelease 方法中删除信号写入器。

    有关虚幻编辑器的更多信息，请参阅 Unreal Engine 4 Documentation。

ReadMsg — 从场景中检索到的数据
标量 | 向量

从三维可视化环境场景数据中检索的数据。在虚幻引擎环境中，您可以使用 Sim3DSet 类来配置场景参与者，以将数据发送到 Simulink 模型。

例如，在虚幻编辑器中，双车道变更场景（Double Lane Change）有一个带标签名为 NumOfConesHit 的 Sim3DSetBoolean 参与者。用它来检索车辆在双车道变换操作期间撞击的圆锥体数量。

参数

expand all

Signal name, SigName — 消息信号名称
mySignal (默认)

指定三维可视化环境中的信号名称。在虚幻引擎环境中，使用位于“详细信息（Details）”窗格中的 Sim3DSet 参与者类“标签（Tags）”属性。

例如，您可以从双车道变更场景中检索数据，以指示在双车道变更操作期间是否撞到了锥体。要从双车道变换场景中检索锥体撞击数据，请将此参数设置为 NumOfConesHit。在双车道变更场景中，Sim3DSet 参与者类的“Tags”属性设置为 NumOfConesHit。

Data type, DataType — 消息数据类型
double* | single | int8* | uint8* | int16* | uint16* | int32 | uint32* | boolean

三维可视化环境信号数据类型。支持的数据类型取决于虚幻引擎工作流程。

在虚幻引擎环境中，实例化要发送到 Simulink 模型的数据类型的 Sim3DSet 参与者类。例如，您可以从双车道变更场景中检索数据，以指示在双车道变更操作期间是否撞到了锥体。要从双车道变换场景中检索锥体撞击数据，请将此参数设置为 boolean。在双车道变更场景中，实例化 Sim3DSetBoolean 参与者类以发送圆锥体命中或未命中的布尔数据。

Message size, MsgSize — 消息维度
[1 1] (默认) | 标量 | 数组

三维可视化环境信号维度。在虚幻引擎环境的蓝图中，将输入设置为 Sim3DSet 参与者类的节点，以指定要发送到 Simulink 模型的数据的维度。

例如，您可以从双车道变更场景中检索数据，以指示在双车道变更操作期间是否撞到了锥体。要从双车道变换场景中检索圆锥体撞击数据，请将此参数设置为 [2 15]. 。在双车道变更场景中，Sim3DSetBoolean 参与者类的蓝图节点的输入设置为 30，即场景中的锥体数量。

Sample time — 采样时间
0.02 (默认) | -1 | 标量

采样时间，以秒为单位。图形帧速率是采样时间的倒数。如果将采样时间设置为 -1，则该模块将使用在 Simulation 3D Scene Configuration 模块中指定的采样时间。
