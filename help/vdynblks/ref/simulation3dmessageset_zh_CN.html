Simulation 3D Message Set

将数据发送到虚幻引擎可视化环境

expand all in page
  • 库:

  • Vehicle Dynamics Blockset / Vehicle Scenarios / Sim3D / Sim3D Core

    Aerospace Blockset / Animation / Simulation 3D

    Simulink 3D Animation / Simulation 3D

描述

Simulation 3D Message Set 模块将数据发送到虚幻引擎® 三维可视化环境。在您的模型中，确保 Simulation 3D Scene Configuration 模块与 Simulation 3D Message Set 模块处于同一级别。

提示

验证 Simulation 3D Message Set 模块是否在 Simulation 3D Scene Configuration 模块之前执行。这样，Simulation 3D Message Set 在虚幻引擎三维可视化环境接收信号数据之前准备好信号数据。要检查模块执行顺序，请右键单击块并选择 Properties。在 General 选项卡上，确认一下优先级 Priority 设置：

  • Simulation 3D Scene Configuration — 0

  • Simulation 3D Message Set — -1

有关执行顺序的更多信息，请参阅 控制并显示执行顺序。

配置场景以接收数据

要使用该模块，您必须在虚幻引擎环境中配置场景以从 Simulink® 模型接收数据：

  1. 安装 Customize 3D Scenes for Vehicle Dynamics Simulations。

  2. 在虚幻® 编辑器中，按照以下一般工作流程实现从 Simulink 接收数据。有关详细信息，请参阅开始与虚幻引擎可视化环境通信。

    虚幻引擎用户

    		 工作流程

    Blueprint

    1. 实例化想要从 Simulink 模型接收的数据类型相对应的参与者 Sim3DGet。此示例显示虚幻编辑器 Sim3DGet 数据类型（从左上角“放置Actor”搜索类中进行搜索）。

    2. 指定与 Simulation 3D Message Set 模块 Signal name 参数匹配的参与者标签名称。

    3. 导航到关卡蓝图（Level Blueprint）：菜单“蓝图”->“打开关卡蓝图”。

    4. 根据您想要从 Simulink 模型接收的数据类型和大小，找到 Sim3DGet 参与者类的蓝图方法。

      例如，在虚幻编辑器中，此图显示的是 Sim3DGetInterger 类中  Read Scalar Integer 方法接收 int32 大小标量数据类型。 

    5. 编译并保存场景。

    笔记

    默认情况下，双车道变更场景 Double Lane Change 有一个标签名为 TrafficLight1 的 Sim3DGetInteger 参与者。

    C++ class

    1. 如果您希望创建一个新的参与者与 Simulink 模型进行交互。从 ASim3dActor 从继承。

    2. 在新的参与者类中：

      • 将指向信号名称的指针声明为类字段。

      • 获取类标签。

      • 创建一个信号读取器并在方法 Sim3dSetup 中分配指针。

      • 在方法 Sim3dStep 中，调用方法 ReadSimulation3DMessage 从 Simulink 模型中读取数据。

      • 在参与者的 Sim3dRelease 方法中删除信号读取器。

    有关虚幻编辑器的更多信息，请参阅 虚幻引擎 4 文档。

端口

Input

expand all

WriteMsg — 发送到场景的数据
标量 | 数组

数据发送到三维可视化环境场景。在虚幻引擎环境中，您可以配置 Sim3DGet 类以从 Simulink 模型接收数据。

例如，在虚幻编辑器中，双车道变更场景 Double Lane Change 有一个标签名为 TrafficLight1 的 Sim3DGetInteger 整型参与者。该整型参与者从 Simulink 模型中读取 int32 数据类型。您可以用它来控制交通信号灯的颜色。

此表提供了与双车道变换场景 Double Lane Change 中的 WriteMsg 值相对应的场景交通信号灯颜色。

Simulation 3D Message Set Block WriteMsg 值

TrafficLight1 颜色
0 红
1 黄
2 绿

参数

expand all

Signal name, SigName — 消息信号名称
mySignal (默认)

指定三维可视化环境中的信号名称。在虚幻引擎环境中，使用“详细信息”（Details）窗格中的 Sim3Get 参与者类。

例如，您可以将数据发送到双车道变更场景，将交通信号灯颜色更改为红色、黄色或绿色。要将数据发送到交通信号灯，请将此参数设置为 TrafficLight1。在双车道变换场景中，将 Sim3dGetInteger 参与者类的“Tags”属性设置为 TrafficLight1。

Sample time — 采样时间
0.02 (默认) | -1 | scalar

采样时间，以秒为单位。图形帧速率是采样时间的倒数。如果将采样时间设置为 -1，则该模块将使用在 Simulation 3D Scene Configuration 模块中指定的采样时间。
