Microsoft .NET 快速入门

有助于您迅速地在 MATLAB® 中开始使用 .NET 的示例和概念

.NET 接口允许您将 .NET 程序集引入 MATLAB 环境中，从这些程序集中包含的类来构造对象，以及对这些对象调用方法。

MATLAB 仅在 Microsoft® Windows® 平台上支持加载为 Microsoft .NET Framework 编译的库。有关与 MATLAB 兼容的 .NET 版本的信息，请参阅其他语言的 MATLAB 接口。

要确定系统是否具有支持的版本，请使用 NET.isNETSupported 函数。.NET 相关的文档请参考链接。
