导航工具箱

设计、仿真和部署自主导航算法
  • Release Notes
  • PDF Documentation
  • 发行说明
  • PDF 文档

导航工具箱™ 提供用于运动规划、同步定位和建图（Simultaneous Localization And Mapping, SLAM）以及惯性导航的算法和分析工具。该工具箱包括可定制的搜索和基于采样的路径规划器，以及用于验证和比较路径的指标。您可以创建二维和三维地图表示，使用 SLAM 算法生成地图，并使用 SLAM 地图生成器应用程序以交互方式可视化和调试地图生成。该工具箱提供用于定位的传感器模型和算法。您可以模拟和可视化 IMU、GPS 和车轮编码器传感器数据，并调整融合滤波器以进行多传感器姿态估计。

提供了自动驾驶、机器人和消费电子应用的参考示例。您可以通过将导航算法直接部署到硬件（使用 MATLAB® Coder™ 或 Simulink® Coder）。
