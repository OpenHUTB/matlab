sim3d.World

用于在虚幻引擎查看器中定义虚拟现实世界的对象

expand all in page

    描述

    使用该 sim3d.World 对象创建和定义虚拟现实世界，并使用 Unreal Engine®运行协同仿真。

    • 在世界中添加或删除自定义参与者。仅显示添加到世界对象的参与者。

    • 存储虚拟现实场景、sim3d.Actor 根对象以及场景中所有参与者的层次结构。

    • （可选）通过指定自定义函数来控制仿真逻辑。输出函数将数据传递到虚幻引擎以更改引擎中参与者的属性。

    • 控制 MATLAB® 和虚幻引擎之间的协同仿真。

    • 定义虚拟世界全局坐标系。

    • 控制场景环境设置。

    • 控制虚拟世界的时序。

    创建

    语法

    world = sim3d.World()
    world = sim3d.World('Name',name)
    world = sim3d.World('RenderOffScreen',true)
    world = sim3d.World('Setup',@setupFcn)
    world = sim3d.World('Update',@updateFcn)
    world = sim3d.World('Output',@outputFcn)
    world = sim3d.World('Release',@releaseFcn)

    描述

    world = sim3d.World() 在新窗口中使用默认名称启动默认的虚幻引擎可执行文件。

    world = sim3d.World('Name',name) 在新窗口中使用指定的名称启动虚幻引擎可执行文件。

    world = sim3d.World('RenderOffScreen',true) 在后台运行虚幻引擎可执行文件的同时运行协同仿真。

    world = sim3d.World('Setup',@setupFcn) 通过在设置之前执行 @setupFcn 来修改协同仿真，此自定义函数可用于导入仿真运行时所需的附加数据。

    world = sim3d.World('Update',@updateFcn) 通过在每个仿真步骤执行 @updateFcn 来修改协同仿真，该自定义函数可用于从虚幻引擎读取数据。

    world = sim3d.World('Output',@outputFcn) 通过在每个仿真步骤执行 @outputFcn 来修改协同仿真。此自定义函数可用于将有关指定 sim3d.Actor 对象的数据发送到虚幻引擎。

    world = sim3d.World('Release',@releaseFcn) 通过在每个仿真步骤执行 @releaseFcn 来修改协同仿真。此功能可用于删除在设置过程中可能导入的其他数据。

    输入参数

    expand all

    名-值参数对

    将可选参数对指定为 Name1=Value1,...,NameN=ValueN，其中 Name 是参数名称，Value 是相应的值。名-值参数对必须出现在其他参数之后，但参数对的顺序并不重要。

    示例： World = sim3d.World('Name',world1)

    Name — 世界的名称
    string

    世界的名称，指定为字符串。

    ExecutablePath — 指向可执行文件的路径
    string

    运行世界的可执行文件路径，指定为字符串。

    Map — 指向虚幻引擎地图的路径
    string

    运行可执行文件时加载的虚幻引擎映射的路径，指定为字符串。

    RenderOffScreen — 在后台运行模拟的选项

    0 (false) | 1 (true)

    在后台运行仿真的选项，指定为 0(false) 或 1(true)。

    Update — 自定义更新函数
    用户自定义函数的句柄

    自定义更新函数，用于从虚幻引擎读取有关指定参与者的数据，指定为用户定义函数的句柄。

    Output — 自定义输出函数
    用户自定义函数的句柄

    自定义输出函数，将指定参与者的变换和颜色等信息返回到虚幻引擎，指定为用户定义函数的句柄。这使得虚幻引擎能够控制仿真。

    Setup — 定制的设置函数
    用户自定义函数的句柄

    在设置过程中读取并运行其他命令的自定义函数，指定为用户定义函数的句柄。这可用于加载仿真所需的数据。

    Release — 定制的释放函数
    用户自定义函数的句柄

    自定义释放函数，在仿真结束后运行附加命令，指定为用户定义函数的句柄。此设置可用于删除为设置导入的其他数据。

    属性

    collapse all

    'Name' — 世界名称
    string

    世界的名称，指定为字符串。

    Actors — 世界中的所有参与者
    structure

    世界中的所有参与者的结构体，其字段为参与者的名字。

    UserData — 用户指定的数据
    structure

    用户指定的在仿真运行时可能需要的数据，指定为结构。

    需要定制的 Update, Output, Setup 和 Release 函数来使用 UserData 存储数据。UserData 确保所有这些功能都可以访问相同的数据。

    Viewports — 主相机的视角
    empty struct (默认) | struct

    主摄像机的视角，定义为包含单个场的结构，Main. Main 包含一个具有这些属性的 sim3d.sensors.MainCamera 对象：

    • Parent – 参与者对象的父对象

    • Children – 参与者对象的子对象

    • ParentWorld – 父世界的句柄

    • SensorIdentifier – 主相机的唯一 ID，数据类型为 uint32。

    • Translation – 主摄像头相对于父车辆的平移，以米为单位。

    • Rotation – 传感器相对于父车辆的旋转，以弧度为单位。

    对象的函数

    createViewport 为世界创建视口
    add 将参与者添加到虚拟现实世界
    run 在虚拟现实世界中运行协同仿真
