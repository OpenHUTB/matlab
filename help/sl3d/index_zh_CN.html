Simulink 3D 动画

在虚拟现实环境中可视化动态系统行为
  • Release Notes
  • PDF Documentation
  • Release Notes
  • PDF Documentation

Simulink® 3D Animation™ 将 Simulink 模型和 MATLAB® 算法链接到虚拟现实场景中的三维图像目标。可以在桌面或实时模拟期间通过更改位置、旋转、缩放和其他对象属性来为虚拟世界制作动画。还可以感知虚拟世界中的碰撞和其他事件，并将它们反馈到 MATLAB 和 Simulink 算法中。来自虚拟摄像机的视频可以传输到 Simulink 进行处理。

Simulink 3D 动画包括用于渲染虚拟场景以及与虚拟场景交互的编辑器和查看器。使用三维世界编辑器，可以导入 CAD 和 URDF 文件格式，并创作由 三维对象组装而成的详细场景。使用立体视觉可以身临其境地观看 三维世界。可以在 MATLAB 图形中合并多个三维场景视图，并使用力反馈操纵杆、空间鼠标或其他硬件设备与虚拟世界进行交互。Simulink 3D 动画支持 X3D，这是一种 ISO 标准文件格式和运行时架构，用于表示 3D 场景和对象并与之通信。
