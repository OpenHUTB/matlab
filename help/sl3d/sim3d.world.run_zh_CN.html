run

在虚拟现实世界中运行协同仿真

collapse all in page

    语法

    run(world,sampleTime)
    run(world,sampleTime,simulationTime)

    描述

    run(world,sampleTime) 使用 sampleTime 中指定的采样周期运行协同仿真。

    run(world,sampleTime,simulationTime) 额外指定仿真时间 simulationTime。

    输入参数

    collapse all

    world — World object
    sim3d.World 对象

    The sim3d.World object where cosimulation is being run.

    sampleTime — 世界的采样周期
    0.02 | 标量

    虚拟现实世界的采样周期，指定为正实标量。

    数据类型：single

    simulationTime — 仿真时间长度
    Inf | 标量

    仿真运行时间的长度，指定为正实数标量。

    数据类型：single

    版本历史

    Introduced in R2022b
