虚幻引擎场景应用

构建虚拟现实场景并与之交互

Simulink® 3D Animation™ 工具箱提供了构建、可视化虚拟现实场景并与之交互的功能，并使用虚幻引擎® 查看器执行基于代理的建模。

  • 访问虚幻引擎中提供的高级功能，例如碰撞检测和 PhysX 物理引擎，并通过 Simulink 三维动画工具箱的 MATLAB® 和 Simulink 界面设置照明和大气条件。

  • 使用 3D 资产以不同支持的文件格式（例如 VRML、X3D、FBX、STL、DAE Collada、URDF 和 SDF）通过算法构建虚拟世界。

  • 使用 MATLAB API 控制和访问虚拟场景中的所有对象并设置视觉、空间和其他属性。还可以使用此 API 来控制其他功能，例如导航行为、碰撞事件以及连接到对象的相机或深度传感器的视频输出。

  • 在工作时创建或删除对象并独立控制对象实例。

功能

collapse all

参与者

copy 复制另一个参与者的所有属性
propagate 将所选属性的值传播给参与者及其子节点
gather 从选定分支中的所有对象返回选定属性的值
findBy 查找符合指定条件的所有参与者
restoreSnapshot 将参与者恢复到指定快照中保存的属性状态
takeSnapshot 拍摄所选属性的快照
createMesh 使用指定值创建新网格
addMesh 在当前网格的顶部添加网格
load 加载或导入三维文件
save 将参与者和子节点保存到 MAT 文件
createShape 为基本图元创建几何体

世界

add 将参与者添加到虚拟现实世界
run 在虚拟现实世界中运行协同仿真
createViewport 为世界创建视窗

传感器

read 返回相机拍摄的图像

类

sim3d.Actor 用于在虚幻引擎查看器中定义参与者对象
sim3d.World 用于在虚幻引擎查看器中定义虚拟现实世界的对象
sim3d.sensors.IdealCamera 捕获图像并使其在 MATLAB 中可用
sim3d.sensors.MainCamera 在虚幻可执行窗口中定义透视图

模块

Simulation 3D Actor 在虚幻引擎查看器中定义参与者
Simulation 3D Actor Transform Get 获取参与者平移、旋转、缩放
Simulation 3D Actor Transform Set 设置参与者平移、旋转、缩放
Simulation 3D Camera Get 相机图像
Simulation 3D Scene Configuration 三维仿真环境的场景配置

Scenes

Empty scene 空环境
