sim3d.Actor

用于在虚幻引擎查看器中定义参与者的对象

expand all in page

    描述

    将 sim3d.Actor 对象用于用户定义的虚幻引擎® C++ 或蓝图参与者。

    创建

    Syntax

    actor = sim3d.Actor
    actor = sim3d.Actor(___,Name,Value)

    描述

    actor = sim3d.Actor 创建一个代表虚拟现实世界中所有项目的默认参与者对象。

    actor = sim3d.Actor(___,Name,Value) 使用一个或多个名-值参数指定附加选项。

    输入参数

    expand all

    名-值参数

    将可选参数对指定为 Name1=Value1,...,NameN=ValueN，其中 Name 是参数名称，Value 是相应的值。名-值参数必须出现在其他参数之后，但参数对的顺序无所谓。

    例子: actor1 = sim3d.Actor('ActorName','vehicle1','Translation', [3, 4, 3], 'Rotation', [], 'Scale',[1 2 1])

    ActorName — 参与者名称
    自动生成的名称（默认） | 字符数组 | 字符串

    参与者的名称，指定为字符数组或字符串。如果未指定参与者名称，则会为参与者分配一个自动生成的名称。

    批注

    如果指定的名称与已存在的参与者相同，则指定的参与者名称将附加一个唯一标识符以进行区分。

    Translation — 相对平移
    [0,0,0] (默认) | 实数 1×3 向量

    参与者对象与其父参与者的相对平移 (x,y,z) ，指定为实数 1×3 的向量（以米为单位）。使用此参数来设置 sim3d.Actor 对象的 Translation 属性。

    数据类型: double

    Rotation — 相对旋转
    [0,0,0] (默认) | 实数 1×3 向量

    参与者对象相对于其父参与者的相对旋转（翻滚、俯仰、偏航），指定为实数 1×3 向量（以弧度为单位）。使用此参数来设置 sim3d.Actor 对象的 Rotation 属性。

    数据类型: double

    Scale — 相对缩放
    [1,1,1] (默认) | 实数 1×3 向量

    x, y 和 z 坐标中的相对缩放，指定为实数 1×3 向量。使用此名-值参数来设置对象  sim3d.Actor 的 Scale 属性。

    数据类型: double

    ActorClassId — 对象类别标识符的语义分割图
    0 (默认) | m×n 数组

    对象类别标识符的语义分段图，以 m×n 数组数组的形式返回。m 是图像的垂直分辨率，n 是图像的水平分辨率。

    该表显示了 Simulation 3D Scene Configuration 模块中可用的默认场景中使用的对象 ID 。如果使用自定义场景，则可以将新对象类型分配给未使用的 ID。如果场景包含没有分配 ID 的对象，则为该对象分配 ID 为 0。不支持车道标记检测。

    依赖

    要启用此端口，请在 Simulation 3D Scene Configuration 模块上，选择 Ground Truth 选项卡上的 Output semantic segmentation。

    除了这些名-值参数之外，还可以将此页面上的任何属性指定为名-值参数。

    输出参数

    collapse all

    actor — 参与者对象
    sim3d.Actor 对象

    参与者对象，作为 sim3d.Actor 对象返回。为参与者分配唯一的标识符和唯一的名称。可以更改参与者名称，但唯一标识符会与参与者相关联，直到参与者被删除为止。虚拟现实世界是由参与者的层次结构组成。

    属性

    expand all

    基础属性

    UserData — 保存参与者数据的结构
    结构体

    保存参与者数据的结构，指定为结构体。可以更新和维护此数据，并在仿真期间或设置方法中使用它来更新结构的字段。

    Parent — 参与者的父对象
    指向父级 sim3d.actor 对象的句柄

    参与者的父级，指定为父 sim3d.Actor 对象的句柄。可以使用 Parent 参数设置此属性。

    Children — 参与者的子对象
    指向子级 sim3d.actor 对象的句柄

    参与者的子级，指定为对象 sim3d.Actor 的句柄。

    该属性是只读的。

    ParentWorld — 父世界方法句柄
    父级 sim3d.World 对象的句柄

    该属性是只读的。

    父世界方法句柄，指定为父级 sim3d.Actor 对象的句柄。

    Translation — 参与者位置的笛卡尔坐标
    [0,0,0] (默认) | 实数 1×3 向量

    虚幻® 编辑器场景中参与者位置的笛卡尔坐标，指定为实数 1×3 向量。可以使用 Translation 参数设置此属性。

    Rotation — 参与者旋转
    [0,0,0] (默认) | 实数 1×3 向量

    虚幻编辑器场景中参与者的旋转，指定为三元素向量。可以使用 Rotation 名-值参数设置此属性。

    Scale — 参与者缩放
    [1,1,1] (默认) | 实数 1×3 向量

    用于在虚幻编辑器场景中的三个方向上进行缩放的参与者的比例，指定为 1×3 的实数向量。可以使用 Scale 参数设置此属性。

    Mobility — 参与者移动类型
    'sim3d.utils.MobilityTypes.Movable' (默认) | 'sim3d.utils.MobilityTypes.Static'

    在仿真过程中响应物理和/或移动参与者的参与者移动性类型，指定为 'sim3d.utils.MobilityTypes.Movable' 或 'sim3d.utils.MobilityTypes.Static'。

    数据类型: string

    网格属性

    Vertices — 网格几何体的顶点
    空数组 (默认) | 实数 n×3 向量

    Vertices of the mesh geometry, specified as a real n-by-3 vector, where n is the number of vertices.

    Example: actor.Vertices = [1, 2, 3; 43, 54, 65]

    Data Types: double

    Faces — 每个三角形的顶点
    空数组 (默认) | 实数 n×3 向量

    每个三角形的顶点，指定为 n×3 向量，其中 n 是定点数。可以使用此属性来定义网格的每个三角形的绘制方式。

    例子: actor.Faces = [1, 2, 3; 4, 5, 6]

    数据类型: double

    Normals — 每个顶点的法线
    空数组 (默认) | real n-by-3 vector

    每个顶点的法线，指定为 n×3 实数向量，其中 n 是顶点数。Normals 的长度必须与 Vertices 相同。

    例子: actor.Normals = [1, 2, 3; 43, 54, 65]

    数据类型: double

    TextureCoordinates — 纹理坐标
    空数组 (默认) | 实数 n×2 向量

    每个顶点的纹理坐标，指定为 n×2 的实数向量，其中 n 是顶点数。该参数是可选的，如果指定，则其长度必须与 Vertices 相同。

    例子: actor.TextureCoordinates = [1 2;3 4]

    VertexColors — 顶点颜色
    空数组 (默认) | 实数 n×3 向量

    顶点颜色，指定为 n×3 的实数向量，其中 n 是顶点数。该参数是可选的，如果指定，则其长度必须与 Vertices 相同。

    例子: actor.VertexColors = [1, 2,3; 4, 5, 6]

    材质属性

    Color — 参与者的基色
    [1,1,1] (默认) | 实数 1×3 的向量

    参与者的基色，指定为 1×3 的实数向量。颜色由颜色 RGB 分量的向量组成 [红 绿 蓝]。大于 1 的值会导致发光并可用于各种指示器。如果 sim3d.Actor 对象具有纹理，则TextureMapping.Blend 可以使用系数来控制它与基色的混合方式。

    Example: actor.Color = [0.3 0.27 0.9]

    Transparency — 参与者的透明度
    0 (默认) | (0,1) 范围内的实正数

    参与者的透明度，指定为 (0,1) 范围内的实正数，其中 0 表示不透明对象，1 表示完全透明对象。

    例子: actor.Transparency = 0.8

    数据类型: double

    Shininess — 参与者的光泽度
    0.7 (默认) | (0,1) 范围内的实正数

    参与者的光泽度，指定为 (0,1) 范围内的实数正数，其中 0 表示不发光的对象，1 表示完全有光泽的对象。

    例子: actor.Shininess = 0.3

    数据类型: double

    Metallic — 参与者的金属外观
    0.7 (默认) | (0,1) 范围内的实正数

    参与者的金属外观，指定为 (0,1) 范围内的实正数，其中 0 表示塑料表面，1 表示金属表面。

    例子: actor.Metallic = 0.1

    数据类型: double

    Flat — 参与者的平面着色系数
    0 (默认) | (0,1) 范围内的实正数

    参与者的平面着色因子，指定为 (0,1) 范围内的实正数，其中 0 表示光滑表面，1 表示多面表面。

    例子: actor.Flat = 0.7

    数据类型: double

    Tessellation — 参与者的细分因子
    0 (默认) | (0,8) 范围内的实正整数

    参与者的细分因子，指定为 (0,8) 范围内的实正整数。使用此属性指定自动几何细化的系数。当使用纹理置换贴图时，此属性非常有用。

    例子: actor.Tessellation = 5

    VertexBlend — 顶点颜色混合系数
    0 (默认) | (0,1) 范围内的实正数

    顶点的颜色混合系数，指定为 (0,1) 范围内的正实数。这个值可以大一些，可以达到发光颜色的效果。

    例子: actor.VertexBlend = 0.5

    TextureTransform — 应用于参与者的纹理变换
    正实数标量

    应用于参与者的纹理变换，指定为正实数标量。使用 TextureTransform 定义纹理位置、速度、比例和角度。

    变换属性

    属性 细节 值 值的示例
    位置

    使用此属性可以设置纹理的位置。可以使用 Position 来沿 U 和 V 坐标移动纹理 。

    • 值 – (1,2) 正实数向量

    • 默认值 – (0,0)

    • 数据类型 – double

    		 actor.TextureTransform.Position = [2, 3]
    速度

    使用此属性可以设置纹理运动的速度。速度表示为模拟时间每秒 U 和 V 坐标的变化。动画表面对物理交互没有影响。

    • 值 – (1,2) 正实数向量

    • 默认值 – (0,0)

    • 数据类型 – double

    		 actor.TextureTransform.Velocity = [2, 3]
    比例

    使用此属性可以设置相对纹理比例。如果纹理小于要覆盖的表面，则比例会在所有方向上自动重复。负值可用于翻转相应坐标中的纹理。

    • 值 – (1,2) 正实数向量

    • 默认值 – (0,0)

    • 数据类型 – double

    		 actor.TextureTransform.Scale = [2, 2]
    角度

    使用此属性可以设置相对纹理角度。纹理始终围绕 UV 坐标中的中心点 [0.5, 0.5] 旋转 。中心固定在纹理上，并根据位置和速度参数随之移动。负值可用于在相应坐标中翻转纹理。

    • 值 – 正实数标量

    • 默认值 – 0

    • 数据类型 – double

    		 actor.TextureTransform.Angle = pi/2

    TextureMapping — 应用于参与者纹理的纹理映射参数
    正实数向量

    应用于参与者纹理的纹理映射参数，指定为正实数向量。用 TextureMapping 定义纹理混合、位移、凹凸因子和粗糙度。

    映射属性

    属性 细节 值 值的示例
    混合

    使用此属性可以设置纹理的混合比率。

    可以使用线性插值 Blend 来设置纹理与表面基色的混合比例。

    要创建发光纹理效果，请使用黑色基色 ([0 0 0]) 和大于 1 的混合值。

    可以为每个颜色通道单独设置值（[红 绿 蓝]）。

    • 值 – (1,3) 正实数向量

    • 默认值 – (0,0,0)

    • 数据类型 – double

    		 actor.TextureMapping.Blend = [1 1 0.5]
    位移

    使用此属性可以设置纹理的位移。

    可以使用 Displacement 根据该位置的纹理颜色沿法线方向移动几何网格的顶点。这种变形只是视觉上的，不会影响物理交互。

    可以为每个颜色通道（[红 绿 蓝]）独立设置值，其范围不受限制。

    • 值 – (1,3) 正实数向量

    • 默认值 – (0,0,0)

    • 数据类型 – double

    		 actor.TextureMapping.Displacement = [1 2 5]
    凹凸因子

    使用此属性可以设置纹理的凹凸因子。

    可以使用 Bump 对法线和光照进行小的操作来创建凹凸不平的表面（例如石墙）的效果。

    可以为每个颜色通道（[红 绿 蓝]）独立设置值，其范围不受限制。

    • 值 – (1,3) 正实数向量

    • 默认值 – (0,0,0)

    • 数据类型 – double

    		 actor.TextureMapping.Bumps = [10 10 0.5]
    粗糙度

    使用此属性可以设置纹理的粗糙度因子。

    可以用 Roughness 局部操纵由 Shininess 属性指定的全局表面光泽度。

    可以为每个颜色通道（[红 绿 蓝]）独立设置值，其范围不受限制。

    • 值 – (1,3) 正实数向量

    • 默认值 – (0,0,0)

    • 数据类型 – double

    		 actor.TextureMapping.Roughness = [1 1 0.5]

    Texture — 参与者形状的源文件
    '' (默认) | 字符数组

    参与者形状的源文件，指定为字符数组。支持的文件类型包括 JPEG、PNG、BMP 和 TGA。文件路径应该是绝对路径。

    示例: actor.Texture = "file.jpg"

    物理属性

    Shadows — 参与者阴影
    0 或 false (默认) | 1 或 true

    参与者阴影，如果参与者不投射阴影，则指定为 0 (false) ，如果投射阴影，则指定为 1 (true) 。

    LinearVelocity — 参与者的线速度
    (0,0,0) (默认) | 正实数向量

    局部坐标中参与者的线速度，指定为正实数向量，以米每秒为单位。

    示例: actor.LinearVelocity = [1 1 1]

    依赖

    为了使速度生效，应该将 Mobility 设置为 'sim3d.utils.MobilityTypes.Movable'。否则参与者将不会移动，并且会看到警告。

    AngularVelocity — 参与者的角速度
    (0,0,0) (默认) | 正实数向量

    局部坐标中参与者的角速度，指定为正实数向量，以弧度每秒为单位。

    示例: actor.AngularVelocity = [2 2 2]

    依赖

    为了使速度生效，应该将 Mobility 设置为 'sim3d.utils.MobilityTypes.Movable'。否则参与者将不会移动，并且会看到警告。

    Mass — 参与者的质量
    0 (默认) | 正实数标量

    参与者的质量，指定为正实数标量，以千克为单位。

    示例: actor.Mass = 12

    CenterOfMass — 参与者的质心
    (0,0,0) (默认) | 正实数向量

    sim3d.Actor 对象物体的质心 ，指定为正实数向量。使用此属性可将重心从局部坐标系的原点移动。

    示例: actor.CenterOfMass = [1 0 1 ]

    Gravity — 对参与者施加重力
    0 或 false (默认) | 1 或 true

    对参与者施加重力，如果没有施加重力，则指定为 0 (false) ，如果施加重力，则指定为 1 (true)。

    示例: actor.Gravity = false

    依赖

    • 当 Physics属性设置为 to 1 或 true 时，此属性起作用。

    • 为了让参与者体验重力的影响，应该将 Mobility 设置为 'sim3d.utils.MobilityTypes.Movable'。否则参与者将不会移动，并且会看到警告。

    Physics — 参与者对物理作用力的反应
    0 或 false (默认) | 1 或 true

    参与者对物理作用力的反应，指定为 0 (false) 或 1 (true)。

    如果 Physics 启用，则参与者独立于其父级参与者对象移动，但与其子级对象一起移动，除非子对象也启用 Physics。

    示例: actor.Physics = true

    Collisions — 对象碰撞
    0 或 false (默认) | 1 或 true

    对象碰撞，如果没有碰撞，则指定为 0 (false)，如果对象将发生碰撞，则指定为 1 (true)。

    示例: actor.Collisions = true

    LocationLocked — 静止平移运动
    0 或 false (默认) | 1 或 true

    静止平移运动，指定为 0 (false) 或 1 (true)。如果启用此属性，参与者将固定在适当的位置。如果参与者定义了非零线速度，它就会与其他对象交互，就像它自己移动一样。可以使用此属性对带式输送机进行建模。

    示例: actor.LocationLocked = true

    RotationLocked — 静止旋转运动
    0 或 false (默认) | 1 或 true

    静止旋转运动，指定为 0 (false) 或 1 (true)。如果启用此属性，则 sim3d.Actor 对象将固定在合适的位置。如果参与者定义了一个非零角速度，它就会与其他对象交互，就像它自己移动一样。可以使用此属性对圆形传送带（转盘）进行建模。

    示例: actor.RotationLocked = true

    对象函数

    copy 复制另一个参与者的所有属性
    findBy 查找符合指定条件的所有参与者
    propagate 将所选属性的值传播给参与者及其子节点
    gather 从选定分支中的所有对象返回选定属性的值
    takeSnapshot 拍摄所选属性的快照
    restoreSnapshot 将参与者恢复到指定快照中保存的属性状态
    createMesh 使用指定值创建新网格
    addMesh 在当前网格的顶部添加网格
    load 加载或导入三维文件
    save 将参与者和孩子保存到 MAT 文件
    createShape 为基本图元创建几何体
