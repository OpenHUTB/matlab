机器人操作系统工具箱

设计、仿真和部署基于机器人操作系统的应用程序
  • Release Notes
  • PDF Documentation
  • 发布说明
  • PDF 文档

机器人操作系统工具箱提供了一个将 MATLAB® 和 Simulink® 与机器人操作系统 （ROS 和 ROS 2）连接起来的接口，使您能够创建机器人操作系统节点网络。 该工具箱包括 Matlab 函数和 Simulink 模块，用于导入、分析和回放 rosbag 文件中记录的机器人操作系统数据。您还可以连接到实时机器人操作系统网络以访问机器人操作系统消息。

该工具箱可让您通过桌面仿真以及连接到外部机器人仿真器（例如 Gazebo）来验证机器人操作系统节点。机器人操作系统工具箱支持 C++ 代码生成（使用 Simulink Coder™），使您能够从 Simulink 模型自动生成机器人操作系统节点并部署到仿真或物理硬件。对 Simulink 外部模式的支持使您可以在模型在硬件上运行时查看消息并更改参数。
