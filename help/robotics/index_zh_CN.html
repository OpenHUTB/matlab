机器人系统工具箱

设计、仿真、测试和部署机器人应用程序
  • Release Notes
  • PDF Documentation
  • 发行说明
  • PDF 文档

机器人系统工具箱™ 提供用于设计、仿真、测试和部署机械手和移动机器人应用程序的工具和算法。对于机械手，该工具箱包括用于碰撞检测、路径规划、轨迹生成、正向和反向运动学以及使用刚体树表示的动力学算法。对于移动机器人，它包括建图、定位、路径规划、路径规划和运动控制算法。该工具箱可构建测试情景并使用提供的参考示例来验证常见的工业机器人应用。它还包括商用工业机器人模型库，可以导入、可视化、仿真并与参考应用程序一起使用。

可以通过结合提供的运动学和动力学模型来开发功能机器人原型。该工具箱可让您通过直接连接到 Gazebo 机器人仿真器来共同模拟机器人应用程序。要在硬件上验证设计，可以连接到机器人平台（例如 Kinova Gen3 和 Universal Robots UR 系列机器人）并生成和部署代码（使用 MATLAB® Coder™ 或 Simulink® Coder）。
